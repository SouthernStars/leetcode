# 131.分割回文串
[leetcode 链接](https://leetcode.cn/problems/palindrome-partitioning/description/)

### 一、题目描述

给你一个字符串 s，请你将 s 分割成一些子串，使每个子串都是
回文串 。返回 s 所有可能的分割方案。

### 二、解题思路

回溯：分割问题

### 三、代码实现

```go
// 回溯，每次选择第 i 到第 j+1 位置的元素，判断是否是回文串，追加答案，递归，恢复现场
func partition(s string) (ans [][]string) {
    one := []string{}
    backtrace(0, s, &one, &ans)
    return ans
}

func backtrace(i int, s string, one *[]string, ans *[][]string) {
    if i==len(s) { // 结束条件        
        *ans = append(*ans, append([]string{}, (*one)...))
        return
    }
    // for-横向遍历
    for j:=i+1; j<=len(s); j++ {
        // 处理节点
        if isHuiwen(s[i:j]) {
            *one = append(*one, s[i:j])
            // 递归-深度遍历
            backtrace(j, s, one, ans)
            *one = (*one)[0:len(*one)-1] 
        }
    }
}

func isHuiwen(s string) bool {
    if len(s)==0 {
        return true
    }
    for i,j:=0,len(s)-1; i<=j; i,j=i+1,j-1 {
        if s[i] != s[j] {
            return false
        }
    }
    return true
}
```
