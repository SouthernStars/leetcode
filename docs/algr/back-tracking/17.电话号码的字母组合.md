# 17.电话号码的字母组合
[leetcode 链接](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/description/)

### 一、题目描述

给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。答案可以按 任意顺序 返回。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

1       2 abc       3 def
4 ghi   5 jkl       6 mno
7 pqrs  8 tuv       9 wxyz

### 二、解题思路

回溯：组合问题

### 三、代码实现

```go
var hashtable map[string]string = map[string]string {
    "2": "abc",
    "3": "def",
    "4": "ghi",
    "5": "jkl",
    "6": "mno",
    "7": "pqrs",
    "8": "tuv",
    "9": "wxyz",
}
func letterCombinations(digits string) (ans []string) {
    if len(digits) == 0 {
        return ans
    }
    backtrace(0, digits, "", &ans)
    return ans
}
func backtrace(i int, digits string, one string, ans *[]string) {
    if i==len(digits) { // 结束条件
        *ans = append(*ans, one)
        return
    }
    seq := hashtable[string(digits[i])]
    // for-横向遍历
    for j:=0; j<len(seq);j++ {
        // 处理节点
        one += string(seq[j])
        // 递归深度遍历
        backtrace(i+1, digits, one, ans)
        // 回溯
        one = one[0:len(one)-1]
    }
}

```
