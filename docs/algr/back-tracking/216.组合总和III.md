# 216.组合总和III
[leetcode 链接](https://leetcode.cn/problems/combination-sum-iii/description/)

### 一、题目描述

找出所有相加之和为 n 的 k 个数的组合，且满足下列条件：

只使用数字1到9 ，每个数字 最多使用一次

返回 所有可能的有效组合的列表 。该列表不能包含相同的组合两次，组合可以以任何顺序返回。

### 二、解题思路

回溯：组合问题

### 三、代码实现

```go
func combinationSum3(k int, n int) (ans [][]int) {
    one := []int{}
    backtrace(1, 0, k, n, &one, &ans)
    return ans
}

func backtrace(i int, sum int, k int, n int, one *[]int, ans *[][]int) {
    if len(*one)==k && sum == n { // 结束条件
        *ans = append(*ans, append([]int{}, (*one)...))
        return
    }
    // for-横向遍历
    for j:=i; j<=9; j++ {
        // 处理节点
        sum += j
        *one = append(*one, j)
        // 递归-深度遍历
        backtrace(j+1, sum, k, n, one, ans)
        // 回溯
        sum -= j
        *one = (*one)[0:len(*one)-1]
    }
}
```
