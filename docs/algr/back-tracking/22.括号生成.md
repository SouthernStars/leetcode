# 22.括号生成
[leetcode 链接](https://leetcode.cn/problems/generate-parentheses/description/)

### 一、题目描述

数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。


### 二、解题思路

回溯

### 三、代码实现

```go
func generateParenthesis(n int) (res []string) {
    chars := []string{"(", ")"}
    backtrace(chars, n, 0, "", &res)
    return res
}
// 生成所有的 结果
func backtrace(chars []string, n int, index int, one string, res *[]string) {
    if index==2*n { // 结束条件，收集结果
        if isValid(one) {
            *res = append(*res, one)
        }
        return
    }
    // for 横向遍历
    for i:=0; i<len(chars); i++ {
        one += chars[i]
        // 递归纵向遍历
        backtrace(chars, n, index+1, one, res)
        // 回溯
        one = one[0:len(one)-1]
    }
}
func isValid(s string) bool {
    sub := 0
    for i:=0; i<len(s); i++ {
        if s[i] == '(' {
            sub ++
        } else {
            sub --
        }
        if sub < 0 {
            return false
        }
    }
    if sub != 0 {
        return false
    }
    return true
}
```
