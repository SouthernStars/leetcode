# 40.组合总和II
[leetcode 链接](https://leetcode.cn/problems/combination-sum-ii/description/)

### 一、题目描述

给定一个候选人编号的集合 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。

candidates 中的每个数字在每个组合中只能使用 一次 。

注意：解集不能包含重复的组合。

### 二、解题思路

回溯：组合问题

处理重复：排序 + 搜索树同层验证是否同值，同值 跳过保证不重复

### 三、代码实现

```go
func combinationSum2(candidates []int, target int) (ans [][]int) {
    one := []int{}
    // 不重复就需要按 顺序 搜索， 在搜索的过程中检测分支是否会出现重复结果 
    // (搜索中同层后续才出现那么前面出现的那个节点就能将结果全部遍历到)
    sort.Ints(candidates)
    backtrace(0, 0, target, &one, candidates, &ans)
    return ans
}

func backtrace(i int, sum int, target int, one *[]int, candidates []int, ans *[][]int) {
    if sum >= target  { // 结束条件
        if sum == target {
            *ans = append(*ans, append([]int{}, (*one)...))
        }
        return
    }
    // for-横向遍历
    for j:=i; j<len(candidates); j++ {
        // 处理节点
        // 重复
        if j>i && candidates[j] == candidates[j-1] {
            continue
        }
        sum += candidates[j]
        *one = append(*one, candidates[j])
        // 递归-深度遍历
        backtrace(j+1, sum, target,  one, candidates, ans)
        // 回溯
        sum -= candidates[j]
        *one = (*one)[0:len(*one)-1]
    }
}
```
