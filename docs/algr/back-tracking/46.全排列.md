# 46.全排列
[leetcode 链接](https://leetcode.cn/problems/permutations/)

### 一、题目描述

给定一个不含重复数字的数组 nums ，返回其 所有可能的全排列 。你可以 按任意顺序 返回答案。

### 二、解题思路

回溯：排列问题

### 三、代码实现

```go
func permute(nums []int) (ans [][]int) {
    one := []int{}
    used := make([]bool, len(nums)) // 记录元素是否已访问
    backtrace(nums, &used, &one, &ans)

    return ans
}

func backtrace(nums []int, used *[]bool, one *[]int, ans *[][]int) {
    if len(nums) == len(*one) {// 结束条件
        *ans = append(*ans, slices.Clone(*one))
        return 
    }
    // for 横向遍历
    for i:=0; i<len(nums); i++ {
        // flag := false
        // for j:=0; j<len(*one); j++ {
        //     if nums[i] == (*one)[j] {
        //         flag = true
        //     }
        // }
        // if flag {
        //     continue
        // }
        if (*used)[i] == true {
            continue
        }
        (*used)[i]=true
        *one = append(*one, nums[i])
        backtrace(nums, used, one, ans)
        *one = (*one)[0:len(*one)-1]
        (*used)[i]=false
    }
}

```
