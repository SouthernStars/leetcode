# 47.全排列II
[leetcode 链接](https://leetcode.cn/problems/permutations-ii/description/)

### 一、题目描述

给定一个可包含重复数字的序列 nums ，按任意顺序 返回所有不重复的全排列。

### 二、解题思路

回溯：排列问题

### 三、代码实现

```go
// 回溯+去重（排序+横向去重）
func permuteUnique(nums []int) (ans [][]int) {
    one := []int{}
    used := make([]bool, len(nums))
    sort.Ints(nums)
    backtrace(nums, &used, &one, &ans)
    return ans
}

func backtrace(nums []int, used *[]bool, one *[]int, ans *[][]int) {
    if len(nums) == len(*one) {
        *ans = append(*ans, slices.Clone(*one))
        return
    }
    // for - 横向遍历
    for i:=0;i<len(nums); i++ {
        if i>0 && nums[i] == nums[i-1] &&(*used)[i-1]==false {// 去重，，保证 横向 只会有不重复的元素，只会是重复元素中的第一个，
            continue                                        // &&(*used)[i-1]==false 处理深度遍历跳过问题，已经回溯回来
        }
        if (*used)[i] {
            continue
        }
        (*used)[i] = true
        *one = append(*one, nums[i])
        // 递归-纵向遍历
        backtrace(nums, used, one, ans)
        // 回溯
        *one = (*one)[0:len(*one)-1]
        (*used)[i] = false
    }
}

```
