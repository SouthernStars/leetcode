# 491.非递减子序列
[leetcode 链接](https://leetcode.cn/problems/non-decreasing-subsequences/description/)

### 一、题目描述

给你一个整数数组 nums ，找出并返回所有该数组中不同的递增子序列，递增子序列中 至少有两个元素 。你可以按 任意顺序 返回答案。

数组中可能含有重复元素，如出现两个整数相等，也可以视作递增序列的一种特殊情况。

### 三、代码实现

```go
func findSubsequences(nums []int) (ans [][]int) {
    one := []int {}
    backtrace(0, nums, &one, &ans)
    return ans
}

func backtrace(i int, nums []int, one *[]int, ans *[][]int) {
    // 结束条件，收集结果
    if i == len(nums) {
        if len(*one)>=2 {
            *ans = append(*ans, append([]int{}, (*one)...)) 
        }
        return 
    }
    if len(*one) == 0 || nums[i] >= (*one)[len(*one)-1] {
        *one = append(*one, nums[i])
        backtrace(i+1, nums, one, ans)
        *one = (*one)[0:len(*one)-1]
    }
    if len(*one) ==0  || nums[i] != (*one)[len(*one)-1] { // 新启一个
        backtrace(i+1, nums, one, ans)
    }
}
```
