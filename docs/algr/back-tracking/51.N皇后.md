# 51.N皇后
[leetcode 链接](https://leetcode.cn/problems/n-queens/description/)

### 一、题目描述

按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

n 皇后问题 研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 n ，返回所有不同的 n 皇后问题 的解决方案。

每一种解法包含一个不同的 n 皇后问题 的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位

输入：n = 4

输出：[[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]

### 二、解题思路

回溯：棋盘问题

### 三、代码实现

```go
func solveNQueens(n int) (ans [][]string) {
    one := [][]byte{}
    for i:=0; i<n; i++ {
        one = append(one, make([]byte, n))
    }
    for i:=0; i<n; i++ {
        for j:=0; j<n;j++ {
            one[i][j] = '.'
        }
    }
    col := make([]bool, n) // 记录列是否有皇后
    right := make([]bool, 2*n-1) // 记录主对角线是否有皇后
    left := make([]bool, 2*n-1) // 记录副对角线是否有皇后
    // one 记录棋盘
    backtrace(0, n, &col, &right, &left, &one, &ans)
    return ans
}

// i 表示第 i 行处理，每行选择一个位置
func backtrace(i int, n int, col *[]bool, right *[]bool, left *[]bool, one *[][]byte, ans *[][]string) {
    if i==n {   // 结束条件，收集结果
        ons := []string{}
        for k:=0; k<n; k++ {
            ons = append(ons, string((*one)[k]))
        }
        *ans = append(*ans, ons)
        return
    }
    // for 横向遍历
    for j:=0; j<n; j++ {
        // 对于坐标 (i, j) row i, col j, right n-1-(i-j) , left i+j,, i 由参数控制，每行
        if (*col)[j] || (*right)[n-1-(i-j)] || (*left)[i+j] {  // 存在皇后
            continue
        }
        (*one)[i][j] = 'Q'
        (*col)[j], (*right)[n-1-(i-j)], (*left)[i+j] = true, true, true
        // 递归-纵向遍历
        backtrace(i+1, n, col, right, left, one, ans)
        // 回溯
        (*one)[i][j] = '.'
        (*col)[j], (*right)[n-1-(i-j)], (*left)[i+j] = false, false, false
    }
}
```
