# 77.组合
[leetcode 链接](https://leetcode.cn/problems/combinations/description/)

### 一、题目描述

给定两个整数 n 和 k，返回范围 [1, n] 中所有可能的 k 个数的组合。

你可以按 任何顺序 返回答案。

### 二、解题思路

回溯：组合问题

### 三、代码实现

```go
func combine(n int, k int) [][]int {
    ans := [][]int{}
    arr := []int{}
    backtrace(0, n, k, &arr, &ans)
    return ans    
}

func backtrace(i int, n int, k int, arr *[]int, ans *[][]int,) {
    if len(*arr)==k {   // 终止条件, 收集结果
        decpy := make([]int, k)
        copy(decpy, *arr)
        *ans = append(*ans, decpy)
        return
    }
    // for-横向遍历
    for i:=i+1;i<=n;i++ {
        // 处理节点, 加入到数组中
        *arr = append(*arr, i)
        // 递归-纵向遍历
        backtrace(i, n, k, arr, ans)
        // 回溯-撤销结果
        *arr = (*arr)[0:len(*arr)-1]    
    }
}
```
