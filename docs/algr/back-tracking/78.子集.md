# 78.子集
[leetcode 链接](https://leetcode.cn/problems/subsets/description/)

### 一、题目描述

你一个整数数组 nums ，数组中的元素 互不相同 。返回该数组所有可能的子集 （幂集）。 解集 不能 包含重复的子集。你可以按 任意顺序 返回解集

### 二、解题思路

回溯：子集问题

### 三、代码实现

```go
func subsets(nums []int) (ans [][]int) {
    one := []int{}
    backtrace(0, nums, &one, &ans)
    return ans
}

func backtrace(i int, nums []int, one *[]int, ans *[][]int) {
    *ans = append(*ans, slices.Clone(*one)) // 收集结果
    if i == len(nums) { // 结束条件
        return
    }
    // for-横向遍历
    for j:=i; j<len(nums);j++ {
        // 处理节点
        // 加入当前节点
        *one = append(*one, nums[j])
        // 递归-纵向遍历
        backtrace(j+1, nums, one, ans)
        // 回溯
        *one = (*one)[0:len(*one)-1]
    }
}

// 1      2               3
// 2 3    3                
// 3 _    _               _  
```
