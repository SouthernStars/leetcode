# 90.子集II
[leetcode 链接](https://leetcode.cn/problems/subsets-ii/description/)

### 一、题目描述

给你一个整数数组 nums ，其中可能包含重复元素，请你返回该数组所有可能的子集 （幂集）。

解集 不能 包含重复的子集。返回的解集中，子集可以按 任意顺序 排列。

### 二、解题思路

回溯：子集问题

### 三、代码实现

```go
// 回溯 + 排序并且横向去重
func subsetsWithDup(nums []int) (ans [][]int) {
    one := []int{}
    sort.Ints(nums)
    backtrace(0, nums, &one, &ans)
    return ans
}
func backtrace(i int, nums []int, one *[]int, ans *[][]int) {
    // 收集结果
    *ans = append(*ans, slices.Clone(*one))
    if i == len(nums) { // 结束条件
        return
    }
    // for - 横向遍历
    for j:=i; j<len(nums); j++ {
        // 横向相同的元素可以跳，避免重复
        if j>i && nums[j]==nums[j-1] {
            continue
        }
        *one = append(*one, nums[j])
        backtrace(j+1, nums, one, ans)
        *one = (*one)[0:len(*one)-1]
    }
}
```
