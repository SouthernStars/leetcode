# 93.复原IP地址
[leetcode 链接](https://leetcode.cn/problems/restore-ip-addresses/description/)

### 一、题目描述

有效 IP 地址 正好由四个整数（每个整数位于 0 到 255 之间组成，且不能含有前导 0），整数之间用 '.' 分隔。

例如："0.1.2.201" 和 "192.168.1.1" 是 有效 IP 地址，但是 "0.011.255.245"、"192.168.1.312" 和 "192.168@1.1" 是 无效 IP 地址。
给定一个只包含数字的字符串 s ，用以表示一个 IP 地址，返回所有可能的有效 IP 地址，这些地址可以通过在 s 中插入 '.' 来形成。你 不能 重新排序或删除 s 中的任何数字。你可以按 任何 顺序返回答案。

### 二、解题思路

回溯：分割问题

### 三、代码实现

```go
func restoreIpAddresses(s string) (ans []string) {
    one := []string{}
    backtrace(0, s, &one, &ans)
    return ans
}

func backtrace(i int, s string, one *[]string, ans *[]string) {
    if len(*one) == 4 && i==len(s) { // 结束条件    
        *ans = append(*ans, strings.Join(*one, "."))
        return
    }
    // for - 横向遍历
    for j:=i+1; j<=i+3&&j<=len(s); j++ {
        // 处理节点
        num,_ := strconv.Atoi(s[i:j])
        // 
        if j>i+1 && s[i]=='0' {    // 处理 0 
            break
        }
        if num >=0 && num <=255 {
            numstr := strconv.Itoa(num)
            *one = append(*one, numstr)
            // 递归-纵向遍历
            backtrace(j, s, one, ans)
            *one = (*one)[0:len(*one)-1]
        }
    }
}
```
