# 509.斐波那契数
[leetcode 链接](https://leetcode.cn/problems/fibonacci-number/description/)

### 一、题目描述

斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：F(0) = 0，F(1) = 1 , F(n) = F(n - 1) + F(n - 2)，其中 n > 1 ; 给定 n ，请计算 F(n) 。

### 二、解题思路

动态规划
1. dp 数组及其下表含义： dp[i] 是第 i 个斐波那契数，size 为 n+1
2. 递推公式：`dp[0]=0, dp[1]=1, dp[2~n]=0, dp[n]=dp[n-1]+dp[n-2] n>1`
3. dp 数组遍历顺序：i 2=>n
4. 打印结果 dp[n]

### 三、代码实现

```go
func fib(n int) int {
    if n==0 || n==1 {
        return n
    }
    dp := make([]int, n+1)
    dp[0], dp[1] = 0, 1
    for i:=2; i<=n; i++ {
        dp[i] = dp[i-1] + dp[i-2]
    }
    return dp[n]
}
```

# 70.爬楼梯
[leetcode 链接](https://leetcode.cn/problems/climbing-stairs/description/)

### 一、题目描述

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

### 二、解题思路

动态规划：同斐波那契数
1. dp数组以及下标的含义：dp[i] 表示爬到 第 i 阶梯 有多少种不同的方法。size为 n+1
2. 递推公式：`dp[1]=1, dp[2]=2, dp[n] = dp[n-1]+dp[n-2]`
3. dp 数组的遍历顺序：3 => n
4. 输出结果 dp[n]

### 三、代码实现

```go
func climbStairs(n int) int {
    if n==1 || n==2 {
        return n
    }
    dp:=make([]int, n+1)
    dp[1], dp[2] = 1, 2
    for i:=3; i<=n; i++ {
        dp[i] = dp[i-1] + dp[i-2]
    }
    return dp[n]
}
```

# **746.使用最小花费爬楼梯**
[leetcode 链接](https://leetcode.cn/problems/min-cost-climbing-stairs/description/)

### 一、题目描述

给你一个整数数组 cost ，其中 cost[i] 是从楼梯第 i 个台阶向上爬需要支付的费用。一旦你支付此费用，即可选择向上爬一个或者两个台阶。你可以选择从下标为 0 或下标为 1 的台阶开始爬楼梯。 请你计算并返回达到楼梯顶部的最低花费。

### 二、解题思路

动态规划：
1. dp[i] 表示从头开始爬到 i 需要的最低费用， dp 数组初始化：dp[0], dp[1] = 0, 0, size为 n+1
2. 状态转移方程：dp[i] = min(dp[i-1]+cost[i-1], dp[i-2]+cost[i-2])
3. 遍历顺序 2 ==> n
4. 输出结果：dp[n]

### 三、代码实现

```go
func minCostClimbingStairs(cost []int) int {
    n := len(cost)
    dp := make([]int, n+1)
    // dp[i] 表示从头开始爬到 i 所需的最小花费
    // 状态转移方程 dp[i] = min(dp[i-1]+cost[i-1], dp[i-2]+cost[i-2])
    dp[0], dp[1] = 0, 0
    for i:=2; i<=n; i++ {
        dp[i] = min(dp[i-1]+cost[i-1], dp[i-2]+cost[i-2])
    }
    return dp[n]
} 
```
