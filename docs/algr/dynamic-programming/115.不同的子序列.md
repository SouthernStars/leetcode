# 115.不同的子序列
[leetcode 链接](https://leetcode.cn/problems/distinct-subsequences/description/)

### 一、题目描述

给你两个字符串 s 和 t ，统计并返回在 s 的 子序列 中 t 出现的个数，结果需要对 10^9 + 7 取模。


### 二、解题思路

动态规划：dp[i][j] 表示 s[0~i] t[0~j] 子序列出现的次数

状态转移方程：
dp[i][j] = dp[i-1][j-1] + dp[i-1][j]        s[i] = t[j]
            dp[i][j-1]                      s[i]!=t[j]

初始化 dp[1~n][0] = 0
        dp[0][*]=1

### 三、代码实现

```go
func numDistinct(s string, t string) int {
    m, n := len(s), len(t)
    dp := make([][]int, n+1)
    for i:=0; i<=n; i++ {
        dp[i]=make([]int, m+1)
    }
    for i:=0; i<=m; i++ {
        dp[0][i] = 1
    }
    for i:=1; i<=n; i++ {
        for j:=1; j<=m; j++ {
            if s[j-1] == t[i-1] {
                dp[i][j] = dp[i-1][j-1] + dp[i][j-1]
            } else {
                dp[i][j] = dp[i][j-1]
            }
        }
    }
    return dp[n][m]
}
```
