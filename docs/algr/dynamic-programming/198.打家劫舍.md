# 198.打家劫舍
[leetcode 链接](https://leetcode.cn/problems/house-robber/description/)

### 一、题目描述

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，
如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

给定一个代表每个房屋存放金额的非负整数数组，计算你 不触动警报装置的情况下 ，一夜之内能够偷窃到的最高金额。



### 二、解题思路

题解：dp[i] 表示前 i 间房屋能偷到的最大金额(i从0开始，包括0)

如果只有一间房屋，那偷窃这间房屋就是最大金额
如果有两间房屋，由于相邻两间房屋存在报警系统，那偷窃房屋中较高金额能获得最大金额

如果第 k >= 2 间房屋，则有两种情况，一种偷第 k 间，一种是不偷
如果偷，则最大金额为 dp[k-2] + money[k]
如果不偷，则最大金额为 dp[k-1]

故状态转移方程为

dp[0] = money[0]

dp[1] = max(money[0], money[1])

dp[k] = max(dp[k-1], dp[k-2]+money[k])      k>=2

### 三、代码实现

```go
func rob(nums []int) int {
    n := len(nums)
    if n==1 {
        return nums[0]
    }
    dp := make([]int, n)
    dp[0] = nums[0]
    dp[1] = max(nums[0], nums[1])

    for k:=2; k<n; k++ {
        dp[k] = max(dp[k-1], dp[k-2]+nums[k])
    }
    return dp[n-1]
}
```
