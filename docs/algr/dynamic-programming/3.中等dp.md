# **343.整数拆分**
[leetcode 链接](https://leetcode.cn/problems/integer-break/description/)

### 一、题目描述

给定一个正整数 n ，将其拆分为 k 个 正整数 的和（ k >= 2 ），并使这些整数的乘积最大化。

返回 你可以获得的最大乘积 。

### 二、解题思路

动态规划：一个正整数 n, 可以拆分成 i 和 n-i, 正整数拆分出的整数的乘积 依赖拆分出的整数。

dp[i] 表示正整数 i 对应的最大乘积。dp[0] = 0, dp[1] = 0

状态转移方程： i 可以拆成 j 和 i-j, 出现两种情况
- 若i-j不继续拆分 ，乘积为 j*(i-j)
- 若i-j继续拆分，乘积位 j*dp[i-j]
  因此状态转移方程 dp[i] = max(j*(i-j), j*dp[i-j]); 遍历所有可能的 j 确定 dp[i]

dp 遍历顺序 2->n

### 三、代码实现

```go
func integerBreak(n int) int {
    dp := make([]int, n+1)
    dp[0],dp[1] = 0,0
    for i:=2; i<=n; i++ {
        dpi := 0
        for j:=1; j<i; j++ {
            dpi = max(dpi, max(j*(i-j), j*dp[i-j]))
        }
        dp[i] = dpi
    }
    return dp[n]
}
```

# **96.不同的二叉搜索树**
[leetcode 链接](https://leetcode.cn/problems/unique-binary-search-trees/description/)

### 一、题目描述

给你一个整数 n ，求恰由 n 个节点组成且节点值从 1 到 n 互不相同的 二叉搜索树 有多少种？返回满足题意的二叉搜索树的种数。

### 二、解题思路

动态规划：
// 动态规划：
// dp[n] 表示由 n 个节点组成的1~n个互不相同的二叉搜索树的个数
// f(i,n) 表示以 i 为根节点，总结点为n 的不同的二叉搜索树的个数
// 状态转移方程：
// f(i,n) = dp[i-1] * dp[n-i]
// dp[n] = sum{i 1=>n} f(i, n)
// => dp[n] = sum{i 1=>n} dp[i-1] * dp[n-i]
// dp[0] 表示空树， = 1，  dp[1] = 1， 只有根节点
// dp 数组遍历顺序 1=>n

### 三、代码实现

```go
func numTrees(n int) int {
    dp:=make([]int, n+1)
    dp[0],dp[1] = 1,1
    for i:=2; i<=n; i++ {
        for j:=1; j<=i; j++ {
            dp[i] += dp[j-1]*dp[i-j]
        } 
    }
    return dp[n]
}
```


# **64.最小路径和**
[leetcode 链接](https://leetcode.cn/problems/minimum-path-sum/description/)

### 一、题目描述

给定一个包含非负整数的 m x n 网格 grid ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。 说明：每次只能向下或者向右移动一步。

### 二、解题思路

动态规划：
- dp[i][j] 表示从头到 （i,j）的最小路径和，dp size m*n
- 初始化第一行和第一列
- 状态转移方程：dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])
- 遍历循序 i: 1=>m-1, j:=>1:n-1
- 结果 dp[m-1][n-1]

### 三、代码实现

```go
func minPathSum(grid [][]int) int {
    m, n := len(grid), len(grid[0])
    dp := make([][]int, m)
    for i:=0; i<m; i++ {
        one:=make([]int, n)
        dp[i] = one
    }
    dp[0][0] = grid[0][0]
        for i:=1;i<n; i++ {
        dp[0][i] = grid[0][i] + dp[0][i-1]
    }
    for i:=1; i<m; i++ {
        dp[i][0] = grid[i][0] + dp[i-1][0]
    }
    for i:=1; i<m; i++ {
        for j:=1; j<n; j++ {
            dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])
        }
    }
    return dp[m-1][n-1]
}
```

# **152.乘积最大子数组**
[leetcode 链接](https://leetcode.cn/problems/maximum-product-subarray/description/)

### 一、题目描述

给你一个整数数组 nums ，请你找出数组中乘积最大的非空连续子数组 （该子数组中至少包含一个数字），并返回该子数组所对应的乘积。 测试用例的答案是一个 32-位 整数。

### 二、解题思路

动态规划：
- dpmax[i] 表示 以 i 为结束位置的子数组的最大值， dpmin[i] 表示以 i 位置为结束位置的子树组的最小值（用于处理nums[i]负数的问题）
- 状态转移方程：dpmax[i] = max(nums[i]*dpmax[i], nums[i]*dpmin[i], nums[i])  dpmin[i] = min(nums[i]*dpmax[i], nums[i]*dpmin[i], nums[i])
- dp遍历顺序 i 1=>n, dpmax[0]=dpmin[0]=nums[0]

### 三、代码实现

```go
func maxProduct(nums []int) int {
    n := len(nums)
    dpmax := make([]int, n)
    dpmin := make([]int, n)
    dpmax[0], dpmin[0] = nums[0], nums[0]
    max_prod := dpmax[0]
    for i:=1; i<n; i++ {
        dpmax[i] = max(nums[i], max(dpmax[i-1]*nums[i], dpmin[i-1]*nums[i]))
        dpmin[i] = min(nums[i], min(dpmax[i-1]*nums[i], dpmin[i-1]*nums[i]))
        max_prod = max(max_prod, dpmax[i])
    }
    return max_prod
}
```

# **221.最大正方形**
[leetcode 链接](https://leetcode.cn/problems/maximal-square/description/)

### 一、题目描述

在一个由 '0' 和 '1' 组成的二维矩阵内，找到只包含 '1' 的最大正方形，并返回其面积。

### 二、解题思路

动态规划：
- dp[i][j] 表示以 （i,j）为右下角的正方形的最大边长，size m*n
- 初始化第一行和第一列，按照 dp[i][j] 的定义，即dp[i][j] 为 matrix[i][j]
- 状态转移方程 dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])  条件 matrix[i][j]==1， 如果 matrix[i][j]==0，则dp[i][j]=0
- dp[i][j] 依赖 上方，左方，左上方三个位置
- 遍历顺序 i 1=>m-1 j:1=>n-1
- 最长的编程需要在遍历中记录

### 三、代码实现

```go
func maximalSquare(matrix [][]byte) int {
    m,n := len(matrix), len(matrix[0])
    dp := make([][]int, m)
    for i:=0; i<m; i++ {
        one := make([]int, n)
        dp[i] = one
    }
    edge_max := 0
    for i:=0; i<n; i++ {
        dp[m-1][i] = int(matrix[m-1][i]-'0')
        edge_max = max(edge_max, dp[m-1][i])
    }
    for i:=0; i<m; i++ {
        dp[i][n-1] = int(matrix[i][n-1]-'0')
        edge_max = max(edge_max, dp[i][n-1])
    }
    
    for i:=m-2; i>=0; i-- {
        for j:=n-2; j>=0; j-- {
            if matrix[i][j] == '1' {
                dp[i][j] = int(matrix[i][j]-'0') + min(dp[i+1][j], min(dp[i][j+1], dp[i+1][j+1]))
                edge_max = max(edge_max, dp[i][j])
            }
        }
    }
    return edge_max * edge_max
}
```
