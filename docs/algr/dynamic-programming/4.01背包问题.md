# **416.分割等和子集**
[leetcode 链接](https://leetcode.cn/problems/partition-equal-subset-sum/description/)

### 一、题目描述

给你一个 只包含正整数 的 非空 数组 nums 。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

### 二、解题思路

0-1背包问题：给定一个只包含正整数的非空数组 nums，判断是否可以从数组中选出一些数字，使得这些数字的和等于整个数组的元素和的一半。

根据数组的长度 n 判断数组是否可以被划分。如果 n<2，或则 sum 是奇数，则不可能将数组分割成元素和相等的两个子集，因此直接返回 false。 如果 sum 是偶数，则背包容量 target= sum/2，使用动态规划，target 表示背包容量

动态规划：
- dp[i][j] 表示在背包容量为 j 在 前 i 个物品中选取的最大容量 size n*(target+1)
- 递推方程 dp[i][j] =  dp[i-1][j]   j<nums[i]   ||   max(dp[i-1][j], dp[i-1][j-nums[i]]+nums[i])          j >= num[i]
- 初始化：dp 第 0 行, 比较 num[0] 和 背包的大小
- 结果 dp[n-1][target] == target

### 三、代码实现

```go
func canPartition(nums []int) bool {
    n := len(nums)
    sum := 0
    for _, e := range nums {
        sum += e
    }
    if sum % 2 ==1 || n<2 {
        return false
    }
    target := sum / 2
    dp := make([][]int, n)
    for i:=0; i<n; i++ {
        dp[i] = make([]int, target+1)
    }
    // 初始化数组
    for i:=0; i<=target;i++ {
        if i>=nums[0] {
            dp[0][i] = nums[0]
        }
    }
    // 遍历 物品  - 遍历背包
    for i:=1; i<n; i++ {
        for j:=1; j<=target; j++ {
            if j<nums[i] {
                dp[i][j] = dp[i-1][j]
            } else {
                dp[i][j] =  max(dp[i-1][j], dp[i-1][j-nums[i]]+nums[i])
            }
        }
    }
    return dp[n-1][target]==target
}
```


# **494.目标和**
[leetcode 链接](https://leetcode.cn/problems/target-sum/description/)

### 一、题目描述

给你一个非负整数数组 nums 和一个整数 target。向数组中的每个整数前添加 '+' 或 '-' ，然后串联起所有整数，可以构造一个 表达式 ：
例如，nums = [2, 1] ，可以在 2 之前添加 '+' ，在 1 之前添加 '-' ，然后串联起来得到表达式 "+2-1"。 返回可以通过上述方法构造的、运算结果等于 target 的不同表达式的数目。

### 二、解题思路

0-1背包问题：假设nums 数组元素和为 sum, 取正号元素的和为 zh，那 zh-(sum-zh) = target => zh = (sum+target)/2 ；如果 sum + target 是奇数，没有方案，返回 0
问题转化为 在数组中选择若干数其和为 zh 的方案数,

0-1背包问题，背包容量为 zh
- dp[i][j] 表示在0~i上选物品 和 为 j 的方案数
    - 状态转移方程：
        - dp[i][j] = dp[i-1][j]   j<nums[i]
        - dp[i-1][j]+dp[i-1][j-nums[i]])       j>=nums[i]

dp 数组初始化
第 0 行 dp[0][0]=1, dp[0][nums[0]] =1
第 0 列 dp[~][0] 根据 nums[0~i] 中 0 的数量 2^k 此次方

dp 数组遍历顺序：先行后列

结果：dp[n-1][zh]


### 三、代码实现

```go
func findTargetSumWays(nums []int, target int) int {
    n := len(nums)
    sum := 0
    for i:=0; i<n; i++ {
        sum += nums[i]
    }
    if (sum + target) % 2 != 0 || (sum + target) < 0 {
        return 0
    } 

    zh := (sum + target) / 2
    // neg+1 为背包容量， 转化为 0-1 背包问题
    dp := make([][]int, n)
    for i:=0; i<n; i++ {
        dp[i] = make([]int, zh+1)
    }
    // dp 数组初始化
    dp[0][0]=1
    if nums[0]<=zh {
        dp[0][nums[0]] = 1
    }
    numsZero := 0
    for i:=0; i<n; i++ {
        if nums[i] == 0 {
            numsZero ++
        }
        dp[i][0] = int(math.Pow(2.0, float64(numsZero)))
    }
    for i:=1;i<n;i++ {
        for j:=1;j<=zh; j++ {
            if j<nums[i] {
                dp[i][j] = dp[i-1][j]
            } else {
                dp[i][j] = dp[i-1][j] + dp[i-1][j-nums[i]]
            }
        }
    }
    return dp[n-1][zh]
}
```

# **1049.最后一块石头的重量II**
[leetcode 链接](https://leetcode.cn/problems/last-stone-weight-ii/description/)

### 一、题目描述

有一堆石头，用整数数组 stones 表示。其中 stones[i] 表示第 i 块石头的重量。每一回合，从中选出任意两块石头，然后将它们一起粉碎。假设石头的重量分别为 x 和 y，且 x <= y。那么粉碎的可能结果如下：
如果 x == y，那么两块石头都会被完全粉碎； 如果 x != y，那么重量为 x 的石头将会完全粉碎，而重量为 y 的石头新重量为 y-x。 最后，最多只会剩下一块 石头。返回此石头 最小的可能重量 。如果没有石头剩下，就返回 0。

### 二、解题思路

0-1背包问题

[2, 7,4,1, 8, 1] sum = 23 本题：尽可能的将石头分成重量相似的两堆，一个 11 ，一个 12，，结果就是 1

问题转化为，给一个 sum/2 的背包，从石头中选择石头尽可能装满容器。结果为 sum-2*dp[n][sum/2]

- 背包大小 target = sum/2     物品 stones[i]  0~n
- dp[i][j] 在背包为 j 的情况下最大的重量  size n*(target+1)
- 递推公式： dp[i][j] = dp[i-1][j]       j<stones[i]   || = max(dp[i-1][j], dp[i-1][j-stone[i]] + stone[i]) j>=stones[i]

dp 数组初始化：第一行 dp[0][i] 根据 stones[0] 进行初始化
第一列 dp[i][0] 初始化为 0

dp 遍历顺序：先行后列

结过： sum - 2*dp[n-1][target]

### 三、代码实现

```go
func lastStoneWeightII(stones []int) int {
    n:=len(stones)
    sum:=0
    for i:=0; i<n; i++ {
        sum += stones[i]
    }
    target := sum / 2
    dp := make([][]int, n)
    for i:=0; i<n; i++ {
        dp[i] = make([]int, target+1)
    }
    for i:=0; i<=target; i++ {
        if i>=stones[0] {
            dp[0][i] = stones[0]
        }
    }
    for i:=1; i<n; i++ {
        for j:=1;j<=target;j++ {
            if j<stones[i] {
                dp[i][j] = dp[i-1][j]
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i-1][j-stones[i]] + stones[i])
            }
        }
    }
    return sum - 2*dp[n-1][target]
}
```

# **474.一和零**
[leetcode 链接](https://leetcode.cn/problems/ones-and-zeroes/description/)

### 一、题目描述

给你一个二进制字符串数组 strs 和两个整数 m 和 n 。 请你找出并返回 strs 的最大子集的长度，该子集中 最多 有 m 个 0 和 n 个 1 。

### 二、解题思路

0-1背包问题：本题是三维背包问题，

dp[i][j][k] : 表示在 j个0和k个1 的条件下最多的字符串数量， 在数组 0~i 中， s*(m+1)*(n+1)

递推公式：dp[i][j][k] = dp[i-1][j][k]    cm(strs[i]) > j || cn(strs[i]) > k
max(dp[i-1][j][k], dp[i-1][j-cm][k-cn] + 1)
初始化：dp[~][0][~] = 0     dp[~][~][0] = 0     dp[0][~][~] = 根据 str[0] 初始化

遍历顺序：先行后列

### 三、代码实现

```go
func findMaxForm(strs []string, m int, n int) int {
    s := len(strs)
    dp := make([][][]int, s)
    for i:=0; i<s;i++ {
        dp[i] = make([][]int, m+1)
        for j:=0; j<=m; j++ {
            dp[i][j] = make([]int, n+1)
        }
    } 
    // 初始化
    for i:=0; i<=m; i++ {
        for j:=0; j<=n; j++ {
            cm, cn := count(strs[0])
            if i>=cm && j>=cn {
                dp[0][i][j] = 1
            }
        }
    }
    for i:=1; i<s; i++ {
        for j:=0; j<=m; j++ {
            for k:=0; k<=n; k++ {
                cm, cn := count(strs[i])
                if cm > j || cn > k {
                    dp[i][j][k] = dp[i-1][j][k]
                } else {
                    dp[i][j][k] = max(dp[i-1][j][k], dp[i-1][j-cm][k-cn] + 1)
                }
            }
        }
    }
    return dp[s-1][m][n]
}
func count(str string) (m, n int) {
    m, n = 0, 0
    for _, e := range str {
        if e == '0' {
            m ++
        } else {
            n ++
        }
    }
    return m, n
}
```
