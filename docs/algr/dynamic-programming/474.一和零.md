# 474.一和零
[leetcode 链接](https://leetcode.cn/problems/ones-and-zeroes/description/)

### 一、题目描述

给你一个二进制字符串数组 strs 和两个整数 m 和 n 。

请你找出并返回 strs 的最大子集的长度，该子集中 最多 有 m 个 0 和 n 个 1 。


### 二、解题思路

0-1背包问题

本题是三维背包问题，

dp[i][j][k] : 表示在 j个0和k个1 的条件下最多的字符串数量

递推公式：dp[i][j][k] = dp[i-1][j][k]    cm(strs[i]) > j || cn(strs[i]) > k
                      max(dp[i-1][j][k], dp[i-1][j-cm][k-cn] + 1)
初始化：dp[~][0][~] = 0     dp[~][~][0] = 0     dp[0][~][~] = 根据 str[0] 初始化

遍历顺序：先行后列

### 三、代码实现

```go
func findMaxForm(strs []string, m int, n int) int {
    s := len(strs)
    dp := make([][][]int, s)
    for i:=0; i<s;i++ {
        dp[i] = make([][]int, m+1)
        for j:=0; j<=m; j++ {
            dp[i][j] = make([]int, n+1)
        }
    } 
    // 初始化
    for i:=0; i<=m; i++ {
        for j:=0; j<=n; j++ {
            cm, cn := count(strs[0])
            if i>=cm && j>=cn {
                dp[0][i][j] = 1
            }
        }
    }
    for i:=1; i<s; i++ {
        for j:=0; j<=m; j++ {
            for k:=0; k<=n; k++ {
                cm, cn := count(strs[i])
                if cm > j || cn > k {
                    dp[i][j][k] = dp[i-1][j][k]
                } else {
                    dp[i][j][k] = max(dp[i-1][j][k], dp[i-1][j-cm][k-cn] + 1)
                }
            }
        }
    }
    return dp[s-1][m][n]
}
func count(str string) (m, n int) {
    m, n = 0, 0
    for _, e := range str {
        if e == '0' {
            m ++
        } else {
            n ++
        }
    }
    return m, n
}
```
