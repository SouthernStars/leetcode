# 494.目标和
[leetcode 链接](https://leetcode.cn/problems/target-sum/description/)

### 一、题目描述

给你一个非负整数数组 nums 和一个整数 target。向数组中的每个整数前添加 '+' 或 '-' ，然后串联起所有整数，可以构造一个 表达式 ：
例如，nums = [2, 1] ，可以在 2 之前添加 '+' ，在 1 之前添加 '-' ，然后串联起来得到表达式 "+2-1"。 返回可以通过上述方法构造的、运算结果等于 target 的不同表达式的数目。

### 二、解题思路

0-1背包问题

假设nums 数组元素和为 sum, 取正号元素的和为 zh，那 zh-(sum-zh) = target => zh = (sum+target)/2

如果 sum + target 是奇数，没有方案，返回 0

问题转化为 在数组中选择若干数其和为 zh 的方案数, 

0-1背包问题，背包容量为 zh

dp[i][j] 表示在0~i上选物品 和 为 j 的方案数

状态转移方程：

dp[i][j] = dp[i-1][j]   j<nums[i]
            dp[i-1][j]+dp[i-1][j-nums[i]])       j>=nums[i]

dp 数组初始化
    第 0 行 dp[0][0]=1, dp[0][nums[0]] =1
    第 0 列 dp[~][0] 根据 nums[0~i] 中 0 的数量 2^k 此次方

dp 数组遍历顺序：先行后列


### 三、代码实现

```go
func findTargetSumWays(nums []int, target int) int {
    n := len(nums)
    sum := 0
    for i:=0; i<n; i++ {
        sum += nums[i]
    }
    if (sum + target) % 2 != 0 || (sum + target) < 0 {
        return 0
    } 

    zh := (sum + target) / 2
    // neg+1 为背包容量， 转化为 0-1 背包问题
    dp := make([][]int, n)
    for i:=0; i<n; i++ {
        dp[i] = make([]int, zh+1)
    }
    // dp 数组初始化
    dp[0][0]=1
    if nums[0]<=zh {
        dp[0][nums[0]] = 1
    }
    numsZero := 0
    for i:=0; i<n; i++ {
        if nums[i] == 0 {
            numsZero ++
        }
        dp[i][0] = int(math.Pow(2.0, float64(numsZero)))
    }
    for i:=1;i<n;i++ {
        for j:=1;j<=zh; j++ {
            if j<nums[i] {
                dp[i][j] = dp[i-1][j]
            } else {
                dp[i][j] = dp[i-1][j] + dp[i-1][j-nums[i]]
            }
        }
    }
    return dp[n-1][zh]
}
```
