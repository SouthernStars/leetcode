# 494.目标和
[leetcode 链接](https://leetcode.cn/problems/target-sum/description/)

### 一、题目描述

给你一个非负整数数组 nums 和一个整数 target 。

向数组中的每个整数前添加 '+' 或 '-' ，然后串联起所有整数，可以构造一个 表达式 ：

例如，nums = [2, 1] ，可以在 2 之前添加 '+' ，在 1 之前添加 '-' ，然后串联起来得到表达式 "+2-1" 。
返回可以通过上述方法构造的、运算结果等于 target 的不同 表达式 的数目。

### 二、解题思路

0-1背包问题

假设nums 数组元素和为 sum, 取负号元素的和为 neg，那 (sum-neg)-neg = target => neg = (sum-target)/2

如果 sum < target ，没有方案，返回 0

问题转化为 在数组中选择若干数 是的 其和为 neg=（sum-target）/2, 即 0-1背包问题，背包容量为 neg=（sum-target）/2

dp[i][j] 表示在前i个数字选择任意数目的数字，其和为j的方案

状态转移方程：

dp[i+1][j] = dp[i][j] + dp[i][j-nums[i]]   j>=nums[i]
         =  dp[i][j]                         j<nums[i]

dp 数组初始化
    第 0 行 dp[0][*] dp[0][0]=1, dp[0][1~neg] = 0

dp 数组遍历顺序：先行后列


### 三、代码实现

```go
func findTargetSumWays(nums []int, target int) int {
    n := len(nums)
    sum := 0
    for i:=0; i<n; i++ {
        sum += nums[i]
    }
    
    if sum < target {
        return 0
    }
    neg := (sum - target) / 2
    // neg+1 为背包容量， 转化为 0-1 背包问题
    dp := make([][]int, n+1)
    for i:=0; i<=n; i++ {
        dp[i] = make([]int, neg+1)
    }
    // dp 数组初始化
    dp[0][0]=1
    for i:=0;i<n;i++ {
        for j:=0;j<=neg; j++ {
            if j<nums[i] {
                dp[i+1][j] = dp[i][j]
            } else {
                dp[i+1][j] = dp[i][j] + dp[i][j-nums[i]]
            }
        }
    }
    return dp[n][neg]
}
```
