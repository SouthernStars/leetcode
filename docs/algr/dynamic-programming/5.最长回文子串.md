# 5.最长回文子串
[leetcode 链接](https://leetcode.cn/problems/longest-palindromic-substring/description/)

### 一、题目描述

给你一个字符串 s，找到 s 中最长的 回文 子串 。


### 二、解题思路

穷举法: 从大到小 遍历 size , 判断是否是回文

动态规划：dp[i][j] 表示 s[i]~s[j] 是否是回文

由于 dp[i][j] 取决于 dp[i+1][j-1] 因此 i 需要倒序， j 需要正序遍历

初始化 dp[i][i] = 1
状态转移方程
dp[i][j] =   s[i] == s[j]                   j=i+1 只有两个字符
=   s[i]==s[j] && dp[i+1][j-1]

动态规划：判断是否是回文子串过程中，记录串的最大长度和边界


### 三、代码实现

```go
func longestPalindrome(s string) string {
    n := len(s) 
    dp:=make([][]bool, n)
    for i:=0;i<n;i++ {
        dp[i] = make([]bool, n)
        dp[i][i]=true
    }
    max_len, left, right := 1, 0, 1
    for i:=n-1;i>=0;i-- {
        for j:=i+1;j<n;j++ {
            if j==i+1 {
                dp[i][j] = s[i]==s[j]
            } else {
                dp[i][j] = s[i]==s[j] && dp[i+1][j-1]
            }
            if dp[i][j] && (j+1-i > max_len) {
                max_len = j+1-i
                left, right = i, j+1
            }
        }
    }
    return s[left:right]
}
```


```go
func longestPalindrome(s string) string {
    for size:=len(s); size>=0; size-- {
        for i:=0; i<=len(s)-size; i++ {
            if (isHuiwen(s[i:i+size])) {
                return s[i:i+size]
            }
        }
    }
    return ""
}

func isHuiwen(s string) bool {    
    for low, high := 0, len(s)-1; low <= high; low, high = low+1, high-1 {
        if s[low] != s[high] {
            return false
        }
    }
    return true
}
```
