# 62.不同路径
[leetcode 链接](https://leetcode.cn/problems/unique-paths/description/)

### 一、题目描述

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为 “Start” ）。 机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

### 二、解题思路

动态规划：

dp[i][j] 表示在 i行j列的机器人到达 右下角有几条不同的路径；初始化d[m-1][n-1]=0, d[m-1][*]=1, d[*][n-1]=1

状态转移方程： d[i][j] = d[i][j+1] + d[i+1][j]  i< m-1; j<n-1

dp 数组遍历顺序： i: m-2==> 0 ; j: n-2 ==> 0

dp 数组打印

### 三、代码实现

```go
func uniquePaths(m int, n int) int {
    dp := make([][]int, m)
    for i:=0; i<m; i++ {
        one:= make([]int, n)
        if i==m-1 {
            for j:=0; j<=n-1; j++ {
                one[j] = 1
            }
        } else {
            one[n-1] = 1
        }
        dp[i] =  one
    }
    for i:=m-2; i>=0; i-- {
        for j:=n-2; j>=0; j-- {
            dp[i][j] = dp[i][j+1] + dp[i+1][j]
        }
    }
    return dp[0][0]
}
```
