# 63.不同路径II
[leetcode 链接](https://leetcode.cn/problems/unique-paths-ii/description/)

### 一、题目描述

给定一个 m x n 的整数数组 grid。一个机器人初始位于 左上角（即 grid[0][0]）。机器人尝试移动到 右下角（即 grid[m - 1][n - 1]）。机器人每次只能向下或者向右移动一步。

网格中的障碍物和空位置分别用 1 和 0 来表示。机器人的移动路径中不能包含 任何 有障碍物的方格。

返回机器人能够到达右下角的不同路径数量。

测试用例保证答案小于等于 2 * 109。


### 二、解题思路

动态规划：

dp[i][j] 表示 i 行 j 列 到达右下角的不同路径的数量。初始化 第 n-1 列，和第m-1行；根据障碍物赋值，遇到障碍物就 break

状态转移方程 dp[i][j] = dp[i+1][j] + dp[i][j+1]；[i][j] 不是无障碍，如果师障碍物 dp[i][j]=0

dp 遍历顺序 i m-2 =>0, j n-2 => 0

打印 dp 数组

### 三、代码实现

```

func uniquePathsWithObstacles(obstacleGrid [][]int) int {
    m, n := len(obstacleGrid), len(obstacleGrid[0])
    dp := make([][]int, m)
    for i:=0; i<m; i++ {
        one := make([]int, n)
        dp[i] = one
    }
    for i:=n-1;i>=0;i-- {
        if obstacleGrid[m-1][i] == 0 {
            dp[m-1][i] = 1
        } else {
            break
        }
    }
    for i:=m-1;i>=0;i-- {
        if obstacleGrid[i][n-1] == 0 {
            dp[i][n-1] = 1
        } else {
            break
        }
    }
    for i:=m-2; i>=0; i-- {
        for j:=n-2; j>=0; j-- {
            if obstacleGrid[i][j] == 0 {
                dp[i][j] = dp[i+1][j] + dp[i][j+1] 
            } else {
                dp[i][j] = 0
            }   
        }    
    }
    return dp[0][0]
}
```
