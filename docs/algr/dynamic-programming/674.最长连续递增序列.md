# 674.最长连续递增序列
[leetcode 链接](https://leetcode.cn/problems/longest-continuous-increasing-subsequence/description/)

### 一、题目描述

给定一个未经排序的整数数组，找到最长且 连续递增的子序列，并返回该序列的长度。

连续递增的子序列 可以由两个下标 l 和 r（l < r）确定，如果对于每个 l <= i < r，都有 nums[i] < nums[i + 1] ，
那么子序列 [nums[l], nums[l + 1], ..., nums[r - 1], nums[r]] 就是连续递增子序列。



### 二、解题思路

动态规划

dp[i] 表示 0~i 连续递增的序列长度

初始化 dp[0] = 1

dp[i] = dp[i-1]+1    num[i]>num[i-1]
        =  1        else

### 三、代码实现

```go
func findLengthOfLCIS(nums []int) int {
    n:=len(nums)
    dp := make([]int, n)
    dp[0] =1
    ans := 1
    for i:=1; i<n; i++ {
        if nums[i] > nums[i-1] {
            dp[i] = dp[i-1] + 1
        } else {
            dp[i] = 1
        }
        if ans < dp[i] {
            ans = dp[i]
        }
    }
    return ans
}
```
