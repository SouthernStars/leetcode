# 1143.最长公共子序列
[leetcode 链接](https://leetcode.cn/problems/longest-common-subsequence/description/)

### 一、题目描述

给定两个字符串 text1 和 text2，返回这两个字符串的最长 公共子序列 的长度。如果不存在 公共子序列 ，返回 0 。
一个字符串的 子序列 是指这样一个新的字符串：它是由原字符串在不改变字符的相对顺序的情况下删除某些字符（也可以不删除任何字符）后组成的新字符串。
例如，"ace" 是 "abcde" 的子序列，但 "aec" 不是 "abcde" 的子序列。两个字符串的 公共子序列 是这两个字符串所共同拥有的子序列。

### 二、解题思路

动态规划：最长公共子序列问题是典型的二维动态规划问题。
- 假设字符串 text1 和 text2 的长度分别为 m 和 n，创建 m+1 行 n+1 列的二维数组 dp，其中 dp[i][j] 表示 text1[0:i] 和 text2[0:j] 的最长公共子序列的长度。
- 初始化： dp[0][*]=0，dp[*][0]=0, 即第一行第一列均为 0
- 状态转移方程：
```
dp[i][j]    =   dp[i-1][j-1] + 1        text1[i−1]=text2[j−1]
            =   max(dp[i-1][j], dp[i][j-1])   text1[i−1]!=text2[j−1]
```

- 遍历顺序：先行后列 i：1=>m; j:1=>n
- 最终结果：dp[m][n]

### 三、代码实现

```go
func longestCommonSubsequence(text1 string, text2 string) int {
    m, n := len(text1), len(text2)
    dp := make([][]int, m+1)
    for i:=0; i<=m; i++ {
        dp[i] = make([]int, n+1)
    }
    for i:=1; i<=m; i++ {
        for j:=1; j<=n; j++ {
            if text1[i-1] == text2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]
}
```

# 392.判断子序列
[leetcode 链接](https://leetcode.cn/problems/is-subsequence/description/)

### 一、题目描述

给定字符串 s 和 t ，判断 s 是否为 t 的子序列。字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。（例如，"ace"是"abcde"的一个子序列，而"aec"不是）。

### 二、解题思路

动态规划：同最长公共子序列问题。即求字符串 s 和 t 的最长子序列长度
结果 return dp[m][n]==m

### 三、代码实现

```go
func isSubsequence(s string, t string) bool {
    m, n := len(s), len(t)
    dp := make([][]int, m+1)
    for i:=0; i<=m; i++ {
        dp[i] = make([]int, n+1)
    }
    for i:=1; i<=m; i++ {
        for j:=1; j<=n; j++ {
            if s[i-1] == t[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]==m
}
```

# 1035.不相交的线（同最长公共子序列问题）
[leetcode 链接](https://leetcode.cn/problems/uncrossed-lines/description/)

### 一、题目描述

在两条独立的水平线上按给定的顺序写下 nums1 和 nums2 中的整数。现在，可以绘制一些连接两个数字 nums1[i] 和 nums2[j] 的直线，这些直线需要同时满足： nums1[i] == nums2[j]
且绘制的直线不与任何其他连线（非水平线）相交。 请注意，连线即使在端点也不能相交：每个数字只能属于一条连线。以这种方法绘制线条，并返回可以绘制的最大连线数。

- 初始化： dp[0][*]=0，dp[*][0]=0, 即第一行第一列均为 0

- 遍历顺序：先行后列 i：1=>m; j:1=>n
- 最终结果：dp[m][n]

### 三、代码实现

```go
func maxUncrossedLines(nums1 []int, nums2 []int) int {

```

> 对于不连续的序列一般是 dp[i][j] 表示 text1[0:i] 和 text2[0:j] 的最长公共子序列的长度。
> 对于连续的序列一般是 dp[i][jh] 表示 以 nums[i] nums[j] 为结尾的的长度，dp[][] 中最大值即为最长重复子数组的长度

# **718.最长重复子数组**
[leetcode 链接](https://leetcode.cn/problems/maximum-length-of-repeated-subarray/description/)

### 一、题目描述

给两个整数数组 nums1 和 nums2 ，返回 两个数组中 公共的 、长度最长的连续子数组的长度 。

### 二、解题思路

动态规划:
- dp[i][j] 表示以 nums[i] nums[j] 为结尾的的长度，dp[][] 中最大值即为最长重复子数组的长度
- 初始化 全0，遍历循序 i:=1 => m  j:=1 => n
- 状态转移方程：dp[i][j] = dp[i-1][j-1]+1  nums1[i-1] == nums2[j-1]
- 结果：需要在计算中记录最大值

### 三、代码实现

```go
func findLength(nums1 []int, nums2 []int) int {
    m, n := len(nums1), len(nums2)
    dp := make([][]int, m+1)
    for i:=0; i<=m; i++ {
        dp[i] = make([]int, n+1)
    }
    res := 0
    for i:=1; i<=m; i++ {
        for j:=1; j<=n; j++ {
            if nums1[i-1] == nums2[j-1] {
                dp[i][j] = dp[i-1][j-1]+1
                res = max(res, dp[i][j])
            }
        }
    }
    return res
}
```

# **115.不同的子序列**
[leetcode 链接](https://leetcode.cn/problems/distinct-subsequences/description/)

### 一、题目描述

给你两个字符串 s 和 t ，统计并返回在 s 的 子序列 中 t 出现的个数。

### 二、解题思路

动态规划：dp[i][j] 表示 s[0~i] t[0~j] 子序列出现的次数

状态转移方程：
```
dp[i][j] = dp[i-1][j-1] + dp[i-1][j]        s[i] = t[j]    完全背包
            dp[i][j-1]                      s[i]!=t[j]
```

- 初始化 dp[*][0] = 0       dp[0][*]=1
- 结果：dp[n][m]

### 三、代码实现

```go
func numDistinct(s string, t string) int {
    m, n := len(s), len(t)
    dp := make([][]int, n+1)
    for i:=0; i<=n; i++ {
        dp[i]=make([]int, m+1)
    }
    for i:=0; i<=m; i++ {
        dp[0][i] = 1
    }
    for i:=1; i<=n; i++ {
        for j:=1; j<=m; j++ {
            if s[j-1] == t[i-1] {
                dp[i][j] = dp[i-1][j-1] + dp[i][j-1]
            } else {
                dp[i][j] = dp[i][j-1]
            }
        }
    }
    return dp[n][m]
}
```

# **583.两个字符串的删除操作**
[leetcode 链接](https://leetcode.cn/problems/delete-operation-for-two-strings/description/)

### 一、题目描述

给定两个单词 word1 和 word2 ，返回使得 word1 和  word2 相同所需的最小步数。每步可以删除任意一个字符串中的一个字符。

### 二、解题思路

动态规划：求两个字符串的最长公共序列长度 s

假设字符串 text1 和 text2 的长度分别为 m 和 n，创建 m+1 行 n+1 列的二维数组 dp，其中

- dp[i][j] 表示 text1[0:i] 和 text2[0:j] 的最长公共子序列的长度。
- 始化： dp[0][*]=0，dp[*][0]=0
- 状态转移方程：
```
dp[i][j]    =   dp[i-1][j-1] + 1        text1[i−1]=text2[j−1]
            =   max(dp[i-1][j], dp[i][j-1])   text1[i−1]!=text2[j−1]
```

- 结果: 是 len(word1)-s + len(word2)-s

### 三、代码实现

```go
func minDistance(word1 string, word2 string) int {
    m, n := len(word1), len(word2)
    dp := make([][]int, m+1)
    for i:=0; i<=m; i++ {
        dp[i] = make([]int, n+1)
    }
    for i:=1; i<=m; i++ {
        for j:=1; j<=n; j++ {
            if word1[i-1] == word2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return m+n-2*dp[m][n]
}
```

# **72.编辑距离**
[leetcode 链接](https://leetcode.cn/problems/edit-distance/description/)

### 一、题目描述

给你两个单词 word1 和 word2， 请返回将 word1 转换成 word2 所使用的最少操作数。 你可以对一个单词进行如下三种操作：插入一个字符; 删除一个字符;替换一个字符。

### 二、解题思路

动态规划：
- dp[i][j] 表示将 w1[0~i] 变为 w2[0~j] 需要的最小步数
- dp 初始化   dp[0][*] = 0,1,2,3 ..    dp[*][0] = 0,1,2,3 ...
- 状态转移方程
```
dp[i][j]       =  dp[i-1][j-1]          w1[i] == w[j]
               = min (*)                w1[i] != w[j]
                    插入对应 dp[i][j-1]+1
                    删除对应 dp[i-1][j]+1
                    替换对应 dp[i-1][j-1]+1
```
- 返回值：dp[m][n]

### 三、代码实现

```go
func minDistance(word1 string, word2 string) int {
    m, n := len(word1), len(word2)
    dp := make([][]int, m+1)
    for i:=0;i <=m;i++ {
        dp[i] = make([]int, n+1)
    }
    for i:=0; i<=m; i++ {
        dp[i][0]=i
    } 
    for i:=0;i<=n;i++ {
        dp[0][i]=i
    }

    for i:=1;i<=m;i++ {
        for j:=1;j<=n;j++ {
            if word1[i-1]!=word2[j-1] {
                dp[i][j] = min(dp[i][j-1], dp[i-1][j], dp[i-1][j-1])+1 
            } else {
				dp[i][j] = dp[i-1][j-1]
            }
        }
    }
    return dp[m][n]
}
```
