# 70.爬楼梯
[leetcode 链接](https://leetcode.cn/problems/climbing-stairs/description/)

### 一、题目描述

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

### 二、解题思路

动态规划：同斐波那契数

1. dp数组以及下标的含义：dp[i] 表示爬到 第 i 阶梯 有多少种不同的方法
2. 递推公式：dp[1]=1, dp[2]=2, dp[n] = dp[n-1]+dp[n-2]
3. dp 数组的初始化：dp[1]=1, dp[2]=2, 其他初始化为 0
4. dp 数组的遍历顺序：3 => n
5. 打印 dp 数组

### 三、代码实现

```go
func climbStairs(n int) int {
    if n==1 || n==2 {
        return n
    }
    dp:=make([]int, n+1)
    dp[1], dp[2] = 1, 2
    for i:=3; i<=n; i++ {
        dp[i] = dp[i-1] + dp[i-2]
    }
    return dp[n]
}

```
