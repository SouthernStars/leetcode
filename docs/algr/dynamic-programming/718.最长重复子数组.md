# 718.最长重复子数组
[leetcode 链接](https://leetcode.cn/problems/maximum-length-of-repeated-subarray/description/)

### 一、题目描述

给两个整数数组 nums1 和 nums2 ，返回 两个数组中 公共的 、长度最长的子数组的长度 。



### 二、解题思路

动态规划: 
dp[i][j] 表示 nums1[0~i] nums2[0~j]以 nums[i] nums[j] 为结尾的公共后缀的长度，dp[][] 中最大值即为最长重复子数组的长度



### 三、代码实现

```go
func findLength(nums1 []int, nums2 []int) int {
    m, n := len(nums1), len(nums2)
    dp := make([][]int, m+1)
    for i:=0; i<=m; i++ {
        dp[i] = make([]int, n+1)
    }
    res := 0
    for i:=1; i<=m; i++ {
        for j:=1; j<=n; j++ {
            if nums1[i-1] == nums2[j-1] {
                dp[i][j] = dp[i-1][j-1]+1
                res = max(res, dp[i][j])
            }
        }
    }
    return res
}
```
