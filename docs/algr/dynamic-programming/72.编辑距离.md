# 72.编辑距离
[leetcode 链接](https://leetcode.cn/problems/edit-distance/description/)

### 一、题目描述

给你两个单词 word1 和 word2， 请返回将 word1 转换成 word2 所使用的最少操作数  。

你可以对一个单词进行如下三种操作：

插入一个字符
删除一个字符
替换一个字符
。


### 二、解题思路

动态规划：
dp[i][j] 表示将 w1[0~i] 变为 w2[0~j] 需要的最小步数

dp 初始化   dp[0][*] = 0 , 1, 2, 3 ..
            dp[*][0] = 0, 1, 2,3 ...
状态转移方程 
dp[i][j]       =  dp[i-1][j-1]          w1[i] == w[j]
               = min (*)                w1[i] != w[j]
                    插入对应 dp[i][j-1]+1
                    删除对应 dp[i-1][j]+1
                    替换对应 dp[i-1][j-1]+1


### 三、代码实现

```go
func minDistance(word1 string, word2 string) int {
    m, n := len(word1), len(word2)
    dp := make([][]int, m+1)
    for i:=0;i <=m;i++ {
        dp[i] = make([]int, n+1)
    }
    for i:=0; i<=m; i++ {
        dp[i][0]=i
    } 
    for i:=0;i<=n;i++ {
        dp[0][i]=i
    }

    for i:=1;i<=m;i++ {
        for j:=1;j<=n;j++ {
            if word1[i-1]!=word2[j-1] {
                dp[i][j] = min(dp[i][j-1], dp[i-1][j], dp[i-1][j-1])+1 
            } else {
                 dp[i][j] = dp[i-1][j-1]
            }
        }
    }
    return dp[m][n]
}
```
