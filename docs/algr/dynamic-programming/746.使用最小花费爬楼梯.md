# 746.使用最小花费爬楼梯
[leetcode 链接](https://leetcode.cn/problems/min-cost-climbing-stairs/description/)

### 一、题目描述

给你一个整数数组 cost ，其中 cost[i] 是从楼梯第 i 个台阶向上爬需要支付的费用。一旦你支付此费用，即可选择向上爬一个或者两个台阶。

你可以选择从下标为 0 或下标为 1 的台阶开始爬楼梯。

请你计算并返回达到楼梯顶部的最低花费。

### 二、解题思路

动态规划：

dp[i] 表示从 i 阶梯爬到楼顶需要的最低费用

状态转移方程：dp[i] = min(dp[i+1]+cost[i], dp[i+2]+cost[i])

dp 数组初始化：dp[n]=0, dp[n-1]=cost[n-1]

遍历顺序 n-2 ==> 0

打印 dp 数组

### 三、代码实现

```go
func minCostClimbingStairs(cost []int) int {
    n := len(cost)
    dp := make([]int, n+1)
    dp[n], dp[n-1] = 0, cost[n-1]
    for i:=n-2; i>=0; i-- {
        dp[i] = min(dp[i+1]+cost[i], dp[i+2]+cost[i])
    }
    // 可以从 0 或者 1 开始
    return min(dp[0], dp[1])
}
```
