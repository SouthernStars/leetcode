# **647.回文子串**
[leetcode 链接](https://leetcode.cn/problems/palindromic-substrings/description/)

### 一、题目描述

给你一个字符串 s ，请你统计并返回这个字符串中 回文子串 的数目。

### 二、解题思路

动态规划：dp[i][j] 表示 s[i]~s[j] 是否是回文 true / false，看成是 两个字符串 s1 和 s2

由于 dp[i][j] 取决于 dp[i+1][j-1] 因此 i 需要倒序， j 需要正序遍历; i: n-1 => 0; j:i+1 => n-1

初始化 dp[i][i] = 1

状态转移方程

```
dp[i][j] =   s[i]==s[j]                   j=i+1 只有两个字符
         =   s[i]==s[j] && dp[i+1][j-1]     j>i+1 超过两个字符串
```

结果即为 true 的个数, 在计算中进行统计

### 三、代码实现

```go
func countSubstrings(s string) int {
    n := len(s) 
    ans := 0
    dp:=make([][]bool, n)
    for i:=0;i<n;i++ {
        dp[i] = make([]bool, n)
        dp[i][i]=true
        ans ++
    }
    for i:=n-1;i>=0;i-- {
        for j:=i+1;j<n;j++ {
            if j==i+1 {
                dp[i][j] = s[i]==s[j]
            } else {
                dp[i][j] = s[i]==s[j] && dp[i+1][j-1]
            }
            if dp[i][j] {
                ans ++
            }
        }
    }
    return ans
}
```

```go
// 枚举每一个可能的回文中心，然后用两个指针分别向左右两边拓展，当两个指针指向的元素相同的时候就拓展，否则停止拓展。
// 需要分两种情况，中心是一个字符还是两个
func countSubstrings(s string) int {
    n := len(s) 
    ans := 0
    for i:=0; i<n; i++ {
        for j:=0; j<=1; j++ { // j==0 表示一个字符，j==1 表示两个字符
            l:=i 
            r:=i+j 
            for l>=0 && r<n && s[l]==s[r] {
                l--
                r++
                ans ++
            }
        }
    }
    return ans
}
```


# **5.最长回文子串**
[leetcode 链接](https://leetcode.cn/problems/longest-palindromic-substring/description/)

### 一、题目描述

给你一个字符串 s，找到 s 中最长回文子串 。

### 二、解题思路

动态规划：dp[i][j] 表示 s[i]~s[j] 是否是回文; 同回文子串，判断是否是回文子串过程中，记录串的最大长度和边界

由于 dp[i][j] 取决于 dp[i+1][j-1] 因此 i 需要倒序， j 需要正序遍历

初始化 dp[i][i] = 1

状态转移方程
```
dp[i][j] =   s[i] == s[j]                   j=i+1 只有两个字符
=   s[i]==s[j] && dp[i+1][j-1]
```

### 三、代码实现

```go
func longestPalindrome(s string) string {
    n := len(s) 
    dp:=make([][]bool, n)
    for i:=0;i<n;i++ {
        dp[i] = make([]bool, n)
        dp[i][i]=true
    }
    max_len, left, right := 1, 0, 1
    for i:=n-1;i>=0;i-- {
        for j:=i+1;j<n;j++ {
            if j==i+1 {
                dp[i][j] = s[i]==s[j]
            } else {
                dp[i][j] = s[i]==s[j] && dp[i+1][j-1]
            }
            if dp[i][j] && (j+1-i > max_len) {
                max_len = j+1-i
                left, right = i, j+1
            }
        }
    }
    return s[left:right]
}
```
