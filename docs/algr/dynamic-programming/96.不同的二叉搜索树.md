# 96.不同的二叉搜索树
[leetcode 链接](https://leetcode.cn/problems/unique-binary-search-trees/description/)

### 一、题目描述

给你一个整数 n ，求恰由 n 个节点组成且节点值从 1 到 n 互不相同的 二叉搜索树 有多少种？返回满足题意的二叉搜索树的种数。

### 二、解题思路

动态规划：
// 动态规划：
// dp[n] 表示由 n 个节点组成的1~n个互不相同的二叉搜索树的个数
// f(i,n) 表示以 i 为根节点，总结点为n 的不同的二叉搜索树的个数
// 状态转移方程：
// f(i,n) = dp[i-1] * dp[n-i]
// dp[n] = sum{i 1=>n} f(i, n)
// => dp[n] = sum{i 1=>n} dp[i-1] * dp[n-i]
// dp[0] 表示空树， = 1，  dp[1] = 1， 只有根节点
// dp 数组遍历顺序 1=>n

### 三、代码实现

```go
func numTrees(n int) int {
    dp:=make([]int, n+1)
    dp[0],dp[1] = 1,1
    for i:=2; i<=n; i++ {
        for j:=1; j<=i; j++ {
            dp[i] += dp[j-1]*dp[i-j]
        } 
    }
    return dp[n]
}
```
