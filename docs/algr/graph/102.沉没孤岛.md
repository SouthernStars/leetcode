# 102.沉没孤岛
[卡玛网 链接](https://kamacoder.com/problempage.php?pid=1174)

### 一、题目描述

题目描述
给定一个由 1（陆地）和 0（水）组成的矩阵，岛屿指的是由水平或垂直方向上相邻的陆地单元格组成的区域，且完全被水域单元格包围。孤岛是那些位于矩阵内部、所有单元格都不接触边缘的岛屿。



现在你需要将所有孤岛“沉没”，即将孤岛中的所有陆地单元格（1）转变为水域单元格（0）。

输入描述
第一行包含两个整数 N, M，表示矩阵的行数和列数。

之后 N 行，每行包含 M 个数字，数字为 1 或者 0，表示岛屿的单元格。

输出描述
输出将孤岛“沉没”之后的岛屿矩阵。 注意：每个元素后面都有一个空格

### 二、解题思路

深（广）度优先遍历:

思路依然是从地图周边出发，将周边空格相邻的陆地都做上标记，然后在遍历一遍地图，遇到 陆地 且没做过标记的，那么都是地图中间的 陆地 ，全部改成水域就行。


### 三、代码实现

深度优先遍历
```go
package main

import "fmt"

func main() {
	var n, m int
	fmt.Scanf("%d %d", &n, &m)
	grid := make([][]int, n)
	vist := make([][]bool, n)
	for i := 0; i < n; i++ {
		grid[i] = make([]int, m)
		vist[i] = make([]bool, m)
	}
	for i := 0; i < n; i++ {
		for j := 0; j < m; j++ {
			fmt.Scanf("%d", &grid[i][j])
		}
	}
	for i := 0; i < n; i++ {
		if grid[i][0] == 1 && !vist[i][0] {
			vist[i][0] = true
			dfs(&grid, &vist, i, 0, false)
		}
		if grid[i][m-1] == 1 && !vist[i][m-1] {
			vist[i][m-1] = true
			dfs(&grid, &vist, i, m-1, false)
		}
	}
	for i := 0; i < m; i++ {
		if grid[0][i] == 1 && !vist[0][i] {
			vist[0][i] = true
			dfs(&grid, &vist, 0, i, false)
		}
		if grid[n-1][i] == 1 && !vist[n-1][i] {
			vist[n-1][i] = true
			dfs(&grid, &vist, n-1, i, false)
		}
	}
	// 然后再遍历一遍，沉没孤岛
	for i := 0; i < n; i++ {
		for j := 0; j < m; j++ {
			if grid[i][j] == 1 && !vist[i][j] {
				vist[i][j] = true
				grid[i][j] = 0
				dfs(&grid, &vist, i, j, true)
			}
		}
	}
	// 输出
	for i := 0; i < n; i++ {
		for j := 0; j < m-1; j++ {
			fmt.Printf("%d ", grid[i][j])
		}
		fmt.Println(grid[i][m-1])
	}
}

// 上下左右
var directions = [4][2]int{{-1, 0}, {1, 0}, {0, -1}, {0, 1}}

func dfs(grid *[][]int, vist *[][]bool, x, y int, down bool) {
	for _, dir := range directions {
		newX, newY := x+dir[0], y+dir[1]
		if newX >= 0 && newX < len(*grid) && newY >= 0 && newY < len((*grid)[0]) && (*grid)[newX][newY] == 1 && (*vist)[newX][newY] == false {
			(*vist)[newX][newY] = true
			if down {
				(*grid)[newX][newY] = 0
			}
			dfs(grid, vist, newX, newY, down)
		}
	}
}
```
