# 107.寻找存在的路径
[卡玛网 链接](https://kamacoder.com/problempage.php?pid=1179)

### 一、题目描述

题目描述
给定一个包含 n 个节点的无向图中，节点编号从 1 到 n （含 1 和 n ）。



你的任务是判断是否有一条从节点 source 出发到节点 destination 的路径存在。

输入描述
第一行包含两个正整数 N 和 M，N 代表节点的个数，M 代表边的个数。

后续 M 行，每行两个正整数 s 和 t，代表从节点 s 与节点 t 之间有一条边。

最后一行包含两个正整数，代表起始节点 source 和目标节点 destination。

输出描述
输出一个整数，代表是否存在从节点 source 到节点 destination 的路径。如果存在，输出 1；否则，输出 0。

### 二、解题思路

并查集

### 三、代码实现

```go
package main

import "fmt"

const MaxNodes = 101

var farther [MaxNodes]int

func main() {
	var N, M int
	fmt.Scanf("%d %d", &N, &M)
	// 初始化并查集
	Init(N)
	for i := 0; i < M; i++ {
		var start, end int
		fmt.Scanf("%d %d", &start, &end)
		Join(start, end)
	}
	var source, dest int
	fmt.Scanf("%d %d", &source, &dest)
	if IsSameRoot(source, dest) {
		fmt.Println(1)
	} else {
		fmt.Println(0)
	}
}

// Init 并查集初始化, 节点编号 1~n
func Init(n int) {
	for i := 1; i <= n; i++ {
		farther[i] = i // 指向自己
	}
}

// FindRoot 寻根
func FindRoot(u int) int {
	if farther[u] == u { // 根就是自己
		return u
	}
	// 根不是自己，则 u 指向最顶层的根  路径压缩
	farther[u] = FindRoot(farther[u])
	return farther[u] // 返回最顶层的根
}

// Join 将 u,v 加入同一个集合
func Join(u, v int) {
	u = FindRoot(u)
	v = FindRoot(v)
	if u != v { // 根不相同，则加入
		farther[v] = u
	}
}

// IsSameRoot 返回是否同根，即是否同一个集合中
func IsSameRoot(u, v int) bool {
	return FindRoot(u) == FindRoot(v)
}
```


# 1971.寻找图中是否存在路径
[leetcode 链接](https://leetcode.cn/problems/find-if-path-exists-in-graph/description/)

### 一、题目描述

有一个具有 n 个顶点的 双向 图，其中每个顶点标记从 0 到 n - 1（包含 0 和 n - 1）。图中的边用一个二维整数数组 edges 表示，其中 edges[i] = [ui, vi] 表示顶点 ui 和顶点 vi 之间的双向边。 每个顶点对由 最多一条 边连接，并且没有顶点存在与自身相连的边。

请你确定是否存在从顶点 source 开始，到顶点 destination 结束的 有效路径 。

给你数组 edges 和整数 n、source 和 destination，如果从 source 到 destination 存在 有效路径 ，则返回 true，否则返回 false 。

### 二、解题思路

并查集

### 三、代码实现

```go
var father []int

func validPath(n int, edges [][]int, source int, destination int) bool {
    father = make([]int, n)
    Init(n)
    for i:=0; i<len(edges); i++ {
        Join(edges[i][0], edges[i][1])
    }
    return IsSameRoot(source, destination)
}

// Init 并查集初始化, 节点编号 1~n
func Init(n int) {
	for i := 0; i < n; i++ {
		father[i] = i // 指向自己
	}
}

// FindRoot 寻根
func FindRoot(u int) int {
	if father[u] == u { // 根就是自己
		return u
	}
	// 根不是自己，则 u 指向最顶层的根  路径压缩
	father[u] = FindRoot(father[u])
	return father[u] // 返回最顶层的根
}

// Join 将 u,v 加入同一个集合
func Join(u, v int) {
	u = FindRoot(u)
	v = FindRoot(v)
	if u != v { // 根不相同，则加入
		father[v] = u
	}
}

// IsSameRoot 返回是否同根，即是否同一个集合中
func IsSameRoot(u, v int) bool {
	return FindRoot(u) == FindRoot(v)
}
```
