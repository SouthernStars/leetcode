# 1971.寻找图中是否存在路径
[leetcode 链接](https://leetcode.cn/problems/find-if-path-exists-in-graph/description/)

### 一、题目描述

有一个具有 n 个顶点的 双向 图，其中每个顶点标记从 0 到 n - 1（包含 0 和 n - 1）。图中的边用一个二维整数数组 edges 表示，其中 edges[i] = [ui, vi] 表示顶点 ui 和顶点 vi 之间的双向边。 每个顶点对由 最多一条 边连接，并且没有顶点存在与自身相连的边。

请你确定是否存在从顶点 source 开始，到顶点 destination 结束的 有效路径 。

给你数组 edges 和整数 n、source 和 destination，如果从 source 到 destination 存在 有效路径 ，则返回 true，否则返回 false 。

### 二、解题思路

并查集

### 三、代码实现

```go
var father []int

func validPath(n int, edges [][]int, source int, destination int) bool {
    father = make([]int, n)
    Init(n)
    for i:=0; i<len(edges); i++ {
        Join(edges[i][0], edges[i][1])
    }
    return IsSameRoot(source, destination)
}

// Init 并查集初始化, 节点编号 1~n
func Init(n int) {
	for i := 0; i < n; i++ {
		father[i] = i // 指向自己
	}
}

// FindRoot 寻根
func FindRoot(u int) int {
	if father[u] == u { // 根就是自己
		return u
	}
	// 根不是自己，则 u 指向最顶层的根  路径压缩
	father[u] = FindRoot(father[u])
	return father[u] // 返回最顶层的根
}

// Join 将 u,v 加入同一个集合
func Join(u, v int) {
	u = FindRoot(u)
	v = FindRoot(v)
	if u != v { // 根不相同，则加入
		father[v] = u
	}
}

// IsSameRoot 返回是否同根，即是否同一个集合中
func IsSameRoot(u, v int) bool {
	return FindRoot(u) == FindRoot(v)
}
```
