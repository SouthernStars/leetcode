# 121.买卖股票的最佳时机
[leetcode 链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/description/)

### 一、题目描述


给定一个数组 prices ，它的第 i 个元素 prices[i] 表示一支给定股票第 i 天的价格。

你只能选择 某一天 买入这只股票，并选择在 未来的某一个不同的日子 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 0 。


### 二、解题思路

贪心：题目要求最大的利润，贪心一下，记录下最小的左边的股票价格，在记录下最大的右边的股票价格，在最小时买入，在最大时卖出，我们就可以得到最大的利润。
代码就是：遍历数组，让low记录下此时最小的价格，同时res记录下最大利润，即当前的股票价格减去low记录的最小价格。



### 三、代码实现

```c++
int maxProfit(vector<int>& prices) {
    int n = prices.size();
    int minprice = prices[0];
    int profit = 0;
    for (int price : prices) {
        if (price < minprice) {
            minprice = price;
        } else {
            profit = max(profit, price - minprice);
        }
    }
    return profit;
}
```
