# 1221.分割平衡字符
[leetcode 链接](https://leetcode.cn/problems/split-a-string-in-balanced-strings/description/)

### 一、题目描述

平衡字符串 中，'L' 和 'R' 字符的数量是相同的。

给你一个平衡字符串 s，请你将它分割成尽可能多的子字符串，并满足：

每个子字符串都是平衡字符串。
返回可以通过分割得到的平衡字符串的 最大数量 。

### 二、解题思路

简单贪心：从头开始，一旦平衡，立即分割

### 三、代码实现
```go
func balancedStringSplit(s string) int {
    balance := 0
    ans := 0
    for i:=0; i<len(s); i++ {
        if s[i] == 'L' {
            balance --
		} else if s[i] == 'R' {
            balance ++  
        }
	    if balance == 0 {
            ans ++
        }
    }
    return ans
}
```
