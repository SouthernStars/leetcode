# 406.根据身高重建队列
[leetcode 链接](https://leetcode.cn/problems/queue-reconstruction-by-height/description/)

### 一、题目描述
假设有打乱顺序的一群人站成一个队列，数组 people 表示队列中一些人的属性（不一定按顺序）。每个 people[i] = [hi, ki] 表示第 i 个人的身高为 hi ，前面 正好 有 ki 个身高大于或等于 hi 的人。

请你重新构造并返回输入数组 people 所表示的队列。返回的队列应该格式化为数组 queue ，其中 queue[j] = [hj, kj] 是队列中第 j 个人的属性（queue[0] 是排在队列前面的人）。

### 二、解题思路

贪心：
从高到底考虑，将每个人的身高从大到小进行排序，（身高相同，按照k进行从小到大排序），尽可能的让高的人排在前面。
按照排序后的顺序，依次放入队列。要求，在插入队列要求它的前面恰好有 ki 个人

### 三、代码实现

```c++
vector<vector<int>> reconstructQueue(vector<vector<int>>& people) {
    sort(people.begin(), people.end(), [](vector<int>& a, vector<int>& b) {
        if (a[0] != b[0]) {
            return a[0] > b[0];
        } else {
            return a[1] < b[1];
        }
    });
    vector<vector<int>> res;
    for (const auto& p : people) {
        res.insert(res.begin() + p[1], p);
    }
    return res;
}
// [[7,0],[4,4],[7,1],[5,0],[6,1],[5,2]]    排序前
// [[7,0],[7,1],[6,1],[5,0],[5,2],[4,4]]    排序后
// 依次插入
// [[7,0]]
// [[7,0],[7,1]]
// [[7,0],[6,1],[7,1]]
// [[5,0],[7,0],[6,1],[7,1]]
// [[5,0],[7,0],[5,2],[6,1],[7,1]]
// [[5,0],[7,0],[5,2],[6,1],[4,4],[7,1]]
```
