# 435.无重叠区间
[leetcode 链接](https://leetcode.cn/problems/non-overlapping-intervals/description/)

### 一、题目描述

给定一个区间的集合 intervals ，其中 intervals[i] = [starti, endi] 。返回 需要移除区间的最小数量，使剩余区间互不重叠 。

注意 只在一点上接触的区间是 不重叠的。例如 [1, 2] 和 [2, 3] 是不重叠的。

### 二、解题思路

贪心：按照右端点排序,进行遍历，并且实时维护上一个选择区间的右端点 right。如果当前遍历到的区间与上一个不重合，就贪心的选择这个区间


### 三、代码实现

```c++
int eraseOverlapIntervals(vector<vector<int>>& intervals) {
    sort(intervals.begin(), intervals.end(),
         [](const vector<int>& a, const vector<int>& b) {
             return a[1] < b[1];
         });
    int cnt = 1, n = intervals.size();
    int pos = intervals[0][1];
    for (int i = 1; i < n; i++) {
        if (intervals[i][0] >= pos) {
            cnt++;
            pos = intervals[i][1];
        }
    }
    return n - cnt;
}
```
