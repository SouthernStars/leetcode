# 53.最大子数组和
[leetcode 链接](https://leetcode.cn/problems/maximum-subarray/description/)

### 一、题目描述

给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

子数组 是数组中的一个连续部分。

### 二、解题思路

贪心：
结果有两类：对于全是负数的序列，是负数中最大的那个

如果正负均有，那结果最大序列，一定是正数起始，然后一直 加 知道 sum 为负数，中间记录最大值

### 三、代码实现

```
func maxSubArray(nums []int) int {
    res := math.MinInt
    sum := 0
    for i:=0; i<len(nums); i++{
        sum += nums[i]  // 记录和
        res = max(res, sum) // 更新
        if sum < 0 {   // 当前和 小于 0 ，从新开始一个序列
            sum = 0
        }
    }
    return res
}
```
