# 209.长度最小的子数组
[leetcode 链接](https://leetcode.cn/problems/minimum-size-subarray-sum/description/)

### 一、题目描述
给定一个含有 n 个正整数的数组和一个正整数 target。找出该数组中满足其总和大于等于 target 的长度最小的子数组 [nl, nl+1, ..., nr-1, nr]，并返回
其长度。如果不存在符合条件的子数组，返回 0。

示例 1：

> 输入：target = 7, nums = [2,3,1,2,4,3]
> 
> 输出：2
>
> 解释：子数组 [4,3] 是该条件下的长度最小的子数组。

### 二、解题思路
暴力破解法：初始化子数组的最小长度为无穷大，枚举数组 nums 中的每个下标作为子数组的开始下标，对于每个开始下标 i，需要找到大于或等于 i 的最
小下标 j，使得从 nums[i] 到 nums[j] 的元素和大于或等于 s，并更新子数组的最小长度（此时子数组的长度是 j−i+1。

时间复杂度：$O(n^2)$

空间复杂度：$O(1)$

---

滑动窗口法：定义两个指针 start 和 end 分别表示子数组（滑动窗口窗口）的开始位置和结束位置，维护变量 sum 存储子数组中的元素和（即从 nums[start] 到 
nums[end] 的元素和）。初始状态下，start 和 end 都指向下标 0，sum 的值为 0。 每一轮迭代，将 nums[end] 加到 sum，如果 sum≥s，则更新子数组的
最小长度（此时子数组的长度是 end−start+1）， 然后将 nums[start] 从 sum 中减去并将 start 右移，直到 sum<s，在此过程中同样更新子数组的最小长
度。在每一轮迭代的最后，将 end 右移。

时间复杂度：$O(n)$

空间复杂度：$O(1)$

---

前缀和+二分查找法
前缀和：建立数组 sum, sum[i] 表示 nums[0]~nums[i-1] 的和, sum[0]=1, 数组sum大小为n+1, 由于元素均正，故 sum 是递增的
二分查找：变量对于每个下标 i, 可以在 sum 数组中利用二分查找找到最小的 j 使得 sum[j]-sum[i]>=target，迭代过程中更新 res

时间复杂度：$O(nlog(n))$

空间复杂度：$O(n)$

### 三、代码实现

暴力破解
```Go
func minSubArrayLen(target int, nums []int) int {
    // 暴力破解, 记录从每个点开始的连续和的最小长度，然后更新这个最小长度
    res := math.MaxInt
    n := len(nums)
    for i:=0; i<n; i++ {
        sum := 0
        for j:=i; j<n; j++ {
            sum += nums[j]
            if sum >= target {
                res = min(j-i+1, res)
                break
            }
        }
    }
    if res == math.MaxInt {
        return 0
    }
    return res
}
func min(a int, b int) int {
    if a < b {
        return a
    }
    return b
}
```

滑动窗口
```Go
func minSubArrayLen(target int, nums []int) int {
    // 滑动窗口：设置 start， end 指针, 初始均指向0， sum记录 nums[i]~nums[j] 的和，res 记录最小长度
    // 每次迭代 end++, 当首次sum >=target 时，更新 res, 并尝试 start 右移，更新 sum 和 res 值，直到首次 sum>target 
    res, sum := math.MaxInt, 0 
    for start,end:=0,0; end < len(nums); end ++ {
        sum += nums[end]
        for sum >= target {
            res = min(end-start+1, res)
            sum -= nums[start]
            start ++
        }   
    }
    if res == math.MaxInt {
        return 0
    }
    return res
}
func min(a int, b int) int {
    if a < b {
        return a
    }
    return b
}
```

前缀和+二分查找法
```Go
func minSubArrayLen(target int, nums []int) int {
    // 前缀和：建立数组 sum, sum[i] 表示 nums[0]~nums[i-1] 的和, sum[0]=1, 数组sum大小为n+1, 由于元素均正，故 sum 是递增的
    // 二分查找：变量对于每个下标 i, 可以在 sum 数组中利用二分查找找到最小的 j 使得 sum[j]-sum[i]>=target，变量过程中更新 res
    // nums = [2,3,1,2,4,3] sums = [0, 2, 5, 6, 8, 12, 15]
    // 对于下标 i 来说 （i 从 1 ~ n）
    // sums[i] 表示 nums[0]~nums[i-1] 和
    // sums[j]-sum[i] >= target  => sum[j] >= sum[i] + target => 在 sum 中找到第一个满足条件的 j => res=j-i+1
    res, n:=math.MaxInt, len(nums)
    sums := make([]int, n+1)
    sums[0] = 0
    for i:=1; i<n+1; i++ {
        sums[i] = sums[i-1]+nums[i-1]
    }
    for i:=1; i<=n; i++ {
        s := sums[i-1] + target
        // 在 sums 中查找第一个大于 s 的下标
        idx := searchInsert(sums, s)
        // mid 指向第一个小于 s 的位置
        if idx <= n {
            res = min(res, idx-i+1)
        }        
    }
    if res == math.MaxInt {
        return 0
    }
    return res
}

// 二分查找，在已排序的数组nums中查找target，如果找不到则返回可以插入target的位置i，
func searchInsert(nums []int, target int) int {
    left, right := 0, len(nums)-1
    for left <= right {
        mid := left + (right-left)/2
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return left
}

func min(a int, b int) int {
    if a < b {
        return a
    }
    return b
}
```
