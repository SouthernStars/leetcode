# 11.盛最多水的容器
[leetcode 链接](https://leetcode.cn/problems/container-with-most-water/description/)

### 一、题目描述

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。


### 二、解题思路

双指针，使用 left，right 指针指向左右边界，每次移动低高度的指针。每次移动计算面积，并更新最大面积
计算公式 (right-left)*height

### 三、代码实现

```go
func maxArea(height []int) int {
    maxa := 0
    left, right := 0, len(height)-1

    for left<=right {
        maxa = max(maxa, (right-left)*min(height[left], height[right]))
        if height[left] < height[right] {
            left ++
        } else {
            right --
        }
    }
    return maxa
}
```
