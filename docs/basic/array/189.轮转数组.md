# 189.轮转数组
[leetcode 链接](https://leetcode.cn/problems/rotate-array/description/)

### 一、题目描述

给定一个整数数组 nums，将数组中的元素向右轮转 k 个位置，其中 k 是非负数。

### 二、解题思路

使用额外的数组


### 三、代码实现

```
func rotate(nums []int, k int) {
    n := len(nums)  
    nums_cpy := append([]int{}, nums...)
    for i:=0; i<n; i++ {
        nums[(i+k)%n] = nums_cpy[i]
    }
}
```
