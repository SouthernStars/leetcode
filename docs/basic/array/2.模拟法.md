# 240.搜索二维矩阵II
[leetcode 链接](https://leetcode.cn/problems/search-a-2d-matrix-ii/description/)

### 一、题目描述

编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target 。该矩阵具有以下特性：

每行的元素从左到右升序排列。 每列的元素从上到下升序排列。

### 二、解题思路

模拟，z 字形查找
我们可以从矩阵 matrix 的右上角 (0,n−1) 进行搜索。在每一步的搜索过程中，如果我们位于位置 (x,y)，那么我们希望在以 matrix 的左下角为左下角、以 (x,y) 为右上角的矩阵中进行搜索，即行的范围为 [x,m−1]，列的范围为 [0,y]：

如果 matrix[x,y]=target，说明搜索完成；

如果 matrix[x,y]>target，由于每一列的元素都是升序排列的，那么在当前的搜索矩阵中，所有位于第 y 列的元素都是严格大于 target 的，因此我们可以将它们全部忽略，即将 y 减少 1；

如果 matrix[x,y]<target，由于每一行的元素都是升序排列的，那么在当前的搜索矩阵中，所有位于第 x 行的元素都是严格小于 target 的，因此我们可以将它们全部忽略，即将 x 增加 1。

在搜索的过程中，如果我们超出了矩阵的边界，那么说明矩阵中不存在 target。

### 三、代码实现

```go
func searchMatrix(matrix [][]int, target int) bool {
    m, n := len(matrix), len(matrix[0])
    x, y := 0, n-1
    for x<m && y>=0 {
        if matrix[x][y] == target {
            return true
        } else if matrix[x][y] < target { // 舍去当前行
            x++
        } else if matrix[x][y] > target { // 舍去当前列
            y--
        }
    }
    return false
}
```

# 48.旋转图像
[leetcode 链接](https://leetcode.cn/problems/rotate-image/description/)

### 一、题目描述

给定一个 n × n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

### 二、解题思路

使用辅助数组

### 三、代码实现

```go
func rotate(matrix [][]int)  {
    // 使用辅助数组 matrix[row][col] -> matrix[col][n-row+1]
    n:=len(matrix)
    temp := make([][]int, n)
    for i:=0; i<n; i++ {
        temp[i] = make([]int, n)
    }
    for row:=0; row<n; row++ {
        for col:=0; col<n; col++ {
            temp[col][n-row-1] = matrix[row][col]
        }
    }
    copy(matrix, temp)
}
```

对矩阵进行转置，即将矩阵的行和列进行交换。

对每一行进行反转，即将每一行的元素顺序进行反转。

```go
class Solution {
public void rotate(int[][] matrix) {
    int n = matrix.length;
    // 第一步，转置矩阵
    for (int i = 0; i < n; i++) {
        for (int j = i; j < n; j++) {
            int temp = matrix[i][j];
            matrix[i][j] = matrix[j][i];
            matrix[j][i] = temp;
        }
    }
    // 第二步，反转每一行
    for (int i = 0; i < n; i++) {
        int left = 0;
        int right = n - 1;
            while (left < right) {
                int temp = matrix[i][left];
                matrix[i][left] = matrix[i][right];
                matrix[i][right] = temp;
                left++;
                right--;
            }
        }
    }
}
```


# 189.轮转数组
[leetcode 链接](https://leetcode.cn/problems/rotate-array/description/)

### 一、题目描述

给定一个整数数组 nums，将数组中的元素向右轮转 k 个位置，其中 k 是非负数。

### 二、解题思路

使用额外的数组

### 三、代码实现

```
func rotate(nums []int, k int) {
    n := len(nums)  
    nums_cpy := append([]int{}, nums...)
    for i:=0; i<n; i++ {
        nums[(i+k)%n] = nums_cpy[i]
    }
}
```

# 59.螺旋矩阵II
[leetcode 链接](https://leetcode.cn/problems/spiral-matrix-ii/description/)

### 一、题目描述
给你一个正整数 n ，生成一个包含 1 到 $n^2$ 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix。

示例：输入：n = 3, 输出：[[1,2,3],[8,9,4],[7,6,5]]

1 2 3

8 9 4

7 6 5

### 二、解题思路
模拟法：初始化一个 n×n 大小的矩阵 matrix，然后模拟整个向内环绕的填入过程：
1.定义当前左右上下边界 left,right,top,bottom，初始值 num = 1，迭代终止值 total = n * n；
2.当 num <= total 时，始终按照 从左到右 从上到下 从右到左 从下到上 填入顺序循环，每次填入后 num++, 得到下一个填入的数字；
3.更新边界：从左到右填完后，上边界 top++，相当于上边界向内缩 1; 从上到下填完后，right--; 从右向左填完后，bottom++; 从下到上填完后 left++;
4.使用num <= total而不是l < r || t < b作为迭代条件，是为了解决当n为奇数时，矩阵中心数字无法在迭代过程中被填充的问题。 最终返回 matrix 即可

时间复杂度：$O(n^2)$

空间复杂度：$O(1)$

### 三、代码实现

```Go
func generateMatrix(n int) [][]int {
    matrix := make([][]int, n)
    for i:=range matrix {
        matrix[i] = make([]int, n)
    }
    left, right, top, bottom, num, total := 0, n-1, 0, n-1, 1, n*n
    for num <= total {
        // 从左向右
        for i:=left; i<=right; i++ {
            matrix[top][i] = num
            num ++
        }
        top ++
        // 从上到下
        for i:=top; i<=bottom; i++ {
            matrix[i][right] = num
            num ++
        }
        right --
        // 从右向左
        for i:= right; i>=left; i-- {
            matrix[bottom][i] = num
            num ++   
        }
        bottom --
        // 从下到上
        for i:=bottom; i>=top; i-- {
            matrix[i][left] = num
            num ++
        }
        left ++  
    }
    return matrix
}
```
