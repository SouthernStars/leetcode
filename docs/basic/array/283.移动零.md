# 283.移动零
[leetcode 链接](https://leetcode.cn/problems/move-zeroes/description/)

### 一、题目描述

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

### 二、解题思路

双指针法


### 三、代码实现

```go
// 双指针 left 左边均是非0， left~right 均是0, right 之后未处理
func moveZeroes(nums []int)  {
    left, right, n := 0, 0, len(nums)

    for right < n {
        if nums[right] != 0 {
            // 与左指针交换
            nums[left], nums[right] = nums[right], nums[left]
            left ++ 
        }
        right ++
    }
}
```
