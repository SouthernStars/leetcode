# 136.只出现一次的数字
[leetcode 链接](https://leetcode.cn/problems/single-number/description/)

### 一、题目描述

给你一个 非空 整数数组 nums ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

### 二、解题思路

位运算：将所有数按位取 异或 ，即可得到结果

相同两个数字按位 取异或结果为0

### 三、代码实现

```c++
int singleNumber(vector<int>& nums) {
    int res = 0;
    for (int num : nums) {
        res ^= num;
    }
    return res;
}
```

# 461.汉明距离
[leetcode 链接](https://leetcode.cn/problems/hamming-distance/description/)

### 一、题目描述
两个整数之间的 汉明距离 指的是这两个数字对应二进制位不同的位置的数目。

给你两个整数 x 和 y，计算并返回它们之间的汉明距离。

### 二、解题思路

位运算：

x 与 y 作异或运算, 得到 z. 安装结论计算 z 变成 0 的次数

结论：记 f(z) 表示 z 和 z−1 进行与运算所得的结果（即 f(z)=z&(z−1)），那么 f(z) 恰为 z 删去其二进制表示中最右侧的 1 的结果。

### 三、代码实现

```c++
int hammingDistance(int x, int y) {
    int z = x ^ y, count = 0;
    while (z) {
        count += (z & 1);
        z >>= 1;
    }
    return count;
}
```

# 338.比特位计数
[leetcode 链接](https://leetcode.cn/problems/counting-bits/description/)

### 一、题目描述

给你一个整数 n ，对于 0 <= i <= n 中的每个 i ，计算其二进制表示中 1 的个数 ，返回一个长度为 n + 1 的数组 ans 作为答案。

### 二、解题思路

位运算：

结论：对于任意整数 x，令 x=x&(x−1) 该运算将 x 的二进制表示的最后一个 1 变成 0。因此，对 x 重复该操作，直到 x 变成 0，则操作次数即为x 的「一比特数」

### 三、代码实现

```c++
vector<int> countBits(int n) {
    vector<int> res(n + 1, 0);
    for (int i = 1; i <= n; i++) {
        int cnt = 0, ret = i;
        while (ret) {
            cnt += (ret & 1);
            ret >>= 1;
        }
        res[i] = cnt;
    }
    return res;
}

```
