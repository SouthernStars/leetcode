# 4.寻找两个正序数组的中位数
[leetcode 链接](https://leetcode.cn/problems/median-of-two-sorted-arrays/description/)

### 一、题目描述

给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。

### 二、解题思路

核心思路是**将中位数问题转化为寻找分割线的问题**：我们在较短的数组上进行二分搜索，寻找一个分割位置使得两个数组左边部分的总长度等于（或比右边多一个）且左边所有元素都小于等于右边元素，这样分割线两侧的四个边界值就能直接计算出中位数，从而在 $O(log(min(m,n))) $时间内高效解决问题。

```txt
//示例
nums1: [1, 3 | 5, 7, 9]
          i=2↑
nums2: [2, 4, 6, 8 | 10, 12, 14]
                j=4↑

四个关键值：
nums1Left = 3 (i-1)
nums1Right = 5 (i)
nums2Left = 8 (j-1) 
nums2Right = 10 (j)

检查条件：
3 ≤ 10 ✓ (nums1Left ≤ nums2Right)
8 ≤ 5 ✗ (nums2Left ≤ nums1Right) ❌ 失败！

结论：i太小了，需要增加（因为nums2Left > nums1Right）
更新：left = i + 1 = 3
```



### 三、代码实现

```c++
double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
    int m = nums1.size(), n = nums2.size();
    if (m > n) {
        return findMedianSortedArrays(nums2, nums1);
    }
    int left = 0, right = m;
    int totalleft = (m + n + 1) / 2;
    while (left <= right) {
        int i = left + (right - left) / 2;
        int j = totalleft - i;
        int nums1left = i > 0 ? nums1[i - 1] : INT_MIN;
        int nums1right = i < m ? nums1[i] : INT_MAX;
        int nums2left = j > 0 ? nums2[j - 1] : INT_MIN;
        int nums2right = j < n ? nums2[j] : INT_MAX;
        if (nums1left <= nums2right && nums2left <= nums1right) {
            if ((m + n) % 2) {
                return max(nums1left, nums2left);
            } else {
                return (max(nums1left, nums2left) +
                        min(nums1right, nums2right)) /
                2.0;
            }
        } else if (nums1left > nums2right) {
            right = i - 1;
        } else {
            left = i + 1;
        }
    }
    return 0.0;
}
```
