# 48.旋转图像
[leetcode 链接](https://leetcode.cn/problems/rotate-image/description/)

### 一、题目描述

给定一个 n × n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

### 二、解题思路

使用辅助数组

### 三、代码实现

```go
func rotate(matrix [][]int)  {
    // 使用辅助数组 matrix[row][col] -> matrix[col][n-row+1]
    n:=len(matrix)
    temp := make([][]int, n)
    for i:=0; i<n; i++ {
        temp[i] = make([]int, n)
    }
    for row:=0; row<n; row++ {
        for col:=0; col<n; col++ {
            temp[col][n-row-1] = matrix[row][col]
        }
    }
    copy(matrix, temp)
}
```

对矩阵进行转置，即将矩阵的行和列进行交换。

对每一行进行反转，即将每一行的元素顺序进行反转。

```go
class Solution {
public void rotate(int[][] matrix) {
    int n = matrix.length;
    // 第一步，转置矩阵
    for (int i = 0; i < n; i++) {
        for (int j = i; j < n; j++) {
            int temp = matrix[i][j];
            matrix[i][j] = matrix[j][i];
            matrix[j][i] = temp;
        }
    }
    // 第二步，反转每一行
    for (int i = 0; i < n; i++) {
        int left = 0;
        int right = n - 1;
            while (left < right) {
                int temp = matrix[i][left];
                matrix[i][left] = matrix[i][right];
                matrix[i][right] = temp;
                left++;
                right--;
            }
        }
    }
}
```
