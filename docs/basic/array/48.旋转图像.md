# 48.旋转图像
[leetcode 链接](https://leetcode.cn/problems/rotate-image/description/)

### 一、题目描述

给定一个 n × n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

### 二、解题思路

使用辅助数组

### 三、代码实现

```go
func rotate(matrix [][]int)  {
    // 使用辅助数组 matrix[row][col] -> matrix[col][n-row+1]
    n:=len(matrix)
    temp := make([][]int, n)
    for i:=0; i<n; i++ {
        temp[i] = make([]int, n)
    }
    for row:=0; row<n; row++ {
        for col:=0; col<n; col++ {
            temp[col][n-row-1] = matrix[row][col]
        }
    }
    copy(matrix, temp)
}
```
