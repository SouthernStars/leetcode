# 560.和为K的子数组
[leetcode 链接](https://leetcode.cn/problems/subarray-sum-equals-k/description/)

### 一、题目描述

给你一个整数数组 nums 和一个整数 k ，请你统计并返回 该数组中和为 k 的子数组的个数 。

子数组是数组中元素的连续非空序列。

### 二、解题思路

枚举：枚举以 i 为起始元素且 和 为 k 的子树组的数量


### 三、代码实现

```go
func subarraySum(nums []int, k int) int {
    ans := 0
    for i:=0; i<len(nums); i++ {
        sum := 0
        for j:=i; j<len(nums); j++ {
            sum += nums[j]
            if sum == k {
                ans ++
            }
        }
    }
    return ans
}
```
