# 74.搜索二维矩阵
[leetcode 链接](https://leetcode.cn/problems/search-a-2d-matrix/description/)

### 一、题目描述

给你一个满足下述两条属性的 m x n 整数矩阵：

每行中的整数从左到右按非严格递增顺序排列。
每行的第一个整数大于前一行的最后一个整数。
给你一个整数 target ，如果 target 在矩阵中，返回 true ；否则，返回 false


### 二、解题思路

两次二分查找


### 三、代码实现

```go
func searchMatrix(matrix [][]int, target int) bool {
    m, n := len(matrix), len(matrix[0])

    low1, high1 := 0, m-1
    for low1 <= high1 {
        mid1 := (low1 + high1) / 2
        if matrix[mid1][0] == target {
            return true
        } else if matrix[mid1][0] > target {
            high1 = mid1 - 1
        } else {
            low1 = mid1 + 1
        }
    }
    if low1-1 >=0 {
       // 然后在 low-1 行进行查找
        low2, high2 := 0, n-1
        for low2 <= high2 {
            mid2 := (low2 + high2)/2 
            if matrix[low1-1][mid2] == target {
                return true
            } else if matrix[low1-1][mid2] > target {
                high2 = mid2 -1
            } else {
                low2 = mid2 + 1
            }   
        }
    }

    return false
}
```
