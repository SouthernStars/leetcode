# 100.相同的树
[leetcode 链接](https://leetcode.cn/problems/same-tree/description/)

### 一、题目描述
给你两棵二叉树的根节点 p 和 q ，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。



### 二、解题思路
递归：递归，相同的树 == 根节点值相同，且左右子树均相同

### 三、代码实现

```go
func isSameTree(p *TreeNode, q *TreeNode) bool {
    if p == nil && q == nil {
        return true
    }
    if p == nil || q == nil {
        return false
    }
    // p, q 均不空
    if p.Val != q.Val {
        return false
    }
    return isSameTree(p.Left, q.Left) && isSameTree(p.Right, q.Right)
}
```
