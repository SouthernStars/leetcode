# 102.二叉树的层序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-level-order-traversal/description/)

### 一、题目描述
给你二叉树的根节点 root ，返回其节点值的 层序遍历 。 （即逐层地，从左到右访问所有节点）。

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历

### 三、代码实现

```c++
vector<vector<int>> levelOrder(TreeNode* root) {
    vector<vector<int>> res;
    if (!root) {
        return res;
    }
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        int curlevel = q.size();
        vector<int> curres;
        for (int i = 0; i < curlevel; i++) {
            TreeNode* cur = q.front();
            curres.push_back(cur->val);
            if (cur->left) {
                q.push(cur->left);
            }
            if (cur->right) {
                q.push(cur->right);
            }
            q.pop();
        }
        res.push_back(curres);
    }
    return res;
}
```

# 107.二叉树的层序遍历II
[leetcode 链接](https://leetcode.cn/problems/binary-tree-level-order-traversal-ii/description/)

### 一、题目描述
给你二叉树的根节点 root ，返回其节点值 自底向上的层序遍历 。 （即按从叶子节点所在层到根节点所在的层，逐层从左向右遍历）

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历, 同二叉树的层序遍历相同，就是结果切片按层 头插

### 三、代码实现

```c++
vector<vector<int>> levelOrderBottom(TreeNode* root) {
    vector<vector<int>> res;
    if (!root) {
        return res;
    }
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        int curlevel = q.size();
        vector<int> curres;
        for (int i = 0; i < curlevel; i++) {
            TreeNode* cur = q.front();
            curres.push_back(cur->val);
            if (cur->left) {
                q.push(cur->left);
            }
            if (cur->right) {
                q.push(cur->right);
            }
            q.pop();
        }
        res.push_back(curres);
    }
    reverse(res.begin(), res.end());
    return res;
}
```

# 199.二叉树的右视图
[leetcode 链接](https://leetcode.cn/problems/binary-tree-right-side-view/description/)

### 一、题目描述
给定一个二叉树的 根节点 root，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。


### 二、解题思路

层序遍历：从顶到底返回每层切片的最后一个元素拼接成的切片

### 三、代码实现

```c++
vector<int> rightSideView(TreeNode* root) {
    vector<int> res;
    if (!root) {
        return res;
    }
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        int curlevel = q.size();
        for (int i = 0; i < curlevel; i++) {
            TreeNode* cur = q.front();
            if (i == curlevel - 1) {
                res.push_back(cur->val);
            }
            if (cur->left) {
                q.push(cur->left);
            }
            if (cur->right) {
                q.push(cur->right);
            }
            q.pop();
        }
    }
    return res;
}
```

# 637.二叉树的层平均值
[leetcode 链接](https://leetcode.cn/problems/average-of-levels-in-binary-tree/description/)

### 一、题目描述
给定一个非空二叉树的根节点 root , 以数组的形式返回每一层节点的平均值。

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历，并记录每层的平均值

### 三、代码实现

```c++
vector<double> averageOfLevels(TreeNode* root) {
    vector<double> res;
    if (!root) {
        return res;
    }
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        int curlevel = q.size();
        double sum = 0;
        for (int i = 0; i < curlevel; i++) {
            TreeNode* cur = q.front();
            sum += cur->val;
            if (cur->left) {
                q.push(cur->left);
            }
            if (cur->right) {
                q.push(cur->right);
            }
            q.pop();
        }
        res.push_back(sum / curlevel);
    }
    return res;
}
```

# 116.填充每个节点的下一个右侧节点指针
[leetcode 链接](https://leetcode.cn/problems/populating-next-right-pointers-in-each-node/description/)

### 一、题目描述

给定一个 完美二叉树 ，其所有叶子节点都在同一层，每个父节点都有两个子节点。二叉树定义如下：

```c++
struct Node {
    int val;
    Node *left;
    Node *right;
    Node *next;
}
```

填充它的每个 next 指针，让这个指针指向其下一个右侧节点。如果找不到下一个右侧节点，则将 next 指针设置为 NULL。

初始状态下，所有 next 指针都被设置为 NULL。

### 二、解题思路

层次遍历：然后连接每层的节点


### 三、代码实现

```c++
Node* connect(Node* root) {
    if (!root) {
        return nullptr;
    }
    Node* p = root;
    queue<Node*> q;
    q.push(p);
    while (!q.empty()) {
        int curlevel = q.size();
        Node* pre = nullptr;
        for (int i = 0; i < curlevel; i++) {
            Node* cur = q.front();
            if (pre) {
                pre->next = cur;
            }
            pre = cur;
            if (cur->left) {
                q.push(cur->left);
            }
            if (cur->right) {
                q.push(cur->right);
            }
            q.pop();
        }
        if (pre) {
            pre->next = nullptr;
        }
    }
    return root;
}
```

# 404.左叶子之和
[leetcode 链接](https://leetcode.cn/problems/sum-of-left-leaves/description/)

### 一、题目描述

给定二叉树的根节点 root ，返回所有左叶子之和。

### 二、解题思路
深度优先遍历：

如果根节点的左叶子不空
- 如果是叶子节点，ans 加上左叶子值，如果不是叶子节点，继续以左叶子节点为根节点 dfs

如果根节点的右叶子
- 以右叶子节点为根节点 dfs

广度优先遍历：使用辅助队列，在左子树入队时判断节点是否是叶子节点

### 三、代码实现


```c++
int sumOfLeftLeaves(TreeNode* root) {
    if (!root) {
        return 0;
    }
    int leftval = 0;
    if (root->left && !root->left->left && !root->left->right) {
        leftval += root->left->val;
    } else {
        leftval += sumOfLeftLeaves(root->left);
    }
    int rightval = sumOfLeftLeaves(root->right);
    return leftval + rightval;
}
```



# 513.找树左下角的值

### 一、题目描述

给定一个二叉树的 根节点 root，请找出该二叉树的 最底层 最左边 节点的值。

假设二叉树中至少有一个节点。



### 二、解题思路

层次遍历：最后一层的第一个节点

### 三、代码实现

```c++
int findBottomLeftValue(TreeNode* root) {
    queue<TreeNode*> q;
    q.push(root);
    int res = root->val;
    while (!q.empty()) {
        int curlevel = q.size();
        res = q.front()->val;
        for (int i = 0; i < curlevel; i++) {
            TreeNode* cur = q.front();
            if (cur->left) {
                q.push(cur->left);
            }
            if (cur->right) {
                q.push(cur->right);
            }
            q.pop();
        }
    }
    return res;
}
```
