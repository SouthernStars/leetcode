# 102.二叉树的层序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-level-order-traversal/description/)

### 一、题目描述
给你二叉树的根节点 root ，返回其节点值的 层序遍历 。 （即逐层地，从左到右访问所有节点）。

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历

### 三、代码实现

```go
func levelOrder(root *TreeNode) (res [][]int) {
    if root == nil {
        return res
    }
    var queue []*TreeNode // 模拟队列
    queue = append(queue, root) // 根节点入队
    for len(queue) != 0 {   // 队列不空
        var layer []int
        var layerqueue []*TreeNode
        for i:=0; i<len(queue); i++ {       // 处理整层
            elem := queue[i]
            layer = append(layer, elem.Val)
            if elem.Left != nil {   // 左右孩子入队
                layerqueue = append(layerqueue, elem.Left)
            }
            if elem.Right != nil {
                layerqueue = append(layerqueue, elem.Right)
            }
        }
        queue = layerqueue 
        res = append(res, layer)    // 处理一层后入队
    }
    return res
}
```

# 107.二叉树的层序遍历II
[leetcode 链接](https://leetcode.cn/problems/binary-tree-level-order-traversal-ii/description/)

### 一、题目描述
给你二叉树的根节点 root ，返回其节点值 自底向上的层序遍历 。 （即按从叶子节点所在层到根节点所在的层，逐层从左向右遍历）

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历, 同二叉树的层序遍历相同，就是结果切片按层 头插

### 三、代码实现

```go
func levelOrderBottom(root *TreeNode) (ans [][]int) {
    if root == nil {
        return ans
    }
    queue := []*TreeNode{}
    queue = append(queue, root) // 根节点入队
    for len(queue) != 0 {
        var layer []int
        layerqueue := []*TreeNode{}
        for i:=0; i<len(queue); i++ {
            elem := queue[i]
            layer = append(layer, elem.Val)
            if elem.Left != nil {
                layerqueue = append(layerqueue, elem.Left)
            }
            if elem.Right != nil {
                layerqueue = append(layerqueue, elem.Right)
            }
        }
        ans = append([][]int{layer}, ans...)
        queue = layerqueue
    }
    return ans
}
```

# 199.二叉树的右视图
[leetcode 链接](https://leetcode.cn/problems/binary-tree-right-side-view/description/)

### 一、题目描述
给定一个二叉树的 根节点 root，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。


### 二、解题思路

层序遍历：从顶到底返回每层切片的最后一个元素拼接成的切片

### 三、代码实现

```go
func rightSideView(root *TreeNode) (res []int) {
    if root == nil {
        return res
    }
    var queue []*TreeNode // 模拟队列
    queue = append(queue, root) // 根节点入队
    for len(queue) != 0 {   // 队列不空
        // 记录该层的最后一个元素
        res = append(res, queue[len(queue)-1].Val)
        // 处理下一层
        var layerqueue []*TreeNode
        for i:=0; i<len(queue); i++ {       // 处理整层
            elem := queue[i]
            if elem.Left != nil {   // 左右孩子入队
                layerqueue = append(layerqueue, elem.Left)
            }
            if elem.Right != nil {
                layerqueue = append(layerqueue, elem.Right)
            }
        }
        queue = layerqueue 
    }
    return res
}
```

# 637.二叉树的层平均值
[leetcode 链接](https://leetcode.cn/problems/average-of-levels-in-binary-tree/description/)

### 一、题目描述
给定一个非空二叉树的根节点 root , 以数组的形式返回每一层节点的平均值。

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历，并记录每层的平均值

### 三、代码实现

```go
func averageOfLevels(root *TreeNode) (ans []float64) {
    if root == nil {
        return ans
    }
    queue := []*TreeNode{}
    queue = append(queue, root)
    for len(queue) != 0 {
        layerqueue := []*TreeNode{}
        k := len(queue)
        var avg float64 = 0.0
        for i:=0; i<k; i++ {
            elem := queue[i]
            avg += float64(elem.Val)
            if elem.Left != nil {
                layerqueue = append(layerqueue, elem.Left)
            }
            if elem.Right != nil {
                layerqueue = append(layerqueue, elem.Right)
            }
        }
        avg /= float64(k)
        ans = append(ans, avg)
        queue = layerqueue
    }
    return ans
}
```

# 429.N叉树的层序遍历
[leetcode 链接](https://leetcode.cn/problems/n-ary-tree-level-order-traversal/description/)

### 一、题目描述

给定一个 N 叉树，返回其节点值的层序遍历。（即从左到右，逐层遍历）。 树的序列化输入是用层序遍历，每组子节点都由 null 值分隔（参见示例）。

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历

### 三、代码实现

```go
func levelOrder(root *Node) (ans [][]int) {
    if root == nil {
        return ans
    }
    queue := []*Node{}
    queue = append(queue, root) 
    for len(queue) != 0 {
        layer := []int{}
        layerqueue := []*Node{}
        for i:=0; i<len(queue); i++ {
            elem := queue[i]
            layer = append(layer, elem.Val)
            for _, c := range elem.Children {
                if c != nil {
                    layerqueue = append(layerqueue, c)
                }
            }
        }
        ans = append(ans, layer)
        queue = layerqueue
    }
    return ans
}
```
