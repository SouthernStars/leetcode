# 105.从前序与中序遍历序列构造二叉树
[leetcode 链接](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/description/)

### 一、题目描述

给定两个整数数组 preorder 和 inorder ，其中 preorder 是二叉树的先序遍历， inorder 是同一棵树的中序遍历，请构造二叉树并返回其根节点。


### 二、解题思路

递归  先序遍历 [根节点，左子树，右子树] 中序遍历[左子树，根节点，右子树]

### 三、代码实现

```go
func buildTree(preorder []int, inorder []int) *TreeNode {
    if len(preorder) == 0 || len(inorder) == 0 {
        return nil
    }
    rootVal := preorder[0]
    rootIdx := 0
    for i, e := range inorder {
        if e == rootVal {
            rootIdx = i
            break
        }
    }
    // preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
    return &TreeNode{Val: rootVal, Left:buildTree(preorder[1:1+rootIdx], inorder[:rootIdx]), Right:buildTree(preorder[1+rootIdx:], inorder[rootIdx+1:])}
}
```
