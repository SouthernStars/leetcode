# 106.从中序与后序遍历序列构造二叉树
[leetcode 链接](https://leetcode.cn/problems/construct-binary-tree-from-inorder-and-postorder-traversal/description/)

### 一、题目描述

给定两个整数数组 inorder 和 postorder ，其中 inorder 是二叉树的中序遍历， postorder 是同一棵树的后序遍历，请你构造并返回这颗 二叉树 。

### 二、解题思路

递归：根据后序遍历的最后一个节点是根节点 [左子树,右子树,根节点]，将中序遍历分为两个部分[左子树，根节点，右子树]，递归建树

### 三、代码实现

```go
func buildTree(inorder []int, postorder []int) *TreeNode {
    if len(inorder) == 0 || len(postorder) == 0 {
        return nil
    }
    rootVal := postorder[len(postorder)-1]  // 根节点值
    rootIdx := 0
    for i, e := range inorder {
        if e == rootVal {
            rootIdx = i
            break
        }
    }
    // inorder = [9,3,15,20,7], postorder = [9,15,7,20,3]
    return &TreeNode{Val:rootVal, Left:buildTree(inorder[:rootIdx], postorder[:rootIdx]), Right:buildTree(inorder[rootIdx+1:], postorder[rootIdx:len(postorder)-1])}
}
```

# 105.从前序与中序遍历序列构造二叉树
[leetcode 链接](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/description/)

### 一、题目描述

给定两个整数数组 preorder 和 inorder ，其中 preorder 是二叉树的先序遍历， inorder 是同一棵树的中序遍历，请构造二叉树并返回其根节点。


### 二、解题思路

递归  先序遍历 [根节点，左子树，右子树] 中序遍历[左子树，根节点，右子树]

### 三、代码实现

```go
func buildTree(preorder []int, inorder []int) *TreeNode {
    if len(preorder) == 0 || len(inorder) == 0 {
        return nil
    }
    rootVal := preorder[0]
    rootIdx := 0
    for i, e := range inorder {
        if e == rootVal {
            rootIdx = i
            break
        }
    }
    // preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
    return &TreeNode{Val: rootVal, Left:buildTree(preorder[1:1+rootIdx], inorder[:rootIdx]), Right:buildTree(preorder[1+rootIdx:], inorder[rootIdx+1:])}
}
```
