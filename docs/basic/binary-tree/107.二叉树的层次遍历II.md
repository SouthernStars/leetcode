# 107.二叉树的层序遍历II
[leetcode 链接](https://leetcode.cn/problems/binary-tree-level-order-traversal-ii/description/)

### 一、题目描述
给你二叉树的根节点 root ，返回其节点值 自底向上的层序遍历 。 （即按从叶子节点所在层到根节点所在的层，逐层从左向右遍历）

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历, 同二叉树的层序遍历相同，就是结果切片按层 头插

### 三、代码实现

```go
func levelOrderBottom(root *TreeNode) (ans [][]int) {
    if root == nil {
        return ans
    }
    queue := []*TreeNode{}
    queue = append(queue, root) // 根节点入队
    for len(queue) != 0 {
        var layer []int
        layerqueue := []*TreeNode{}
        for i:=0; i<len(queue); i++ {
            elem := queue[i]
            layer = append(layer, elem.Val)
            if elem.Left != nil {
                layerqueue = append(layerqueue, elem.Left)
            }
            if elem.Right != nil {
                layerqueue = append(layerqueue, elem.Right)
            }
        }
        ans = append([][]int{layer}, ans...)
        queue = layerqueue
    }
    return ans
}
```
