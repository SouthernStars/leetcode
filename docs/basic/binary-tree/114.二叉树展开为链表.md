# 114.二叉树展开为链表
[leetcode 链接](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/description/)

### 一、题目描述

给你二叉树的根结点 root ，请你将它展开为一个单链表：

展开后的单链表应该同样使用 TreeNode ，其中 right 子指针指向链表中下一个结点，而左子指针始终为 null 。
展开后的单链表应该与二叉树 先序遍历 顺序相同。
### 二、解题思路

思路：先序遍历

在 C++ 中，函数参数默认是**值传递**。这意味着：

```c++
void preorder(TreeNode* root, TreeNode* pre)  // pre 是值传递
```

当你在函数内部修改 `pre` 时，外部的 `pre` 并不会改变！


### 三、代码实现

```c++
void flatten(TreeNode* root) {
    TreeNode* dummy = new TreeNode(0);
    preorder(root, dummy);
}
void preorder(TreeNode* root, TreeNode*& pre) {
    if (!root) {
        return;
    }
    TreeNode* right = root->right;
    pre->right = root;
    pre->left = nullptr;
    pre = root;
    preorder(root->left, pre);
    preorder(right, pre);
}
```
