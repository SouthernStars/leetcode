# 129.求根节点到叶节点数字之和
[leetcode 链接](https://leetcode.cn/problems/sum-root-to-leaf-numbers/description/)

### 一、题目描述
给你一个二叉树的根节点 root ，树中每个节点都存放有一个 0 到 9 之间的数字。 每条从根节点到叶节点的路径都代表一个数字：

例如，从根节点到叶节点的路径 1 -> 2 -> 3 表示数字 123 。
计算从根节点到叶节点生成的 所有数字之和。

### 二、解题思路

递归：深度优先遍历

### 三、代码实现

``` go
func sumNumbers(root *TreeNode) int {
    ans := dfs(root, 0)  
    return ans
}

func dfs(root *TreeNode, c int) int {
    if root == nil {
        return 0
    }
    sum := c*10 + root.Val
    if root.Left == nil && root.Right == nil {
        return sum
    }

    return dfs(root.Left, sum)+dfs(root.Right, sum)
}
```
