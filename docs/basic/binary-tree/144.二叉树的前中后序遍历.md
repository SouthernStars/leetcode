# 144.二叉树的前序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-preorder-traversal/description/)

### 一、题目描述
给你二叉树的根节点 root ，返回它节点值的 前序 遍历。

### 二、解题思路

递归先序遍历： 如果二叉树不空，
- 访问根节点
- 递归遍历左子树 
- 递归遍历右子树

迭代法：显示的使用栈进行模拟, 一直向左走，直到为 nil, 在根据栈顶元素向右走。入栈以前进行访问

### 三、代码实现

递归先序遍历

```c++
vector<int> preorderTraversal(TreeNode* root) {
    vector<int> res;
    preorder(root, res);
    return res;
}
void preorder(TreeNode* root, vector<int>& res) {
    if (!root) {
        return;
    }
    res.push_back(root->val);
    preorder(root->left, res);
    preorder(root->right, res);
}
```

迭代遍历
```c++
vector<int> preorderTraversal(TreeNode* root) {
    vector<int> res;
    stack<TreeNode*> st;
    while (root || !st.empty()) {
        if (root) {
            res.push_back(root->val);
            st.push(root);
            root = root->left;
        } else {
            root = st.top()->right;
            st.pop();
        }
    }
    return res;
}
```

# 145.二叉树的后序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-postorder-traversal/)

### 一、题目描述
给定一个二叉树的根节点 root ，返回 它的 中序 遍历

### 二、解题思路

递归后序遍历： 如果二叉树不空，
- 递归遍历左子树
- 递归遍历右子树
- 访问根节点

迭代法：显示的使用栈进行模拟, 一直向左走，直到为 nil, 在根据栈顶元素向右走。在切换为右元素前进行访问，访问元素的条件是，栈顶元素的右指针为空，或该节点的右节点已经被访问

### 三、代码实现

递归后序遍历

```c++
vector<int> postorderTraversal(TreeNode* root) {
    vector<int> res;
    posorder(root, res);
    return res;
}
void posorder(TreeNode* root, vector<int>& res) {
    if (!root) {
        return;
    }
    posorder(root->left, res);
    posorder(root->right, res);
    res.push_back(root->val);
}
```

迭代法

```c++
vector<int> postorderTraversal(TreeNode* root) {
    vector<int> res;
    stack<TreeNode*> st;
    TreeNode* pre;
    while (root || !st.empty()) {
        while (root) {
            st.push(root);
            root = root->left;
        }
        root = st.top();
        if (!root->right || root->right == pre) {
            res.push_back(root->val);
            pre = root;
            root = nullptr;
            st.pop();
        } else {
            root = root->right;
        }
    }
    return res;
}
```

# 94.二叉树的中序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-inorder-traversal/)

### 一、题目描述
给定一个二叉树的根节点 root ，返回 它的 中序 遍历

### 二、解题思路

递归中序遍历： 如果二叉树不空，
- 递归遍历左子树
- 访问根节点
- 递归遍历右子树

迭代法：显示的使用栈进行模拟, 一直向左走，直到为 nil, 在根据栈顶元素向右走。切换为右元素前进行访问

### 三、代码实现

递归中序遍历

```c++
vector<int> inorderTraversal(TreeNode* root) {
    vector<int> res;
    inorder(root, res);
    return res;
}
void inorder(TreeNode* root, vector<int>& res) {
    if (!root) {
        return;
    }
    inorder(root->left, res);
    res.push_back(root->val);
    inorder(root->right, res);
}
```

迭代法

```c++
vector<int> inorderTraversal(TreeNode* root) {
    vector<int> res;
    stack<TreeNode*> st;
    while (root || !st.empty()) {
        while (root) {
            st.push(root);
            root = root->left;
        }
        TreeNode* node = st.top();
        res.push_back(node->val);
        st.pop();
        root = node->right;
    }
    return res;
}
```
