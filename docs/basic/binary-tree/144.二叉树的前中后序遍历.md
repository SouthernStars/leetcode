# 144.二叉树的前序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-preorder-traversal/description/)

### 一、题目描述
给你二叉树的根节点 root ，返回它节点值的 前序 遍历。

### 二、解题思路

递归先序遍历： 如果二叉树不空，
- 访问根节点
- 递归遍历左子树 
- 递归遍历右子树

迭代法：显示的使用栈进行模拟, 一直向左走，直到为 nil, 在根据栈顶元素向右走。入栈以前进行访问

### 三、代码实现

递归先序遍历

```go
func preorderTraversal(root *TreeNode) []int {
    ans := []int{}
    preOrder(root, &ans)
    return ans
}
func preOrder(root *TreeNode, ans *[]int) {
    if root == nil {
        return
    }
    *ans = append(*ans, root.Val) // 访问根节点
    preOrder(root.Left, ans)    // 递归遍历左子树
    preOrder(root.Right, ans)   // 递归遍历右子树

}
```

迭代遍历
```go
func preorderTraversal(root *TreeNode) []int {
    node := root
    stack := []*TreeNode{}
    ans := []int{}
    for node != nil || len(stack)>0 {
        for node != nil {
            ans = append(ans, node.Val)
            stack = append(stack, node)
            node = node.Left
        }
        node = stack[len(stack)-1].Right
        stack = stack[:len(stack)-1]
    }
    return ans
}
```

# 145.二叉树的后序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-postorder-traversal/)

### 一、题目描述
给定一个二叉树的根节点 root ，返回 它的 中序 遍历

### 二、解题思路

递归后序遍历： 如果二叉树不空，
- 递归遍历左子树
- 递归遍历右子树
- 访问根节点

迭代法：显示的使用栈进行模拟, 一直向左走，直到为 nil, 在根据栈顶元素向右走。在切换为右元素前进行访问，访问元素的条件是，栈顶元素的右指针为空，或该节点的右节点已经被访问

### 三、代码实现

递归后序遍历

```go
func postorderTraversal(root *TreeNode) []int {
    ans := []int{}
    postOrder(root, &ans)
    return ans
}
func postOrder(root *TreeNode, ans *[]int) {
    if root == nil {
		return
    }
    postOrder(root.Left, ans)
    postOrder(root.Right, ans)
    *ans = append(*ans, root.Val)
}
```

迭代法

```go
func postorderTraversal(root *TreeNode) []int {
    res := []int{}
    node := root
    stack := []*TreeNode{}
    var prev *TreeNode
    for node!=nil || len(stack)>0 {
        for node != nil {
            stack = append(stack, node)
            node = node.Left
        }
        node = stack[len(stack)-1]
        stack = stack[0:len(stack)-1]
        if node.Right == nil || node.Right == prev { // 访问元素的条件是，node的右指针为空，或者该节点的右节点已经被访问
            res = append(res, node.Val)
            prev = node // prev 记录上次访问的元素
            node = nil
        } else {
            stack = append(stack, node)
            node = node.Right
        }
        
    }
    return res
}
```

# 94.二叉树的中序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-inorder-traversal/)

### 一、题目描述
给定一个二叉树的根节点 root ，返回 它的 中序 遍历

### 二、解题思路

递归中序遍历： 如果二叉树不空，
- 递归遍历左子树
- 访问根节点
- 递归遍历右子树

迭代法：显示的使用栈进行模拟, 一直向左走，直到为 nil, 在根据栈顶元素向右走。切换为右元素前进行访问

### 三、代码实现

递归中序遍历

```Go
func inorderTraversal(root *TreeNode) (res []int) {
    inorder(root, &res)
    return res
}
func inorder (node *TreeNode, res *[]int) {
    if node != nil {
        inorder(node.Left, res)
        *res = append(*res, node.Val)
        inorder(node.Right, res)
    }
}
```

迭代法

```go
func inorderTraversal(root *TreeNode) []int {
    node := root
    stack := []*TreeNode{}
    ans := []int{}
    for node != nil || len(stack) > 0 {
        for node != nil {
            stack = append(stack, node)
            node = node.Left
        }
        ans = append(ans, stack[len(stack)-1].Val)
        node = stack[len(stack)-1].Right
        stack = stack[0:len(stack)-1]
    }
    return ans
}
```
