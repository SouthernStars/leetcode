# 145.二叉树的后序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-postorder-traversal/)

### 一、题目描述
给定一个二叉树的根节点 root ，返回 它的 中序 遍历

### 二、解题思路

递归后序遍历： 如果二叉树不空，
- 递归遍历左子树
- 递归遍历右子树
- 访问根节点

迭代法：显示的使用栈进行模拟, 一直向左走，直到为 nil, 在根据栈顶元素向右走。在切换为右元素前进行访问，访问元素的条件是，栈顶元素的右指针为空，或该节点的右节点已经被访问

### 三、代码实现

递归后序遍历

```go
func postorderTraversal(root *TreeNode) []int {
    ans := []int{}
    postOrder(root, &ans)
    return ans
}
func postOrder(root *TreeNode, ans *[]int) {
    if root == nil {
		return
    }
    postOrder(root.Left, ans)
    postOrder(root.Right, ans)
    *ans = append(*ans, root.Val)
}
```

迭代法

```go
func postorderTraversal(root *TreeNode) []int {
    res := []int{}
    node := root
    stack := []*TreeNode{}
    var prev *TreeNode
    for node!=nil || len(stack)>0 {
        for node != nil {
            stack = append(stack, node)
            node = node.Left
        }
        node = stack[len(stack)-1]
        stack = stack[0:len(stack)-1]
        if node.Right == nil || node.Right == prev { // 访问元素的条件是，node的右指针为空，或者该节点的右节点已经被访问
            res = append(res, node.Val)
            prev = node // prev 记录上次访问的元素
            node = nil
        } else {
            stack = append(stack, node)
            node = node.Right
        }
        
    }
    return res
}
```
