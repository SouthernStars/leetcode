# 145.二叉树的后序遍历
[leetcode 链接](https://leetcode.cn/problems/binary-tree-postorder-traversal/)

### 一、题目描述
给定一个二叉树的 根节点 root，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。


### 二、解题思路

层序遍历：从顶到底返回每层切片的最后一个元素拼接成的切片

### 三、代码实现

```go
func rightSideView(root *TreeNode) (res []int) {
    if root == nil {
        return res
    }
    var queue []*TreeNode // 模拟队列
    queue = append(queue, root) // 根节点入队
    for len(queue) != 0 {   // 队列不空
        // 记录该层的最后一个元素
        res = append(res, queue[len(queue)-1].Val)
        // 处理下一层
        var layerqueue []*TreeNode
        for i:=0; i<len(queue); i++ {       // 处理整层
            elem := queue[i]
            if elem.Left != nil {   // 左右孩子入队
                layerqueue = append(layerqueue, elem.Left)
            }
            if elem.Right != nil {
                layerqueue = append(layerqueue, elem.Right)
            }
        }
        queue = layerqueue 
    }
    return res
}
```
