# 222.完全二叉树的节点个数
[leetcode 链接](https://leetcode.cn/problems/count-complete-tree-nodes/description/)

### 一、题目描述

给你一棵 完全二叉树 的根节点 root ，求出该树的节点个数。

完全二叉树 的定义如下：在完全二叉树中，除了最底层节点可能没填满外，其余每层节点数都达到最大值，并且最下面一层的节点都集中在该层最左边的若干位置。若最底层为第 h 层，则该层包含 1~ 2h 个节点。

### 二、解题思路

递归（后序遍历改造）:当成普通的二叉树进行统计

递归(利用完全二叉树的特性)：可以通过完全二叉树的特性，如果最左节点和最右节点具有相同的高度，那么是完全二叉树，可以通过公式计算节点数目

### 三、代码实现


```go 
func countNodes(root *TreeNode) int {
    if root == nil {
        return 0
    }
    return 1+countNodes(root.Left)+countNodes(root.Right)
}
```

```go
func countNodes(root *TreeNode) int {
    if root == nil {
        return 0
    }
    left_high, right_high := 0, 0
    left_node, right_node := root, root
    for left_node != nil {
        left_node = left_node.Left
        left_high ++
    }
    for right_node != nil {
        right_node = right_node.Right
        right_high ++
    }
    // 如果是完全二叉树
    if left_high == right_high {
        return (2<<(left_high-1)) - 1
    }
    // 如果不是二叉树，就递归 后序遍历改造，处理左右子树
    return 1+countNodes(root.Left)+countNodes(root.Right)
}
```
