# 226.翻转二叉树
[leetcode 链接](https://leetcode.cn/problems/invert-binary-tree/description/)

### 一、题目描述

给你一棵二叉树的根节点 root ，翻转这棵二叉树，并返回其根节点。

### 二、解题思路

递归先序遍历改造: 先序遍历访问根节点时，交换左右子树，然后递归处理左子树 和 右子树

### 三、代码实现

递归先序遍历改造

```go
func invertTree(root *TreeNode) *TreeNode {
    if root == nil {    // 根节点为空
        return nil
    }
    if root.Left == nil && root.Right == nil { // 叶子节点
        return root
    }
    // 递归先序遍历
    root.Left, root.Right = root.Right, root.Left
    invertTree(root.Left)
    invertTree(root.Right)
    return root
}
```
