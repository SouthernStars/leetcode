# 236.二叉树的最近公共祖先
[leetcode 链接](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/description/)

### 一、题目描述

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

### 二、解题思路

递归
存在三种情况 关于 x 是公共祖先：（由于是自顶向下递归）一定是最近的公共祖先
递归向下遍历，当前节点为 x
1. 如果  x==p || x==q, 表面 x 是 p 或者 q 的祖先（递归出口）
2. 在左右子树上递归遍历
3. 如果 x.Left 是 p（q） 的祖先，x.Right是q（p）的祖先，，表明 x 是最近的公共祖先
4. 否则 如果 返回的那个不空，则为公共祖先

### 三、代码实现

```c++
TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
    if (!root || root == p || root == q) {
        return root;
    }
    TreeNode* left = lowestCommonAncestor(root->left, p, q);
    TreeNode* right = lowestCommonAncestor(root->right, p, q);
    if (left && right) {
        return root;
    }
    return left ? left : right;
}
```
