# 257.二叉树的所有路径
[leetcode 链接](https://leetcode.cn/problems/binary-tree-paths/description/)

### 一、题目描述
给你一个二叉树的根节点 root ，按 任意顺序 ，返回所有从根节点到叶子节点的路径。

### 二、解题思路

深（广）度优先遍历：

在深度优先搜索遍历二叉树时，我们需要考虑当前的节点以及它的孩子节点。

- 如果当前节点不是叶子节点，则在当前的路径末尾添加该节点，并继续递归遍历该节点的每一个孩子节点。
- 如果当前节点是叶子节点，则在当前路径末尾添加该节点后我们就得到了一条从根节点到叶子节点的路径，将该路径加入到答案即可


广度优先遍历：

我们维护一个队列，存储节点以及根到该节点的路径。一开始这个队列里只有根节点。在每一步迭代中，我们取出队列中的首节点，如果它是叶子节点，则将它对应的路径加入到答案中。如果它不是叶子节点，则将它的所有孩子节点加入到队列的末尾。当队列为空时广度优先搜索结束，我们即能得到答案。


### 三、代码实现

深度优先遍历

```go
var paths []string // 记录所有路径
func binaryTreePaths(root *TreeNode) []string {
    paths = []string{}
    dfs(root, "")
    return paths
}
// path 是尚未访问 root 的前缀路径
func dfs(root *TreeNode, path string) {
    if root != nil {
        pathSb := path + strconv.Itoa(root.Val) // 拼接当前值
        if root.Left == nil && root.Right == nil { // 如果是叶子节点，路径拼接完毕
            paths = append(paths, pathSb)
        } else {  // 如果不是叶子节点，需要拼接 ->
            pathSb += "->"
            dfs(root.Left, pathSb)
            dfs(root.Right, pathSb)
        }
    }
}
```

广度优先遍历

```go
func binaryTreePaths(root *TreeNode) []string {
    paths := []string{}
    if root == nil {
        return paths
    }
    queue := []*TreeNode{}
    pathqueue := []string{}
    queue = append(queue, root)
    pathqueue = append(pathqueue, strconv.Itoa(root.Val))
    for i:=0; i<len(queue); i++ {
        node, path := queue[i], pathqueue[i]
        if node.Left == nil && node.Right == nil { // 叶子节点
            paths = append(paths, path)
            continue
        }
        if node.Left != nil {
            queue = append(queue, node.Left)
            pathqueue = append(pathqueue, path + "->" + strconv.Itoa(node.Left.Val))
        }
        if node.Right != nil {
            queue = append(queue, node.Right)
            pathqueue = append(pathqueue, path + "->" + strconv.Itoa(queue[i].Right.Val))
        }
    }
    return paths
}
```
