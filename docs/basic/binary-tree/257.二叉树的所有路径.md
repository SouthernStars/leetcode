# 257.二叉树的所有路径
[leetcode 链接](https://leetcode.cn/problems/binary-tree-paths/description/)

### 一、题目描述
给你一个二叉树的根节点 root ，按 任意顺序 ，返回所有从根节点到叶子节点的路径。

### 二、解题思路

深（广）度优先遍历：

在深度优先搜索遍历二叉树时，我们需要考虑当前的节点以及它的孩子节点。

- 如果当前节点不是叶子节点，则在当前的路径末尾添加该节点，并继续递归遍历该节点的每一个孩子节点。
- 如果当前节点是叶子节点，则在当前路径末尾添加该节点后我们就得到了一条从根节点到叶子节点的路径，将该路径加入到答案即可


### 三、代码实现

深度优先遍历

```go
var paths []string // 记录所有路径
func binaryTreePaths(root *TreeNode) []string {
    paths = []string{}
    dfs(root, "")
    return paths
}
// path 是尚未访问 root 的前缀路径
func dfs(root *TreeNode, path string) {
    if root != nil {
        pathSb := path + strconv.Itoa(root.Val) // 拼接当前值
        if root.Left == nil && root.Right == nil { // 如果是叶子节点，路径拼接完毕
            paths = append(paths, pathSb)
        } else {  // 如果不是叶子节点，需要拼接 ->
            pathSb += "->"
            dfs(root.Left, pathSb)
            dfs(root.Right, pathSb)
        }
    }
}
```

# 112.路径总和
[leetcode 链接](https://leetcode.cn/problems/path-sum/description/)

### 一、题目描述

给你二叉树的根节点 root 和一个表示目标和的整数 targetSum 。判断该树中是否存在 根节点到叶子节点 的路径，这条路径上所有节点值相加等于目标和 targetSum 。如果存在，返回 true ；否则，返回 false 。

### 二、解题思路

深度优先遍历：与二叉树的所有路径同理
广度优先遍历：与二叉树的所有路径同理


### 三、代码实现

深度优先遍历

```go
func hasPathSum(root *TreeNode, targetSum int) bool {
    return dfs(root, 0, targetSum)
}
func dfs(root *TreeNode, pathsum int, targetSum int) bool{
    if root != nil {
        pathsum += root.Val   // 计算根节点到当前节点的和
        if root.Left == nil && root.Right == nil { // 叶子节点
            if pathsum == targetSum {
                return true
            }
        } else {
            return dfs(root.Left, pathsum, targetSum) || dfs(root.Right, pathsum, targetSum)
        }
    }
    return false
}
```

广度优先遍历

```go

func hasPathSum(root *TreeNode, targetSum int) bool {
    if root == nil {
        return false
    }
    queue := []*TreeNode{}
    queue = append(queue, root)
    sum := []int{}
    sum = append(sum, root.Val)
    for i:=0; i<len(queue); i++ {
        node, tsum := queue[i], sum[i]  // 出队时判断
        if isLeafNode(node) && tsum == targetSum {
            return true
        }
        if node.Left != nil {
            queue = append(queue, node.Left)
            csum := tsum+node.Left.Val
            sum = append(sum, csum)
        }
        if node.Right != nil {
            queue = append(queue, node.Right)
            csum := tsum+node.Right.Val
            sum = append(sum, csum)
        }
    }
    return false
}
func isLeafNode(root *TreeNode) bool {
    if root.Left == nil && root.Right == nil {
        return true
    }
    return false
}
```
