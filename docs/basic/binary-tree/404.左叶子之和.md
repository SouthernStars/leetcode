# 404.左叶子之和
[leetcode 链接](https://leetcode.cn/problems/sum-of-left-leaves/description/)

### 一、题目描述

给定二叉树的根节点 root ，返回所有左叶子之和。

### 二、解题思路
深度优先遍历：

如果根节点的左叶子不空
- 如果是叶子节点，ans 加上左叶子值，如果不是叶子节点，继续以左叶子节点为根节点 dfs

如果根节点的右叶子不空
- 以右叶子节点为根节点 dfs

广度优先遍历：使用辅助队列，在左子树入队时判断节点是否是叶子节点

### 三、代码实现

深度优先遍历


```go
func sumOfLeftLeaves(root *TreeNode) int {
    if root == nil {
        return 0    
    }
    ans := dfs(root)
    return ans
}
func dfs(root *TreeNode) (ans int) {
    if root.Left != nil {
        if isLeafNode(root.Left) {
            ans += root.Left.Val
        } else {
            ans += dfs(root.Left)
        }
        }
        if root.Right != nil {
            ans += dfs(root.Right)
        }
    return
}
func isLeafNode(root *TreeNode) bool {
    if root.Left == nil && root.Right == nil {
        return true
    }
    return false
}
```

广度优先遍历

```go
func sumOfLeftLeaves(root *TreeNode) int {
    ans := 0
    queue := []*TreeNode{}
    queue = append(queue, root)
    for len(queue) != 0 {
        layerqueue := []*TreeNode{}
        for i:=0; i<len(queue); i++ {
            if queue[i].Left != nil {
                if isLeafNode(queue[i].Left) {
                    ans += queue[i].Left.Val
                }
                layerqueue = append(layerqueue, queue[i].Left)  
            }
            if queue[i].Right != nil {
                layerqueue = append(layerqueue, queue[i].Right)
            }
        }
        queue = layerqueue
    }
    return ans
}
func isLeafNode(root *TreeNode) bool {
    if root.Left == nil && root.Right == nil {
        return true
    }
    return false
}
```
