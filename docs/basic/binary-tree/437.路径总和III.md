# 437.路径总和III
[leetcode 链接](https://leetcode.cn/problems/path-sum-iii/description/)

### 一、题目描述

给定一个二叉树的根节点 root ，和一个整数 targetSum ，求该二叉树里节点值之和等于 targetSum 的 路径 的数目。

路径 不需要从根节点开始，也不需要在叶子节点结束，但是路径方向必须是向下的（只能从父节点到子节点）。

### 二、解题思路

深度优先遍历：
编写递归函数 rootSum(p, sum) 返回以 p 为起点 和 为 sum 的路径，递归公式 rootSum(p,sum) = rootSum(p.Left, sum-p.Val) + root(p.Right, sum-p.Val) , 如果当前节点值刚和等于sum, 返回1
该递归函数用于计算一个节点的路径数，
然后先序遍历计算整个树的


### 三、代码实现

```go
func pathSum(root *TreeNode, targetSum int) (ans int) {
    if root == nil {
        return 0
    }
    ans += rootSum(root, targetSum)
    ans += pathSum(root.Left, targetSum)
    ans += pathSum(root.Right, targetSum)
    return ans
}

func rootSum(root *TreeNode, sum int) (res int) {
    if root == nil {
        return
    }
    if root.Val == sum {
        res ++
    }
    res += rootSum(root.Left, sum-root.Val)
    res += rootSum(root.Right, sum-root.Val)
    return res
}
```
