# 501.二叉搜索树中的众数
[leetcode 链接](https://leetcode.cn/problems/find-mode-in-binary-search-tree/description/)

### 一、题目描述

给你一个含重复值的二叉搜索树（BST）的根节点 root ，找出并返回 BST 中的所有 众数（即，出现频率最高的元素）。

如果树中有不止一个众数，可以按 任意顺序 返回。

假定 BST 满足如下定义：

- 结点左子树中所含节点的值 小于等于 当前节点的值
- 结点右子树中所含节点的值 大于等于 当前节点的值
- 左子树和右子树都是二叉搜索树

### 二、解题思路

中序遍历：记录众数

### 三、代码实现

```c++
vector<int> findMode(TreeNode* root) {
    vector<int> ans;
    if (!root) {
        return ans;
    }
    int maxcnt = 0, curmaxcnt = 0;
    int pre = INT_MAX;
    inorder(root, curmaxcnt, maxcnt, pre, ans);
    if (curmaxcnt > maxcnt) {
        maxcnt = curmaxcnt;
        ans.clear();
        ans.push_back(pre);
    } else if (curmaxcnt == maxcnt) {
        ans.push_back(pre);
    }
    return ans;
}
void inorder(TreeNode*& root, int& curmaxcnt, int& maxcnt, int& pre, vector<int>& ans) {
    if (!root) {
        return;
    }
    inorder(root->left, curmaxcnt, maxcnt, pre, ans);
    if (pre == INT_MAX) {
        curmaxcnt = 1;
        pre = root->val;
    } else if (pre == root->val) {
        curmaxcnt++;
    } else {
        if (curmaxcnt > maxcnt) {
            maxcnt = curmaxcnt;
            ans.clear();
            ans.push_back(pre);
        } else if (curmaxcnt == maxcnt) {
            ans.push_back(pre);
        }
        curmaxcnt = 1;
        pre = root->val;
    }
    inorder(root->right, curmaxcnt, maxcnt, pre, ans);
}
```
