# 513.找树左下角的值
[leetcode 链接](https://leetcode.cn/problems/find-bottom-left-tree-value/description/)

### 一、题目描述

给定一个二叉树的 根节点 root，请找出该二叉树的 最底层 最左边 节点的值。

假设二叉树中至少有一个节点。



### 二、解题思路

层次遍历：最后一层的第一个节点

### 三、代码实现

```go
func findBottomLeftValue(root *TreeNode) int {
    // 假设二叉树至少有一个节点  
    queue := []*TreeNode{}
    queue = append(queue, root)
    var ans int
    for len(queue) != 0 {
        ans = queue[0].Val
        layerqueue := []*TreeNode{}
        for i:=0; i<len(queue); i++ {
            if queue[i].Left != nil {
                layerqueue = append(layerqueue, queue[i].Left)  
            }
            if queue[i].Right != nil {
                layerqueue = append(layerqueue, queue[i].Right)
            }
        }
        queue = layerqueue
    }
    return ans
}


```
