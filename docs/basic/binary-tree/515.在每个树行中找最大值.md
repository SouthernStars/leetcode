# 515.在每个树行中找最大值
[leetcode 链接](https://leetcode.cn/problems/find-largest-value-in-each-tree-row/description/)

### 一、题目描述

给定一棵二叉树的根节点 root ，请找出该二叉树中每一层的最大值。

### 二、解题思路

层序遍历：使用 队列 来辅助进行层序遍历

### 三、代码实现

```go
func largestValues(root *TreeNode) (ans []int) {
    if root == nil {
        return ans
    }
    queue := []*TreeNode{}
    queue = append(queue, root)
    for len(queue) != 0 {
        max := math.MinInt
        layerqueue := []*TreeNode{}
        for i:=0; i<len(queue); i++ {
            elem := queue[i]
            if elem.Val > max {
                max = elem.Val
            }
            if elem.Left != nil {
                layerqueue = append(layerqueue, elem.Left)
            }
            if elem.Right != nil {
                layerqueue = append(layerqueue, elem.Right)
            }
        }
        ans = append(ans, max)
        queue = layerqueue
    }
    return ans
}
```
