# 530.二叉搜索树的最小绝对差
[leetcode 链接](https://leetcode.cn/problems/minimum-absolute-difference-in-bst/description/)

### 一、题目描述

给你一个二叉搜索树的根节点 root ，返回 树中任意两不同节点值之间的最小差值 。

差值是一个正数，其数值等于两值之差的绝对值

### 二、解题思路

中序遍历，二叉搜索树的中序遍历是递增的

### 三、代码实现

```go
 // 中序遍历，二叉搜索树的中序遍历是递增的
func getMinimumDifference(root *TreeNode) int {
    ans, pre := math.MaxInt, -1
    inorder(root, &pre, &ans)
    return ans
}
func inorder(root *TreeNode, pre *int, minDiff *int) {
    if root == nil {
        return
    }
    inorder(root.Left, pre, minDiff)
    // 处理
    if *pre == -1 {
        *pre = root.Val
    } else {
        *minDiff = min(root.Val - *pre, *minDiff)
        *pre = root.Val
    }
    inorder(root.Right, pre, minDiff)
}

```
