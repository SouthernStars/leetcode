# 654.最大二叉树
[leetcode 链接](https://leetcode.cn/problems/maximum-binary-tree/description/)

### 一、题目描述

给定一个不重复的整数数组 nums 。 最大二叉树 可以用下面的算法从 nums 递归地构建:

1. 创建一个根节点，其值为 nums 中的最大值。

2. 递归地在最大值 左边 的 子数组前缀上 构建左子树。

3. 递归地在最大值 右边 的 子数组后缀上 构建右子树。

返回 nums 构建的 最大二叉树 。

### 二、解题思路

递归：先序遍历改造

### 三、代码实现

```
func constructMaximumBinaryTree(nums []int) *TreeNode {
    if len(nums) == 0 {
        return nil
    }
    // 找根节点 即最大值
    best := 0
    for i, e := range nums {
        if e>nums[best] {
            best = i
        }
    }
    return &TreeNode{Val:nums[best], Left:constructMaximumBinaryTree(nums[:best]), Right:constructMaximumBinaryTree(nums[best+1:])}
}
```
