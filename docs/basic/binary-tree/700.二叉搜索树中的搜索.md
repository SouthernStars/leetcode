# 700.二叉搜索树中的搜索
[leetcode 链接](https://leetcode.cn/problems/search-in-a-binary-search-tree/description/)

### 一、题目描述

给定二叉搜索树（BST）的根节点 root 和一个整数值 val。

你需要在 BST 中找到节点值等于 val 的节点。 返回以该节点为根的子树。 如果节点不存在，则返回 null 。

### 二、解题思路

递归法、迭代法

### 三、代码实现

迭代法
```c++
TreeNode* searchBST(TreeNode* root, int val) {
    while (root) {
        if (root->val == val) {
            return root;
        } else if (root->val > val) {
            root = root->left;
        } else {
            root = root->right;
        }
    }
    return root;
}
```

递归法

```c++
TreeNode* searchBST(TreeNode* root, int val) {
    if (!root || root->val == val) {
        return root;
    } else if (root->val > val) {
        return searchBST(root->left, val);
    } else {
        return searchBST(root->right, val);
    }
    return nullptr;
}
```
