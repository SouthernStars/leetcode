# 75.颜色分类
[leetcode 链接](https://leetcode.cn/problems/sort-colors/description/)

### 一、题目描述
给定一个包含红色、白色和蓝色、共 n 个元素的数组 nums ，原地 对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。

必须在不使用库内置的 sort 函数的情况下解决这个问题。

示例 1：

输入：nums = [2,0,2,1,1,0]

输出：[0,0,1,1,2,2]

### 二、解题思路

方法一：计数排序

时间复杂度：$O(N*K)$

空间复杂度：$O(K)$

方法二：双指针

使用 p0, p1分别执行 0 和 1 应该插入的位置，初始都是 0，然后遍历数组，遇到 0与p0交换位置，遇到1与p1交换位置；注意，在遇到 0 时交换位置时，把1交换到 i 的位置，需要在与 p1 进行交换位置。

时间复杂度：$O(N)$

空间复杂度：$O(1)$

### 三、代码实现

```Go
func sortColors(nums []int)  {
    // 方法一 计数排序：统计数组 nums 中 0 1 2 的个数，然后重新整个数组
   var cnt [3]int
   for _, i:= range nums {
       cnt[i]++  
   }
   idx := 0
   for i, v := range cnt {
        for k:=0; k<v; k++ {
            nums[idx] = i 
            idx ++
        } 
   }
}
```

```Go
func sortColors(nums []int)  {
    // 使用 p0, p1 执行 0 和 1改插入的位置, 最终 0 1 排在前列，2排在后边
    p0, p1 := 0, 0
    for i, v:=range nums {
        if v==0 {
            nums[p0], nums[i] = nums[i], nums[p0]
            // 处理 0 覆盖 1 的问题，是把交换的 1 补到 p1所指位置，相当于后移
            if p0 < p1 { // nums[i] == 1
                nums[i], nums[p1] = nums[p1], nums[i]
            }
            p0++
            p1++
        } else if v==1 {
            nums[p1], nums[i] = nums[i], nums[p1]
            p1++
        }
    }
}
```
