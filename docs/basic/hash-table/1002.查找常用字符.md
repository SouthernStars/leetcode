# 1002.查找常用字符
[leetcode 链接](https://leetcode.cn/problems/find-common-characters/description/)

### 一、题目描述
给你一个字符串数组 words，请你找出所有在 words 的每个字符串中都出现的共用字符（包括重复字符），并以数组形式返回。你可以按任意顺序返回答案。

### 二、解题思路

根据题目的要求，如果字符 c 在所有字符串中均出现了 k 次及以上，那么最终答案中需要包含 k 个 c。因此，我们可以使用 minfreq[c] 存储字符 c 在所有字符串中出现次数的最小值。

我们可以依次遍历每一个字符串。当我们遍历到字符串 s 时，我们使用 freq[c] 统计 s 中每一个字符 c 出现的次数。在统计完成之后，我们再将每一个 minfreq[c] 更新为其本身与 freq[c] 的较小值。这样一来，当我们遍历完所有字符串后，minfreq[c] 就存储了字符 c 在所有字符串中出现次数的最小值。

由于题目保证了所有的字符均为小写字母，因此我们可以用长度为 26 的数组分别表示 minfreq 以及 freq。

在构造最终的答案时，我们遍历所有的小写字母 c，并将 minfreq[c] 个 c 添加进答案数组即可。


### 三、代码实现

```c++
vector<string> commonChars(vector<string>& words) {
    vector<int> fre(26, 0), minfre(26, INT_MAX);
    for (auto& word : words) {
        fill(fre.begin(), fre.end(), 0);
        for (auto& c : word) {
            fre[c - 'a']++;
        }
        for (int i = 0; i < 26; i++) {
            minfre[i] = min(minfre[i], fre[i]);
        }
    }
    vector<string> res;
    for (int i = 0; i < 26; i++) {
        for (int j = 0; j < minfre[i]; j++) {
            res.push_back(string(1, i + 'a'));
        }
    }
    return res;
}
```
