# 1207.独一无二的出现次数
[leetcode 链接](https://leetcode.cn/problems/unique-number-of-occurrences/description/)

### 一、题目描述

给你一个整数数组 arr，如果每个数的出现次数都是独一无二的，就返回 true；否则返回 false。

### 二、解题思路

哈希表：
首先使用哈希表记录每个数字的出现次数；随后再利用新的哈希表，统计不同的出现次数的数目。如果不同的出现次数的数目等于不同数字的数目，则返回 true，否则返回 false。



### 三、代码实现

```c++
bool uniqueOccurrences(vector<int>& arr) {
    unordered_map<int, int> map;
    unordered_set<int> set;
    for (auto& a : arr) {
        map[a]++;
    }
    for (auto& mp : map) {
        if (set.count(mp.second)) {
            return false;
        }
        set.insert(mp.second);
    }
    return true;
}
```
