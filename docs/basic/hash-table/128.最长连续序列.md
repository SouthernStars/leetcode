# 128.最长连续序列
[leetcode 链接](https://leetcode.cn/problems/longest-consecutive-sequence/description/)

### 一、题目描述

给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

### 二、解题思路

哈希表：使用哈希表记录出现的数
，然后遍历，对于每个数，遍历 v+1, v+2 ... 是否出现，记录连续序列的长度，并记录最大的长度，对于每个 v, 需要先看 v-1 是否出现，如果已经出现，则可以跳过，因为 v-1 开始的序列长度一定更长


### 三、代码实现

```go
func longestConsecutive(nums []int) int {
    hashtable := map[int]bool{}
    for i:=0; i<len(nums); i++ {
        hashtable[nums[i]] = true
    }
    max_len := 0
    for k, _ := range hashtable {
        if !hashtable[k-1] {
            cur, cur_len := k, 1
            for hashtable[cur+1] {
                cur_len ++
                cur ++
            }
            if cur_len > max_len {
                max_len = cur_len
            }
        }
    }
    return max_len
}
```
