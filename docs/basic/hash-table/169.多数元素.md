# 169.多数元素
[leetcode 链接](https://leetcode.cn/problems/majority-element/description/)

### 一、题目描述

给定一个大小为 n 的数组 nums ，返回其中的多数元素。多数元素是指在数组中出现次数 大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

### 二、解题思路

哈希表：统计每个元素出现次数，并记录多数元素

Boyer-Moore 投票算法: 如果我们把众数记为 +1，把其他数记为 −1，将它们全部加起来，显然和大于 `0`，从结果本身我们可以看出众数比其他数多。


### 三、代码实现

```c++
int majorityElement(vector<int>& nums) {
    unordered_map<int, int> map;
    int res = 0, fre = 0;
    for (auto& num : nums) {
        map[num]++;
        if (fre < map[num]) {
            fre = map[num];
            res = num;
        }
    }
    return res;
}
```

Boyer-Moore 投票算法:

```c++
int majorityElement(vector<int>& nums) {
    int fre = 0, res;
    for (auto& num : nums) {
        if (fre == 0) {
            res = num;
            fre = 1;
        } else if (res == num) {
            fre++;
        } else {
            fre--;
        }
    }
    return res;
}
```
