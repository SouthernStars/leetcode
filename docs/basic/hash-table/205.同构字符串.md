# 205.同构字符串
[leetcode 链接](https://leetcode.cn/problems/isomorphic-strings/description/)

### 一、题目描述
给定两个字符串 s 和 t ，判断它们是否是同构的。

如果 s 中的字符可以按某种映射关系替换得到 t ，那么这两个字符串是同构的。

每个出现的字符都应当映射到另一个字符，同时不改变字符的顺序。不同字符不能映射到同一个字符上，相同字符只能映射到同一个字符上，字符可以映射到自己本身。



### 二、解题思路
哈希表法： 使用哈希表记录映射关系

### 三、代码实现

``` go
func isIsomorphic(s string, t string) bool {
    if len(s) != len(t) {
        return false
    }
    hashtables := map[byte]byte{}
    hashtablet := map[byte]byte{}
    for i:=0; i<len(s); i++ {
        if v,ok := hashtables[s[i]]; ok {
            if v != t[i] {
                return false
            }
        } else {
            if v, ok := hashtablet[t[i]]; ok {
                if v != s[i] {
                    return false
                }
            }
            hashtables[s[i]]=t[i] 
            hashtablet[t[i]]=s[i]
        }
    }
    return true
}
```
