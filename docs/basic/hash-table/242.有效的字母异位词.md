# 242.有效的字母异位词
[leetcode 链接](https://leetcode.cn/problems/valid-anagram/description/)

### 一、题目描述
给定两个字符串 s 和 t，编写一个函数来判断 t 是否是 s 的字母异位词。字母异位词是通过重新排列不同单词或短语的字母而形成的单词或短语，并使用所有原字母一次。

### 二、解题思路
哈希表法：使用哈希表记录 s 的字母及其个数，遍历 t中的每个字符，在哈希表中查找是否出现以及字符数量是否足够。同赎金信

### 三、代码实现

```Go
func isAnagram(s string, t string) bool {
    if len(s) != len(t) {
        return false
    }
    hashtable := map[rune]int{}
    for _,i := range s {
        hashtable[i] ++
    }
    for _,j := range t {
        if v, ok := hashtable[j]; ok {
            if v > 0 {
                hashtable[j] --
            } else {
                return false
            }

        } else {
            return false
        }
    }
    return true
}
```
