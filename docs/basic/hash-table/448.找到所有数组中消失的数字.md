# 448.找到所有数组中消失的数字
[leetcode 链接](https://leetcode.cn/problems/find-all-numbers-disappeared-in-an-array/description/)

### 一、题目描述

给你一个含 n 个整数的数组 nums ，其中 nums[i] 在区间 [1, n] 内。请你找出所有在 [1, n] 范围内但没有出现在 nums 中的数字，并以数组的形式返回结果。

### 二、解题思路

哈希表

原地修改：

1. **遍历数组**：对于每个数字 `nums[i]`
   - 计算它应该对应的索引：`index = abs(nums[i]) - 1`
   - 将 `nums[index]` 标记为负数（表示数字 `index + 1` 出现过）
2. **收集结果**：
   - 再次遍历数组
   - 如果 `nums[i] > 0`，说明数字 `i + 1` 没有出现过

### 三、代码实现

哈希表：

```c++
vector<int> findDisappearedNumbers(vector<int>& nums) {
    unordered_set<int> set(nums.begin(), nums.end());
    vector<int> res;
    int n = nums.size();
    for (int i = 1; i <= n; i++) {
        if (!set.count(i)) {
            res.push_back(i);
        }
    }
    return res;
}
```

原地修改：

```c++
vector<int> findDisappearedNumbers(vector<int>& nums) {
    int n = nums.size();
    for (int i = 0; i < n; i++) {
        int index = abs(nums[i]) - 1;
        if (nums[index] > 0) {
            nums[index] = -nums[index];
        }
    }
    vector<int> res;
    for (int i = 0; i < n; i++) {
        if (nums[i] > 0) {
            res.push_back(i + 1);
        }
    }
    return res;
}
```
