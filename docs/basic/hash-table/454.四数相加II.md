# 454.四数相加II
[leetcode 链接](https://leetcode.cn/problems/4sum-ii/description/)

### 一、题目描述
给你四个整数数组 nums1、nums2、nums3 和 nums4，数组长度都是 n ，请你计算有多少个元组 (i, j, k, l) 能满足：

- 0 <= i, j, k, l < n
- nums1[i] + nums2[j] + nums3[k] + nums4[l] == 0

### 二、解题思路
分组 + 哈希表

我们可以将四个数组分成两部分，A 和 B 为一组，C 和 D 为另外一组。对于 A 和 B，我们使用二重循环对它们进行遍历，得到所有 A[i]+B[j] 的值并存入哈希映射中。对于哈希映射中的每个键值对，每个键表示一种 A[i]+B[j]，对应的值为 A[i]+B[j] 出现的次数。

对于 C 和 D，我们同样使用二重循环对它们进行遍历。当遍历到 C[k]+D[l] 时，如果 −(C[k]+D[l]) 出现在哈希映射中，那么将 −(C[k]+D[l]) 对应的值累加进答案中。

时间复杂度：$O(N^2)$

空间复杂度：$O(N^2)$

### 三、代码实现

```c++
int fourSumCount(vector<int>& nums1, vector<int>& nums2, vector<int>& nums3,vector<int>& nums4) {
    unordered_map<int, int> map1;
    for (int num1 : nums1) {
        for (int num2 : nums2) {
            map1[num1 + num2]++;
        }
    }
    int res = 0;
    for (int num3 : nums3) {
        for (int num4 : nums4) {
            int cur = -num3 - num4;
            if (map1.count(cur)) {
                res += map1[cur];
            }
        }
    }
    return res;
}
```
