# 1.两数之和
[leetcode 链接](https://leetcode.cn/problems/two-sum/description/)

### 一、题目描述
给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target  的那 两个 整数，并返回它们的数组下标。你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。你可以按任意顺序返回答案。

### 二、解题思路
哈希表法：变量数组 nums 的元素，根据 target - nums[i] 可以得到满足要求的一个数值，将 此值作为 key, 索引 i 作为 value, 建立哈希表。在迭代的过程中，先查找 nums[i] 在哈希表中是否存在，如果存在，则找到结果。

### 三、代码实现

```Go
func twoSum(nums []int, target int) []int {
    hashtable := map[int]int{}
    for i, e := range nums {
        if j, ok := hashtable[target-e]; ok {
            return []int{j, i}
        }
        hashtable[e] = i
    }
    return nil
}
```
