# 143.重排链表
[leetcode 链接](https://leetcode.cn/problems/reorder-list/description/)

### 一、题目描述

给定一个单链表 L 的头节点 head ，单链表 L 表示为：
L0 → L1 → … → Ln-1 → Ln

请将其重新排列后变为：
L0 → Ln → L1 → Ln-1 → L2 → Ln - 2 → …

不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

### 二、解题思路

思路：快慢指针找中点+翻转链表+合并链表


### 三、代码实现

```
// 注意到目标链表即为将原链表的左半端和反转后的右半端合并后的结果
func reorderList(head *ListNode)  {
    if head == nil {
        return
    }
    // 将链表分为两个
    middle := middleNode(head)
    l1 := head
    l2 := middle.Next
    middle.Next = nil 
    // 翻转链表
    l2 = reverseList(l2)
    // 合并链表
    mergeList(l1, l2)
    return
}

func mergeList(head1 *ListNode, head2 *ListNode) {
    var tmp1, tmp2 *ListNode

    for head1 != nil && head2 != nil {
        tmp1 = head1.Next
        tmp2 = head2.Next
        head1.Next = head2
        head1 = tmp1
        head2.Next = tmp1
        head2 = tmp2
    }
}

func reverseList(head *ListNode) *ListNode {
    dummy := &ListNode{Val:0, Next: nil} 
    tmp := head
    for tmp != nil {
        nextTmp := tmp.Next
        tmp.Next = dummy.Next
        dummy.Next=tmp
        tmp = nextTmp
    }
    return dummy.Next
}

func middleNode(head *ListNode) *ListNode {
    // 将链表分为两个
    slow, fast := head, head
    for fast != nil && fast.Next != nil {
        slow = slow.Next
        fast = fast.Next.Next
    }
    return slow
}
```
