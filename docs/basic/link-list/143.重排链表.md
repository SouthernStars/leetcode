# 143.重排链表
[leetcode 链接](https://leetcode.cn/problems/reorder-list/description/)

### 一、题目描述

给定一个单链表 L 的头节点 head ，单链表 L 表示为：
L0 → L1 → … → Ln-1 → Ln

请将其重新排列后变为：
L0 → Ln → L1 → Ln-1 → L2 → Ln - 2 → …

不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

### 二、解题思路

思路：快慢指针找中点(一个走一步，一个走两步)+翻转链表+合并链表


### 三、代码实现

```c++
void reorderList(ListNode* head) {
    if (!head || !head->next) {
        return;
    }
    ListNode* node = head;
    int len = 0;
    while (node) {
        len++;
        node = node->next;
    }
    node = head;
    for (int i = 1; i < (len + 1) / 2; i++) {
        node = node->next;
    }
    ListNode* p = node->next;
    node->next = nullptr;
    while (p) {
        ListNode* q = p->next;
        p->next = node->next;
        node->next = p;
        p = q;
    }
    ListNode* first = head;
    ListNode* second = node->next;
    node->next = nullptr;
    while (second) {
        node = second->next;
        second->next = first->next;
        first->next = second;
        second = node;
        first = first->next->next;
    }
}
```
