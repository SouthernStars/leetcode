# 148.排序链表
[leetcode 链接](https://leetcode.cn/problems/sort-list/description/)

### 一、题目描述

给你链表的头结点 head ，请将其按 升序 排列并返回 排序后的链表。

### 二、解题思路

思路：归并排序

（1）找到链表的中点，以中点为分界，将链表拆分成两个子链表。寻找链表的中点可以使用快慢指针的做法，快指针每次移动 2 步，慢指针每次移动 1 步，当快指针到达链表末尾时，慢指针指向的链表节点即为链表的中点。

（2）对两个子链表分别排序。

（3）将两个排序后的子链表合并，得到完整的排序后的链表

（4）述过程可以通过递归实现。递归的终止条件是链表的节点个数小于或等于 1，即当链表为空或者链表只包含 1 个节点时，不需要对链表进行拆分和排序。


### 三、代码实现

```go
func sortList(head *ListNode) *ListNode {
    return sort(head, nil)
}
func sort(head, tail *ListNode) *ListNode {
    if head == nil {
        return head
    }
    if head.Next == tail {  // 单个节点
        head.Next = nil 
        return head
    }
    slow, fast := head, head
    for fast!=tail {
        slow=slow.Next
        fast=fast.Next
        if fast != tail {
            fast = fast.Next
        }
    }
    mid := slow
    return merge(sort(head, mid), sort(mid, tail))
}

func merge(head1, head2 *ListNode) *ListNode {
    dummyHead := &ListNode{0, nil}
    tail, temp1, temp2 := dummyHead, head1, head2
    for temp1 != nil && temp2 != nil {
        if temp1.Val <= temp2.Val {
            tail.Next = temp1
            temp1 = temp1.Next
        } else {
            tail.Next = temp2
            temp2 = temp2.Next
        }
        tail = tail.Next
    }
    if temp1 != nil {
        tail.Next = temp1
    } else if temp2 != nil {
        tail.Next = temp2
    }
    return dummyHead.Next
}
```
