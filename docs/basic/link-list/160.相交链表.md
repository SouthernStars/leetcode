# 160.链表相交
[leetcode 链接](https://leetcode.cn/problems/intersection-of-two-linked-lists/description/)

### 一、题目描述
给你两个单链表的头节点 headA 和 headB，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 null。
如图所示两个链表在节点 c1 开始相交：
A:  a1 -> a3 ---------
                      | -> c1 -> c2 -> c3
B:  b1 -> b2 -> b3 ---

### 二、解题思路

哈希表法：遍历链表 A 并保存到哈希表中，然后遍历链表 B, 并在哈希表中查找元素，第一个元素即为所求

双指针：如果两个链表相交，那么相交点之后的长度是相同的，我们需要做的事情是，让两个链表从同距离末尾同等距离的位置开始遍历。长链表先走


时间复杂度：$O(m+n)$

空间复杂度：$O(1)$


### 三、代码实现

双指针：法一
```Go
// 不存在相交节点情况，两个链表的尾节点不同
func getIntersectionNode(headA, headB *ListNode) *ListNode {
    // 分别遍历两个链表，并记录两个链表的长度
    dumllyA := &ListNode{0, headA}
    pa, lena := dumllyA, 0
    for pa.Next != nil {
        pa = pa.Next
        lena ++
    }
    lena ++

    dumllyB := &ListNode{0, headB}
    pb, lenb := dumllyB, 0
    for pb.Next != nil {
        pb = pb.Next
        lenb ++
    }
    lenb ++

    if pa != pb { // 尾结点都不同，没有相交节点
        return nil 
    }
    // 存在相交节点
    var lo, sh *ListNode
    var step int
    if lena > lenb {
        lo = dumllyA
        sh = dumllyB
        step = lena - lenb
    } else {
        lo = dumllyB
        sh = dumllyA
        step = lenb - lena
    }
    // 长的多走 step 步
    for i:=0; i<step; i++ {
        lo = lo.Next
    }
    // 然后同步走
    var res *ListNode
    for lo != nil {
        lo = lo.Next
        sh = sh.Next
        if lo == sh {
            res = lo
            break
        }
    }
    return res
}
```
