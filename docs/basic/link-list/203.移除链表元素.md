# 203.移除链表元素
[leetcode 链接](https://leetcode.cn/problems/remove-linked-list-elements/description/)

### 一、题目描述
给你一个链表的头节点 head 和一个整数 val ，请你删除链表中所有满足 Node.val == val 的节点，并返回 新的头节点。

### 二、解题思路
递归法： 对于给定的链表，首先对除了头节点 head 以外的节点进行删除操作，然后判断 head 的节点值是否等于给定的 val。如果 head 的节点值等于 val，则 head 需要被删除，因此删除操作后的头节点为 head.next；如果 head 的节点值不等于 val，则 head 保留，因此删除操作后的头节点还是 head。上述过程是一个递归的过程。 递归的终止条件是 head 为空，此时直接返回 head。当 head 不为空时，递归地进行删除操作，然后判断 head 的节点值是否等于val 并决定是否要删除 head。

时间复杂度：$O(n)$

空间复杂度：$O(n)$

---

迭代法：迭代的方法删除链表中所有节点值等于特定值的节点

时间复杂度：$O(n)$

空间复杂度：$O(1)$

### 三、代码实现

递归法：

```Go
func removeElements(head *ListNode, val int) *ListNode {
    // 递归法
    if head == nil {  // 递归结束条件
        return head
    }
    head.Next = removeElements(head.Next, val)
    
    // 对于一个链表，该递归函数处理如下：如果节点值与目标值相等 返回下一个节点，如果不同，返回改节点即可
    // Go 有垃圾回收机制，无需显示释放空间
    if head.Val == val {
        return head.Next
    }
    return head
}
```

迭代法：

```Go
func removeElements(head *ListNode, val int) *ListNode {
    if head == nil {
        return nil
    }
    // 添加假头结点，方便处理
    dummyHead := &ListNode{Next: head}
    // 迭代法 
    for p:= dummyHead; p.Next != nil; {
        if p.Next.Val == val {
            p.Next = p.Next.Next
        } else {
            p = p.Next
        }
    }
    return dummyHead.Next
}
```
