# 206.反转链表
[leetcode 链接](https://leetcode.cn/problems/reverse-linked-list/description/)

### 一、题目描述
给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

### 二、解题思路
头插法：使用头插法创建表的思路进行翻转链表

时间复杂度：O(n)

空间复杂度：O(1)

### 三、代码实现

```c++
ListNode* reverseList(ListNode* head) {
    ListNode* newHead = nullptr;
    ListNode* node = head;
    while (node) {
        ListNode* temp = node->next;
        node->next = newHead;
        newHead = node;
        node = temp;
    }
    return newHead;
}
```
