# 21.合并两个有序链表
[leetcode 链接](https://leetcode.cn/problems/merge-two-sorted-lists/description/)

### 一、题目描述

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

### 二、解题思路

思路：合并链表


### 三、代码实现

```go
func mergeTwoLists(list1 *ListNode, list2 *ListNode) *ListNode {
    // 设置伪头结点
    dummy := &ListNode{0, nil}
    tail := dummy
    // 拼接节点
    for list1 != nil && list2 != nil {
        if list1.Val <= list2.Val { // 拼接 list1
            tail.Next = list1
            tail = tail.Next
            list1 = list1.Next
        } else {
            tail.Next = list2
            tail = tail.Next
            list2 = list2.Next
        }
    }
    if list1!=nil {
        tail.Next = list1
    }
    if list2!=nil {
        tail.Next = list2
    }
    return dummy.Next
}   
```
