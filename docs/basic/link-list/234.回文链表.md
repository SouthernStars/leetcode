# 234.回文链表
[leetcode 链接](https://leetcode.cn/problems/palindrome-linked-list/description/)

### 一、题目描述

给你一个单链表的头节点 head ，请你判断该链表是否为 回文链表 。如果是，返回 true ；否则，返回 false 。

### 二、解题思路

思路： 将数据复制到数组中，使用双指针判断是否回文

### 三、代码实现

```go
func isPalindrome(head *ListNode) bool {
    arr := []int{}
    for head != nil {
        arr = append(arr, head.Val)
        head = head.Next
    }
    low, high := 0, len(arr)-1
    for low <= high {
        if arr[low] != arr[high] {
            return false
        } else {
            low ++
            high --
        }
    }
    return true
}
```
