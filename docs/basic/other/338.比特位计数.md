# 338.比特位计数
[leetcode 链接](https://leetcode.cn/problems/counting-bits/description/)

### 一、题目描述

给你一个整数 n ，对于 0 <= i <= n 中的每个 i ，计算其二进制表示中 1 的个数 ，返回一个长度为 n + 1 的数组 ans 作为答案。

### 二、解题思路

位运算：

结论：对于任意整数 x，令 x=x&(x−1) 该运算将 x 的二进制表示的最后一个 1 变成 0。因此，对 x 重复该操作，直到 x 变成 0，则操作次数即为x 的「一比特数」

### 三、代码实现

```go
func countBits(n int) []int {
    ans := []int{}
    for i:=0; i<=n; i++ {
        ans = append(ans, oneCount(i))
    }
    return ans
}
func oneCount(x int) int {
    res:=0
    for x!=0 {
        res ++
        x=x&(x-1)
    }
    return res
}

```
