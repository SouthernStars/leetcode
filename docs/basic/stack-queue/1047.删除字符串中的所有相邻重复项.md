# 1047.删除字符串中的所有相邻重复项
[leetcode 链接](https://leetcode.cn/problems/remove-all-adjacent-duplicates-in-string/description/)

### 一、题目描述
给出由小写字母组成的字符串 s，重复项删除操作会选择两个相邻且相同的字母，并删除它们。

在 s 上反复执行重复项删除操作，直到无法继续删除。

在完成所有重复项删除操作后返回最终的字符串。答案保证唯一。

### 二、解题思路

使用 栈 处理

具体操作如下：遍历每个字符，如果栈为空，则将该字符入栈，如果栈不空，则与栈顶元素比较看是否相同，如果相同，则出栈，否则，将元素入栈，直到遍历完字符串，栈中即为最后的结果

时间复杂度：$O(n)$

空间复杂度：$O(n)$

### 三、代码实现

```Go
func removeDuplicates(s string) string {
    stack := []rune{}
    for _, e := range s {
        if len(stack)==0 {
            stack = append(stack, e)
        } else {
            top := stack[len(stack)-1]
            if top == e {
                stack = stack[0:len(stack)-1]
            } else {
                stack = append(stack, e)
            }
        }
    }
    return string(stack)
}
```
