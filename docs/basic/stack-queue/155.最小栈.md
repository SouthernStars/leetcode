# 155.最小栈
[leetcode 链接](https://leetcode.cn/problems/min-stack/description/)

### 一、题目描述

设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:

MinStack() 初始化堆栈对象。
void push(int val) 将元素val推入堆栈。
void pop() 删除堆栈顶部的元素。
int top() 获取堆栈顶部的元素。
int getMin() 获取堆栈中的最小元素。

### 二、解题思路

栈，设置辅助栈，每入栈一个元素即为一个栈状态，可以记录此时的最小值，并入辅助栈中记录


### 三、代码实现

```c++
stack<int> mainStack, minStack;
MinStack() {}

void push(int val) {
    mainStack.push(val);
    if (minStack.empty() || minStack.top() >= val) {
        minStack.push(val);
    }
}

void pop() {
    if (minStack.top() == mainStack.top()) {
        minStack.pop();
    }
    mainStack.pop();
}

int top() { return mainStack.top(); }

int getMin() { return minStack.top(); }
```
