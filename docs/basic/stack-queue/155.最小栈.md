# 155.最小栈
[leetcode 链接](https://leetcode.cn/problems/min-stack/description/)

### 一、题目描述

设计一个支持 push ，pop ，top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:

MinStack() 初始化堆栈对象。
void push(int val) 将元素val推入堆栈。
void pop() 删除堆栈顶部的元素。
int top() 获取堆栈顶部的元素。
int getMin() 获取堆栈中的最小元素。

### 二、解题思路

栈，设置辅助栈，每入栈一个元素即为一个栈状态，可以记录此时的最小值，并入辅助栈中记录


### 三、代码实现

```go
type MinStack struct {
    stack []int     // 基本栈
    minStack []int  // 辅助栈，记录最小值, 对应每个入栈操作后记录当前状态的最小元素
}
func Constructor() MinStack {
    return MinStack{
        stack: []int{},
        minStack: []int{math.MaxInt},
    }
}
func (this *MinStack) Push(val int)  {
    this.stack = append(this.stack, val) // 入栈
    top:=this.minStack[len(this.minStack)-1] // 判断 val 是否更小并入最小栈
    this.minStack = append(this.minStack, min(val, top))
}
func (this *MinStack) Pop()  {
    this.stack = this.stack[0:len(this.stack)-1] // 出栈
    this.minStack = this.minStack[0:len(this.minStack)-1]
}
func (this *MinStack) Top() int {
    return this.stack[len(this.stack)-1]
}
func (this *MinStack) GetMin() int {
    return this.minStack[len(this.minStack)-1]
}
```
