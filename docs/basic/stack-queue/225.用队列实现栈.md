# 225.用队列实现栈
[leetcode 链接](https://leetcode.cn/problems/implement-stack-using-queues/description/)

### 一、题目描述
请你仅使用两个队列实现一个后入先出（LIFO）的栈，并支持普通栈的全部四种操作（push、top、pop 和 empty）。

实现 MyStack 类：

- void push(int x) 将元素 x 压入栈顶。
- int pop() 移除并返回栈顶元素。
- int top() 返回栈顶元素。
- boolean empty() 如果栈是空的，返回 true ；否则，返回 false 。


### 二、解题思路

// 使用 queue1 和 queue2 来模拟栈，queue1 用来存栈中元素，queue2 用来辅助 queue1 的栈相关操作
// 入栈操作：将元素 e 进入  queue2 队列中，然后将 queue1 中所有元素出队放入 queue2 中，然后交换 queue1 和 queue2
// 出栈操作：出栈 queue1 的 0 号元素
// 判空操作：判断 queue1 是否为空

### 三、代码实现

```Go
type MyStack struct {
    queue1 []int
    queue2 []int
}
func Constructor() MyStack {
    return MyStack{queue1:[]int{}, queue2:[]int{}}
}
func (this *MyStack) Push(x int)  {
    this.queue2 = append(this.queue2, x)
    // for len(this.queue1) > 0 {
    //     this.queue2 = append(this.queue2, this.queue1[0])
    //     this.queue1 = this.queue1[1:]
    // }
    if len(this.queue1) > 0 {
        this.queue2 = append(this.queue2, this.queue1...)
        this.queue1 = []int{}
    }
    this.queue1, this.queue2 = this.queue2, this.queue1
}
func (this *MyStack) Pop() int {
    x := this.queue1[0]
    this.queue1 = this.queue1[1:]
    return x
}
func (this *MyStack) Top() int {
    return this.queue1[0]
}
func (this *MyStack) Empty() bool {
    return len(this.queue1)==0
}
```
