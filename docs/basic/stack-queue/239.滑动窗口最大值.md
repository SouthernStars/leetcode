# 239.滑动窗口最大值
[leetcode 链接](https://leetcode.cn/problems/sliding-window-maximum/description/)

### 一、题目描述
给你一个整数数组 nums，有一个大小为 k 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。 返回 滑动窗口中的最大值 。

### 二、解题思路

大顶堆： 对于「最大值」，可以使用 优先队列（堆），其中大根堆可以实时维护一系列元素中的最大值

具体操作如下：初始时，将数组 nums 的前 k 个元素放入 大根堆中。每当向右移动窗口时，将新元素放入 大根堆中，此时堆顶元素即为堆中所有元素的最大值，此时这个最大值可能并不在滑动窗口中（出现在滑动窗口边界的左侧），因此，当继续向右移动窗口时，这个值就永远不可能出现在滑动窗口中了，可以永久性地从对重移除。不断移除堆顶元素，直到其确实在滑动窗口中，此时，堆顶元素就是滑动窗口中的最大值


### 三、代码实现

```Go
type Elem struct {
    value int
    index int
}
type IHeap []Elem
func (h IHeap) Len() int {
    return len(h)
}
func (h IHeap) Less(i int, j int) bool { // 大根堆
    return h[i].value > h[j].value
}
func (h IHeap) Swap(i int, j int) {
    h[i], h[j] = h[j], h[i]
}
func (h *IHeap) Push(x interface{}) { // 供 heap.Push 调用
    *h = append(*h, x.(Elem))
}
func (h *IHeap) Pop() interface{} { // 供 heap.Pop 调用
    n := len(*h)
    x := (*h)[n-1]
    *h = (*h)[0:n-1]
    return x
}
func maxSlidingWindow(nums []int, k int) []int {
    hp := &IHeap{}
    heap.Init(hp)
    for i:=0; i<k; i++ {
        heap.Push(hp, Elem{value: nums[i], index: i})
    }
    n := len(nums)
    ans := []int{}
    ans = append(ans, (*hp)[0].value)
    for i:=k; i<n; i++ {
        heap.Push(hp, Elem{value: nums[i], index: i})
        for (*hp)[0].index <= i-k {    // 根据当前堆顶元素的 index 判断是否在 窗口中，没有则 Pop
            heap.Pop(hp)
        }
        // 此时，堆顶元素为当前窗口的最大值
        ans = append(ans, (*hp)[0].value)
    }
    return ans
}
```
