# 739.每日温度
[leetcode 链接](https://leetcode.cn/problems/daily-temperatures/description/)

### 一、题目描述
给定一个整数数组 temperatures ，表示每天的温度，返回一个数组 answer ，其中 answer[i] 是指对于第 i 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 0 来代替。

### 二、解题思路

单调栈：维护一个单调栈，记录索引；遍历每个元素，然后将栈中弹出比当前温度低的元素，这些元素的下一个更高温度就是当前这天的温度；然后将该元素入栈。

时间复杂度：$O(n)$

空间复杂度：$O(n)$

### 三、代码实现

```Go
// 单调栈
func dailyTemperatures(temperatures []int) []int {
    n:=len(temperatures)
    ans := make([]int, n)
    stack := []int{}
    for i:=0; i<n; i++ {
        t:=temperatures[i]
        // 栈不空并且栈顶元素小于当前温度
        for len(stack)>0 && t>temperatures[stack[len(stack)-1]] {
            // 出栈
            top := stack[len(stack)-1]
            stack = stack[0:len(stack)-1]
            ans[top] = i-top
        }
        // 入栈 索引下标
        stack = append(stack, i)
    }
    return ans
}
```
