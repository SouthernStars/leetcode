# 84.柱状图中最大的矩形
[leetcode 链接](https://leetcode.cn/problems/largest-rectangle-in-histogram/description/)

### 一、题目描述
给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。求在该柱状图中，能够勾勒出来的矩形的最大面积。

### 二、解题思路

穷举法：具体操作如下：穷举 左右 边界，并找到在此边界中的 高度的最小值。乘积即为当前矩形的面积。通过比较记录最大面积

**同样也可以枚举「高」，我们可以使用一重循环枚举某一根柱子，将其固定为矩形的高度 h。随后我们从这跟柱子开始向两侧延伸，直到遇到高度小于 h 的柱子，就确定了矩形的左右边界。**

时间复杂度：$O(n^2)$

空间复杂度：$O(1)$


单调栈：具体操作如下：该方法是对枚举 高 的优化，需要进行向左右两边扩展，使得扩展到的柱子的高度均不小于 h。 维护一个单调栈 记录每根柱子左侧的第一个小于当前高度的柱子。使用同样的方法，从右进行遍历，记录当前柱子右侧的第一个小于当前高度的柱子

时间复杂度：$O(n)$

空间复杂度：$O(n)$

### 三、代码实现

单调栈
```c++
int largestRectangleArea(vector<int>& heights) {
    int n = heights.size();
    vector<int> left(n, -1), right(n, n);
    stack<int> st;
    for (int i = 0; i < n; i++) {
        while (!st.empty() && heights[i] < heights[st.top()]) {
            right[st.top()] = i;
            st.pop();
        }
        st.push(i);
    }
    st = stack<int>();
    for (int i = n - 1; i >= 0; i--) {
        while (!st.empty() && heights[i] < heights[st.top()]) {
            left[st.top()] = i;
            st.pop();
        }
        st.push(i);
    }
    int res = 0;
    for (int i = 0; i < n; i++) {
        res = max(res, (right[i] - left[i] - 1) * heights[i]);
    }
    return res;
}
```


# 85.最大矩形
[leetcode 链接](https://leetcode.cn/problems/maximal-rectangle/description/)

### 一、题目描述
给定一个仅包含 0 和 1 、大小为 rows x cols 的二维二进制矩阵，找出只包含 1 的最大矩形，并返回其面积。


### 二、解题思路

单调栈：转化成 84 题，然后遍历柱状图，得出全局最大值

### 三、代码实现


```c++
int maximalRectangle(vector<vector<char>>& matrix) {
    int m = matrix.size(), n = matrix[0].size();
    if (!m || !n) {
        return 0;
    }
    int maxArea = 0;
    vector<int> heights(n, 0);
    for (int i = 0; i < m; i++) {
        // 设置每列高度
        for (int j = 0; j < n; j++) {
            if (matrix[i][j] == '1') {
                heights[j] += 1;
            } else {
                heights[j] = 0;
            }
        }
        maxArea = max(maxArea, largestRectangleArea(heights));
    }
    return maxArea;
}

        while (!st.empty() && heights[i] < heights[st.top()]) {
            right[st.top()] = i;
            st.pop();
        }
        st.push(i);
    }
    st = stack<int>();
    for (int i = n - 1; i >= 0; i--) {
        while (!st.empty() && heights[i] < heights[st.top()]) {
            left[st.top()] = i;
            st.pop();

        st.push(i);
    }
    int res = 0;
    for (int i = 0; i < n; i++) {
        res = max(res, (right[i] - left[i] - 1) * heights[i]);
    }
    return res;
}
```
