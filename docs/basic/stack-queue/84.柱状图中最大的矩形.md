# 84.柱状图中最大的矩形
[leetcode 链接](https://leetcode.cn/problems/largest-rectangle-in-histogram/description/)

### 一、题目描述
给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。求在该柱状图中，能够勾勒出来的矩形的最大面积。

### 二、解题思路

穷举法：具体操作如下：穷举 左右 边界，并找到在此边界中的 高度的最小值。乘积即为当前矩形的面积。通过比较记录最大面积

**同样也可以枚举「高」，我们可以使用一重循环枚举某一根柱子，将其固定为矩形的高度 h。随后我们从这跟柱子开始向两侧延伸，直到遇到高度小于 h 的柱子，就确定了矩形的左右边界。**

时间复杂度：$O(n^2)$

空间复杂度：$O(1)$


单调栈：具体操作如下：该方法是对枚举 高 的优化，需要进行向左右两边扩展，使得扩展到的柱子的高度均不小于 h。 维护一个单调栈 记录每根柱子左侧的第一个小于当前高度的柱子。使用同样的方法，从右进行遍历，记录当前柱子右侧的第一个小于当前高度的柱子

时间复杂度：$O(n)$

空间复杂度：$O(n)$

### 三、代码实现

穷举法：

```Go
// 方案1：穷举
// 具体操作如下：穷举 左右 边界，并找到在此边界中的 高度的最小值。乘积即为当前矩形的面积。通过比较记录最大面积
func largestRectangleArea(heights []int) int {
    maxArea := 0
    n := len(heights)
    for left:=0; left < n; left ++ {
        minHeight := math.MaxInt
        for right:=left; right < n; right ++ {
            if heights[right] < minHeight {
                minHeight = heights[right]
            }
            tempArea := (right-left+1)*minHeight
            if tempArea > maxArea {
                maxArea = tempArea
            }
        }
    }
    return maxArea
}
func largestRectangleArea(heights []int) int {
    maxArea := 0
    n := len(heights)
    for mid:=0; mid < n; mid ++ {
        // 枚举高
        height := heights[mid]
        // 查找左右边界
        left, right := mid, mid
        for left-1>=0&&heights[left-1]>=height {
            left--
        }
        for right+1<n&&heights[right+1]>=height {
            right++
        }
        tmpArea := (right-left+1) * height
        if tmpArea > maxArea {
            maxArea = tmpArea
        }
    }
    return maxArea
}
```

单调栈
```Go
func largestRectangleArea(heights []int) int {
    n := len(heights)
    lhashtable := map[int]int{}
    lstack := []int{}
    for i:=0; i<n; i++ {
        for len(lstack) > 0 && heights[lstack[len(lstack)-1]]>=heights[i]{
            lstack = lstack[0:len(lstack)-1]
        }
        if len(lstack) == 0 {
            lhashtable[i] = -1
        } else {
            lhashtable[i] = lstack[len(lstack)-1]
        }
        lstack = append(lstack, i)
    }
    rhashtable := map[int]int{}
    rstack := []int{}
    for i:=n-1; i>=0; i-- {
        for len(rstack) > 0 && heights[rstack[len(rstack)-1]]>=heights[i]{
            rstack = rstack[0:len(rstack)-1]
        }
        if len(rstack) == 0 {
            rhashtable[i] = n
        } else {
            rhashtable[i] = rstack[len(rstack)-1]
        }
        rstack = append(rstack, i)
    }
    ans := 0
    // 枚举中间柱子
    for mid:=0; mid<n; mid++ {
        h := heights[mid]
        left := lhashtable[mid]+1 
        right := rhashtable[mid]-1
        tempArea := (right-left+1)*h
        if tempArea > ans {
            ans = tempArea
        }
    }
    return ans
}
```


# 85.最大矩形
[leetcode 链接](https://leetcode.cn/problems/maximal-rectangle/description/)

### 一、题目描述
给定一个仅包含 0 和 1 、大小为 rows x cols 的二维二进制矩阵，找出只包含 1 的最大矩形，并返回其面积。


### 二、解题思路

单调栈：转化成 84 题，然后遍历柱状图，得出全局最大值

### 三、代码实现


```go
func maximalRectangle(matrix [][]byte) int {
    // m 行 n 列
    m, n := len(matrix), len(matrix[0])

    graph := make([][]int, m)
    for i:=0; i<m; i++ {
        one := make([]int, n)
        for j:=0; j<n; j++ {
            if i==0 {
                if matrix[i][j] == '1' {
                    one[j] = 1
                }
            } else {
                if matrix[i][j] == '0' {
                    one[j] = 0
                } else {
                    one[j] = 1+graph[i-1][j]
                }
            }
        }
        graph[i] = one
    }
    // 对于每一行 0~m 应用单调栈
    maxArea := 0
    for i:=0; i<m; i++ {
        // 左栈记录第一个比它小的元素
        lstack := []int{}
        lhashtable := map[int]int{}
        for j:=0; j<n; j++ {
            for len(lstack) > 0 && graph[i][lstack[len(lstack)-1]] >= graph[i][j] {
                lstack = lstack[0:len(lstack)-1]
            }
            if len(lstack)==0 {
                lhashtable[j] = -1
            } else {
                lhashtable[j] = lstack[len(lstack)-1]
            }
            lstack = append(lstack, j)
        }
        // 右栈记录第一个比它小的元素
        rstack := []int{}
        rhashtable := map[int]int{}
        for j:=n-1; j>=0; j-- {
            for len(rstack) > 0 && graph[i][rstack[len(rstack)-1]] >= graph[i][j] {
                rstack = rstack[0:len(rstack)-1]
            }
            if len(rstack)==0 {
                rhashtable[j] = n
            } else {
                rhashtable[j] = rstack[len(rstack)-1]
            }
            rstack = append(rstack, j)
        }
        // 枚举中间柱子，计算面积
        for j:=0; j<n; j++ {
            height := graph[i][j]
            lindex := lhashtable[j]+1
            rindex := rhashtable[j]-1
            area := height*(rindex-lindex+1)
            maxArea = max(maxArea, area)
        }
    }
    return maxArea
}
```
