# 344.反转字符串
[leetcode 链接](https://leetcode.cn/problems/reverse-string/description/)

### 一、题目描述
编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 s 的形式给出。
不要给另外的数组分配额外的空间，你必须原地修改输入数组、使用 O(1) 的额外空间解决这一问题。


### 二、解题思路

双指针：low, high 分别指向字符串两端，进行交换

时间复杂度：$O(n)$

空间复杂度：$O(1)$

### 三、代码实现

```c++
void reverseString(vector<char>& s) {
    int n = s.size();
    for (int i = 0; i < n / 2; i++) {
        swap(s[i], s[n - i - 1]);
    }
}
```
