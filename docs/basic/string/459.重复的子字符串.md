# 459.重复的子字符串
[leetcode 链接](https://leetcode.cn/problems/repeated-substring-pattern/description/)

### 一、题目描述
给定一个非空的字符串 s ，检查是否可以通过由它的一个子串重复多次构成。


### 二、解题思路
穷举法：

如果一个长度为 n 的字符串 s 可以由它的一个长度为 n′ 的子串 s′ 重复多次构成，那么：
- n 一定是 n′ 的倍数；
- s′ 一定是 s 的前缀；
- 对于任意的 i∈[n′,n)，有 s[i]=s[i−n′]。

也就是说，s 中长度为 n′ 的前缀就是 s′ ，并且在这之后的每一个位置上的字符 s[i]，都需要与它之前的第 n′ 个字符 s[i−n′] 相同。

因此，我们可以从小到大枚举 n′ ，并对字符串 s 进行遍历，进行上述的判断。注意到一个小优化是，因为子串至少需要重复一次，所以 n′ 不会大于 n 的一半，我们只需要在 [1, n/2] 的范围内枚举 n′ 即可。

时间复杂度：$O(n^2)$

空间复杂度：$O(1)$



字符串匹配：

- `(s + s).find(s, 1)`：从位置 1 开始找 `s`
- 如果找到的位置**不等于** `s.size()` → 说明在中间找到了另一个 `s` → `true`
- 如果找到的位置**等于** `s.size()` → 说明只在末尾找到（开头的那个被跳过了，末尾的是拼接后的后半部分）→ `false`

### 三、代码实现

穷举法：

```c++
bool repeatedSubstringPattern(string s) {
    int n = s.size();
    for (int i = 1; i <= n / 2; i++) {
        if (n % i == 0) {
            bool match = true;
            for (int j = i; j < n; j++) {
                if (s[j] != s[j - i]) {
                    match = false;
                    break;
                }
            }
            if (match) {
                return true;
            }
        }
    }
    return false;
}

```

字符串匹配：

```java

bool repeatedSubstringPattern(string s) {
    return (s + s).find(s, 1) != s.size();
}
```

kmp:

```java
vector<int> getNext(string& s) {
    int n = s.size();
    vector<int> next(n, -1);
    next[0] = 0;
    for (int i = 1, j = 0; i < n; i++) {
        while (j > 0 && s[i] != s[j]) {
            j = next[j - 1];
        }
        if (s[i] == s[j]) {
            j++;
        }
        next[i] = j;
    }
    return next;
}
bool repeatedSubstringPattern(string s) {
    vector<int> next = getNext(s);
    int len = next[s.size() - 1], n = s.size();
    return len > 0 && (n % (n - len) == 0);
}
```
