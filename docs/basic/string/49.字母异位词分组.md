# 49.字母异位词分组
[leetcode 链接](https://leetcode.cn/problems/group-anagrams/description/)

### 一、题目描述
给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单

### 二、解题思路

排序+哈希表：对字符串的字符进行排序，排序相同的为一个分组 使用 map , 字符串作为 key

### 三、代码实现

```go
func groupAnagrams(strs []string) (ans [][]string) {
    mp := make(map[string][]string, 0)
    for _, s:=range strs {
        by :=[]byte(s) // 字节化
        sort.Slice(by, func(a, b int) bool { return by[a] < by[b]})
        sortedstr := string(by)
        mp[sortedstr] = append(mp[sortedstr], s) 
    }
    for _, v:=range mp {
        ans = append(ans, v)
    }
    return ans
}
```
