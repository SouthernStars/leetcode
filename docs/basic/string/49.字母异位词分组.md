# 49.字母异位词分组
[leetcode 链接](https://leetcode.cn/problems/group-anagrams/description/)

### 一、题目描述
给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单

### 二、解题思路

排序+哈希表：对字符串的字符进行排序，排序相同的为一个分组 使用 map , 字符串作为 key

### 三、代码实现

```c++
vector<vector<string>> groupAnagrams(vector<string>& strs) {
    unordered_map<string, vector<string>> mp;
    for (auto& str : strs) {
        string cur = str;
        sort(cur.begin(), cur.end());
        mp[cur].push_back(str);
    }
    vector<vector<string>> res;
    for (auto& p : mp) {
        res.push_back(p.second);
    }
    return res;
}
```
