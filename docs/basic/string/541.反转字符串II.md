# 541.反转字符串II
[leetcode 链接](https://leetcode.cn/problems/reverse-string-ii/description/)

### 一、题目描述
给定一个字符串 s 和一个整数 k，从字符串开头算起，每计数至 2k 个字符，就反转这 2k 字符中的前 k 个字符。

- 如果剩余字符少于 k 个，则将剩余字符全部反转。
- 如果剩余字符小于 2k 但大于或等于 k 个，则反转前 k 个字符，其余字符保持原样。

### 二、解题思路

模拟法：反转每个下标从 2k 的倍数开始的，长度为 k 的子串。若该子串长度不足 k，则反转整个子串。

时间复杂度：$O(n)$

空间复杂度：$O(n)$

### 三、代码实现

```Go
func reverseStr(s string, k int) string {
    barr := ([]byte)(s)
    length := len(barr)
    for i:=0; i<=length/(2*k); i++ {
        start := i*(2*k)
        end := i*(2*k)+k
        if end > length {
            end = length
        }
        reverse(barr[start:end])
    }
    return (string)(barr)
}

func reverse(s []byte) {
    low, high := 0, len(s)-1
    for low <= high {
        s[low], s[high] = s[high], s[low]
        low ++
        high--
    }
}
```
