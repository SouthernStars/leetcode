# 844.比较含退格的字符串
[leetcode 链接](https://leetcode.cn/problems/backspace-string-compare/description/)

### 一、题目描述
给定 s 和 t 两个字符串，当它们分别被输入到空白的文本编辑器后，如果两者相等，返回 true 。# 代表退格字符。

注意：如果对空文本输入退格字符，文本继续为空。

### 二、解题思路

模拟法

### 三、代码实现

```go
func backspaceCompare(s string, t string) bool {
    s1 := deal(s)
    t1 := deal(t)
    return s1 == t1
}

func deal(str string) string {
    ans := ""
    for i:=0; i<len(str); i++ {
        if str[i] != '#' {
            ans += str[i:i+1]
        } else {
            if len(ans) != 0 {
                ans = ans[0:len(ans)-1]
            }    
        }
    }
    return ans
}
```
