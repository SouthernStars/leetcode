### AB62 数位染色

描述:小红拿到了一个正整数x。她可以将其中一些数位染成红色。然后她想让所有染红的数位数字之和等于没染色的数位数字之和。她不知道能不能达成目标。你能告诉她吗？

输入描述：一个正整数 x ，1 ≤ x ≤ 10^18

输出描述： 如果小红能按要求完成染色，输出"Yes"。否则输出"No"。

示例1
- 输入：1234567
- 输出： Yes
- 说明： 将3、4、7染成红色即可，这样3+4+7=1+2+5+6

```go
package main

import (
    "fmt"
)

// 分隔等和子集
func main() {
    var x int
    fmt.Scan(&x)
    // 辗转相除获取数位
    num := []int{}
    sum := 0
    for x != 0 {
        yu := int(x % 10)
        x = x / 10
        num = append(num, yu)
        sum += int(yu)
    }
    if sum % 2 != 0 || len(num) < 2 {
        fmt.Printf("No")
        return
    }
    // 使用动态规划，0-1背包问题
    // 背包容量为 target，物品为 num
    // dp[i][j] 表示在容量为 j 的情况下能装东西的最大价值
    // 状态转移方程： dp[i][j] = max(dp[i-1][j], dp[i-1][j-num[i-1]]+num[i-1])   num[i-1] <= j
    //                        = dp[i-1][j]    num[i-1] > j
    target := sum / 2 
    m := len(num)
    dp := make([][]int, m+1)
    for i:=0; i<=m; i++ {
        dp[i] = make([]int, target+1)
    }
    // 初始化 dp[0][*] = 0  dp[*][0] = 0
    for i:=1; i<=m; i++ {
        for j:=1; j<=target; j++ {
            if num[i-1] > j { // 装不了
                dp[i][j] = dp[i-1][j]
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i-1][j-num[i-1]]+num[i-1])
            }
        }
    }
    if dp[m][target] == target {
        fmt.Printf("Yes")
    } else {
        fmt.Printf("No")
    }
    return
}
```
